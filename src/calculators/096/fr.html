Comment utiliser cette calculatrice

Suivez ces étapes simples pour obtenir des résultats précis :

  1. Suma tous Les costes fijos mensuales: alquiler de nave/oficina, gestoría, seguros, salarios base, software, teléfono
  2. Define Le prix de venta medio par travail/obra/unite (ou Le ingreso medio par actuación)
  3. Determina Le cout variable par travail: matériaux directos, subcontratas, dietas, combustible, comisiones
  4. Divide Les costes fijos entre Le marge de contribución: obas Le número mínimo de trabajos pour no perder dinero
  5. Multiplica Le número de trabajos par Le prix de venta pour Obtenir La facturación mínima de equilibrio
