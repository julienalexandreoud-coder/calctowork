Calcula el número mínimo de obras o trabajos que necesitas al mes para no perder dinero. Introduce costes fijos, precio de venta y coste variable unitario.

Cómo usar esta calculadora

Sigue estos pasos para obtener resultados precisos:

  1. Suma todos los costes fijos mensuales: alquiler de nave/oficina, gestoría, seguros, salarios base, software, teléfono
  2. Define el precio de venta medio por trabajo/obra/unidad (o el ingreso medio por actuación)
  3. Determina el coste variable por trabajo: materiales directos, subcontratas, dietas, combustible, comisiones
  4. Calcula el margen de contribución: precio_venta âˆ’ coste_variable (lo que cada trabajo aporta para cubrir costes fijos)
  5. Divide los costes fijos entre el margen de contribución: obtienes el número mínimo de trabajos para no perder dinero
  6. Multiplica el número de trabajos por el precio de venta para obtener la facturación mínima de equilibrio

Cómo funciona

Esta calculadora utiliza fórmulas estándar para calcular resultados precisos basados en tus valores. Ingresa los datos y pulsa Calcular para obtener resultados al instante.
