How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calculatete area: 6m x 4m = 24m²
  2. Calculatete volume: 24m² x 0.25m = 6m³ of concrete
  3. Calculatete cement: 6m³ x 7 bags/m³ = 42 bags
  4. Calculatete steel: 6m³ x 120 kg/m³ = 720 kg of steel
  5. Calculatete aggregates: 6m³ x 0.55 = 3.30m³ sand, 6m³ x 0.80 = 4.80m³ gravel
