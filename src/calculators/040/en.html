How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Assume optimal speed: 1.5 m/s
  2. Calculate flow: 2 L/s = 0.002 m3/s
  3. Calculate diameter: sqrt((4 x 0.002) / (pi x 1.5)) = 0.041m = 41mm
  4. Select commercial: DN40 (40mm) or DN50 (50mm) depending on length
