How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Set up proportion: 7.5 is to 22.5 as 12.8 is to x
  2. Cross multiply: x = (22.5 × 12.8) ÷ 7.5
  3. Calculate numerator: 22.5 × 12.8 = 288
  4. Divide: 288 ÷ 7.5 = 38.4 (cost of 12.8kg)
