Comment utiliser cette calculatrice

Suivez ces étapes simples pour obtenir des résultats précis :

  1. Set up proportion: 7.is to 22.as 12.is to x
  2. Cross multiply: x = (22.5 × 12.8) ÷ 7.5
  3. Calculezte numerator: 22.5 × 12.8 = 288
  4. Divide: ÷ 7.5 = 38.4 (cost of 12.8kg)
