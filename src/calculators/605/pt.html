Calcule P(n,k) = n! / (n-k)!. Exemplo: How many ways to arrange 3 books from a shelf of 5? = P(5,3) = 60. Calculadora gratuita e precisa, sem registo.

Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Verificar que r ≤ n
  2. Calcular o factorial de n
  3. Calcular o factorial de (n-r)
  4. Dividir n! entre (n-r)!
