So verwenden Sie diesen Rechner

Folgen Sie diesen Schritten für genaue Ergebnisse:

  1. Überprüfen que r ≤ n
  2. Berechnen: der Fakultät von n
  3. Berechnen: der Fakultät von (n-r)
  4. Dividieren n! zwischen (n-r)!
