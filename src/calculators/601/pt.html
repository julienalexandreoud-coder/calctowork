O que é a Mediana e quando deve utilizá-la?

A mediana é uma medida de tendência central que representa o valor intermédio de um conjunto de dados ordenados. Diferente da média, a mediana não é influenciada por valores extremos (outliers), tornando-a ideal para analisar distribuições assimétricas, como salários, preços de imóveis ou tempos de espera.

Utilize a Calculadora Mediana quando:

  • Precisa de encontrar o valor típico de um conjunto com poucos dados (ex.: notas de 5 alunos).
  • Os seus dados contêm outliers que distorceriam a média (ex.: rendimentos com valores muito altos ou baixos).
  • Trabalha com variáveis ordinais onde a média não faz sentido (ex.: classificações de 1 a 5).
  • Quer uma análise rápida sem complicações estatísticas.

Esta calculadora é especialmente útil em contextos portugueses, como análise de preços no supermercado, avaliação de alunos ou comparação de salários em euros.

Fórmula explicada passo a passo

A Calculadora Mediana utiliza a seguinte lógica, apresentada de forma simplificada:

  1. Lista de valores (v1 a v10): Os campos v1, v2, v3, v4, v5, v6, v7, v8, v9 e v10 recebem os números que pretende analisar. Apenas os campos preenchidos com números válidos são considerados.
  2. Ordenação: O algoritmo ordena os valores do menor para o maior (função sort).
  3. Cálculo do número de elementos (n): Conta quantos valores válidos foram inseridos (ex.: se preencheu 5 campos, n = 5).
  4. Verificação de paridade:
    • Se n for ímpar: a mediana é o valor na posição central (Math.floor(n/2) arredonda para baixo). Exemplo: com 7 números, a posição central é o 4.º valor ordenado.
    • Se n for par: a mediana é a média aritmética dos dois valores centrais. Exemplo: com 6 números, soma o 3.º e o 4.º valor e divide por 2.
  5. Resultado: A calculadora devolve o valor exato da mediana.

Variáveis da fórmula:

Variável Descrição
v1 a v10 Campos de input onde insere os números (apenas números válidos são processados)
vals[ ] Array que armazena todos os números válidos após validação
sort(a,b) Função que ordena os números de forma crescente
n Número total de elementos válidos no array
n%2 === 1 Verifica se n é ímpar (resto da divisão por 2 igual a 1)
Math.floor(n/2) Arredonda para baixo o valor de n/2 (índice do elemento central para n ímpar)
(vals[n/2-1] + vals[n/2]) / 2 Média dos dois valores centrais quando n é par

Exemplos práticos com a Calculadora Mediana

Exemplo 1: Número ímpar de dados

Cenário: Um grupo de 5 amigos em Lisboa registou os seus gastos num café: 4€, 7€, 2€, 9€, 5€.

Passos na calculadora:

  • Insira nos campos: v1=4, v2=7, v3=2, v4=9, v5=5 (deixe v6 a v10 vazios).
  • A calculadora ordena os valores: [2, 4, 5, 7, 9].
  • Como n=5 (ímpar), a mediana é o 3.º valor: 5€.

Interpretação: Metade dos amigos gastou até 5€, a outra metade gastou 5€ ou mais. A mediana de 5€ representa melhor o gasto típico do que a média (5,4€), que é ligeiramente puxada para cima pelo valor 9€.

Exemplo 2: Número par de dados

Cenário: Preços de 6 artigos num supermercado em Portugal: 1,50€, 3,20€, 2,80€, 4,00€, 2,50€, 3,60€.

Passos na calculadora:

  • Insira: v1=1.5, v2=3.2, v3=2.8, v4=4.0, v5=2.5, v6=3.6 (use ponto como separador decimal ou vírgula, conforme a calculadora aceitar; em Portugal usa-se vírgula, mas muitas ferramentas aceitam ambos).
  • Ordenação: [1.5, 2.5, 2.8, 3.2, 3.6, 4.0].
  • Como n=6 (par), a mediana é a média entre o 3.º e o 4.º valor: (2.8 + 3.2) / 2 = 3,00€.

Interpretação: O preço mediano dos artigos é 3,00€. Metade dos produtos custa menos de 3€, metade custa mais. A mediana é resistente ao valor extremo de 4,00€.

Erros comuns ao calcular a mediana

  • Não ordenar os dados: Muitas pessoas esquecem que a mediana exige dados ordenados. A calculadora faz isto automaticamente, mas ao fazer manualmente, é o erro mais frequente.
  • Confundir mediana com média: A média soma tudo e divide; a mediana é o valor central. Exemplo: [1, 2, 100] tem média ≈34,3, mas mediana=2. São conceitos distintos.
  • Esquecer os dois valores centrais (n par): Para conjuntos com número par de dados, muitos calculam apenas um valor, mas é obrigatório fazer a média dos dois centrais.
  • Inserir dados não numéricos: Se colocar texto ou deixar campos vazios com caracteres especiais, a calculadora ignora-os, mas pode confundir o resultado se não validar. A nossa calculadora trata automaticamente estes casos.
  • Usar vírgula errada: Em Portugal, o separador decimal é a vírgula (ex.: 2,5). Se a calculadora exigir ponto, converta ou ajuste. Verifique o formato aceite antes de inserir.

Perguntas Frequentes (FAQ)

A Calculadora Mediana funciona com dados em euros?

Sim, funciona perfeitamente com valores em euros ou qualquer outra unidade. Basta inserir os números (ex.: 12.50 ou 12,50). O resultado será o valor mediano, independentemente da moeda ou unidade de medida.

Posso usar a calculadora com menos de 10 números?

Claro. Pode inserir quantos números quiser, desde 1 até 10. Se inserir apenas 1 número, a mediana será esse mesmo número. Se inserir 2, será a média dos dois. A calculadora adapta-se automaticamente ao número de valores preenchidos.

O que acontece se eu inserir letras ou campos vazios?

A calculadora ignora campos vazios ou com texto não numérico. Apenas números válidos (inteiros ou decimais) são processados. Se todos os campos estiverem vazios ou inválidos, a calculadora apresenta um erro, pois não há dados para calcular.
