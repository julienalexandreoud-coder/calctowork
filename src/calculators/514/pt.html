Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Multiplicar duración por bitrate
  2. Dividir por 8 para convertir bits a bytes
  3. Dividir por 1024 para MB

Fórmula

Tamano (MB) = (Duración × Bitrate) / (8 × 1024)
