How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calculatete wall volume: 5m x 2m x 0.40m = 4m³
  2. Calculatete stone: 4m³ x 0.75 = 3m³ of stone approximately
  3. Calculatete mortar: 4m³ x 350 kg/m³ = 1400 kg of mortar
  4. Allow 15-20% mortar for filling joints

How It Works

Formula

Volume = Length x Height x Thickness; Stone = Volume x 0.75; Mortar = Volume x 0.35
