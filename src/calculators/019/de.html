How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Berechnen: Volumen des Mauer: 5m × 2m × 0.40m = 4m³
  2. Berechnen: Stein: 4m³ × 0.75 = 3m³ von Stein aproximadamente
  3. Berechnen: Mörtel: 4m³ × 350 kg/m³ = 1400 kg Mörtel
  4. Berücksichtigen 15-20% von Mörtel für Verfüllung von juntas

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Länge Mauer
  • Höhe Mauer
  • Espesor Mauer

Result Outputs

  • Volumen Mauer
  • Stein (kg)
  • Mörtel
  • Sacos Zement

Formula

Volumen = Länge × hoch × Dicke; Stein = Volumen × 0.75; Mörtel = Volumen × 0.35
