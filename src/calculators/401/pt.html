Calcule as suas calorias diárias (TDEE). Insira peso, altura, idade, sexo e nível de atividade. Com intervalos de referencia e contexto de saude.

Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Calcula BMR usando fórmula Mifflin-St Jeor segundododo género
  2. Selecciona nivel de atividade (1=sedentario, 5=muy activo)
  3. Multiplica BMR por factor de atividade correspondiente
  4. Para perder peso resta 500 kcal, para ganar suma 500 kcal

Como funciona

Esta calculadora utiliza fórmulas padrão para calcular resultados precisos com base nos seus valores. Insira os dados e clique em Calcular para obter resultados instantâneos.

Campos de entrada

  • Peso (kg)
  • Altura (cm)
  • Idade (anos)
  • Sexo (1=Masculino, 2=Feminino)

Resultados

  • Calorias manutenção
  • Para emagrecer (−500 kcal)
  • Para ganhar peso (+500 kcal)
  • Taxa metabólica basal (BMR)

Fórmula

BMR = (10 × peso) + (6.25 × altura) - (5 × edad) ± 5/161; TDEE = BMR × factor atividade

Perguntas frequentes

Esta calculadora usa fórmulas padrão com base nas entradas fornecidas. Os resultados são tipicamente precisos em 95%+ quando as entradas estão corretas e as unidades de medida são consistentes.

Sim, esta calculadora suporta vários sistemas de unidades. Selecione sua unidade preferida no menu suspenso. Os resultados se ajustam automaticamente às suas unidades selecionadas.
