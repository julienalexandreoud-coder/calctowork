How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calcule BMR en utilisant formule Mifflin-St Jeor selon género
  2. Sélectionnez nivel de actividad (1=sedentario, 5=tres activo)
  3. Multiplica BMR par facteur de actividad correspondiente
  4. Pour perder poids rcette kcal, pour ganar suma kcal

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Poids (kg)
  • Taille (cm)
  • Âge (ans)
  • Sexe (1=Homme, 2=Femme)

Result Outputs

  • Calories maintien
  • Pour maigrir (−500 kcal)
  • Pour prendre du poids (+500 kcal)
  • Métabolisme de base (BMR)

Formula

BMR = (10 × poids) + (6.25 × hauteur) - (5 × edad) ± 5/161; TDEE = BMR × facteur actividad
