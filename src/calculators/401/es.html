Calcula tus calorías diarias de mantenimiento (TDEE). Introduce peso, altura, edad, sexo y nivel de actividad. Resultado en kcal/día.

Cómo usar esta calculadora

Sigue estos pasos para obtener resultados precisos:

  1. Calcula BMR usando fórmula Mifflin-St Jeor según género
  2. Selecciona nivel de actividad (1=sedentario, 5=muy activo)
  3. Multiplica BMR por factor de actividad correspondiente
  4. Para perder peso resta 500 kcal, para ganar suma 500 kcal

Cómo funciona

Esta calculadora utiliza fórmulas estándar para calcular resultados precisos basados en tus valores. Ingresa los datos y pulsa Calcular para obtener resultados al instante.

Campos de entrada

  • Peso (kg)
  • Altura (cm)
  • Edad (años)
  • Sexo (1=Hombre, 2=Mujer)

Resultados

  • Calorías mantenimiento
  • Para adelgazar (−500 kcal)
  • Para ganar peso (+500 kcal)
  • Metabolismo basal (BMR)

Fórmula

BMR = (10 × peso) + (6.25 × altura) - (5 × edad) ± 5/161; TDEE = BMR × factor actividad
