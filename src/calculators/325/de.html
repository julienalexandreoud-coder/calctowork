So verwenden Sie diesen Rechner

Folgen Sie diesen Schritten für genaue Ergebnisse:

  1. Geben Sie den Wert für ein: dividendo jährlich durch acción
  2. Geben Sie den Wert für ein: preis von die acción
  3. Klicken Sie auf Berechnen
  4. Überprüfen Sie das Ergebnis: yield %

Wie es funktioniert

Dieser Rechner verwendet Standardformeln, um genaue Ergebnisse basierend auf Ihren Eingaben zu berechnen. Geben Sie Ihre Werte ein und klicken Sie auf Berechnen.

Eingabefelder

  • Dividendo jährlich durch acción
  • Preis von die acción

Ergebnisse

  • Yield %

Formel

Dividend Yield% = (Annual Dividend ÷ Share Price) × 100
