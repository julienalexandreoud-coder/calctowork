Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Calcular interés mensual (balance × tasa/12)
  2. Restar interés do pago para obter abono a capital
  3. Reducir balance por o abono a capital
  4. Repetir até saldo cero

Como funciona

Esta calculadora utiliza fórmulas padrão para calcular resultados precisos com base nos seus valores. Insira os dados e clique em Calcular para obter resultados instantâneos.

Campos de entrada

  • Saldo
  • Taxa %
  • Pagamento mensal

Resultados

  • Meses
  • Juros totais

Fórmula

Simulación iterativa: Balance_n+1 = Balance_n - (Pago - Interés)
