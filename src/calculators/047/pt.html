A Calculadora de Quadro Elétrico é uma ferramenta essencial para eletricistas, engenheiros e estudantes que precisam dimensionar corretamente a proteção de uma instalação elétrica. Esta calculadora permite determinar rapidamente o disjuntor geral, o diferencial recomendado e o número estimado de circuitos, utilizando uma metodologia clara e baseada nas normas técnicas portuguesas.

O que a Calculadora de Quadro Elétrico faz e quando utilizá-la

Esta calculadora foi desenvolvida para simplificar o processo de dimensionamento de quadros elétricos residenciais e comerciais de pequeno porte. Com apenas três informações de entrada, ela calcula automaticamente três parâmetros fundamentais para qualquer projeto elétrico:

  • Corrente máxima (A): O valor da corrente total que o circuito deverá suportar, considerando a potência instalada e o fator de simultaneidade.
  • Disjuntor geral recomendado (A): O valor comercial do disjuntor geral (ICP) mais adequado para proteger a instalação.
  • Número de circuitos estimados: A quantidade mínima de circuitos terminais (com disjuntores de 16A) necessária para distribuir a carga de forma segura.

Deve utilizar esta ferramenta sempre que estiver a projetar ou remodelar um quadro elétrico, especialmente para instalações monofásicas (230V) em Portugal. A calculadora é particularmente útil nas fases iniciais do projeto, antes de detalhar cada circuito específico, pois fornece uma visão geral rápida e precisa das necessidades de proteção do quadro.

A fórmula explicada variável a variável

A calculadora utiliza uma lógica simples mas eficaz, baseada na Lei de Ohm e nas potências típicas de disjuntores comerciais. Eis como cada variável influencia o resultado:

1. Potência instalada (potencia_kw)

Este é o valor total da potência de todos os equipamentos que serão ligados no quadro, expresso em quilowatts (kW). É o ponto de partida do cálculo. Quanto maior a potência, maior será a corrente e, consequentemente, o disjuntor necessário.

2. Tensão da instalação (tension_v)

A tensão nominal do sistema. Em Portugal, o padrão para residências é 230V (monofásico). Se a tensão for diferente de 230V, a calculadora assume um sistema trifásico e utiliza a raiz quadrada de 3 no denominador para calcular a corrente. Isto é fundamental porque a corrente num sistema trifásico é menor do que num monofásico para a mesma potência.

3. Fator de simultaneidade (factor_simultaneidad)

Representa a probabilidade de todos os equipamentos estarem ligados ao mesmo tempo. Um valor típico para residências é 0,7 (70%), mas pode variar conforme o tipo de instalação. Por exemplo, numa cozinha industrial pode ser mais perto de 1, enquanto numa habitação raramente ultrapassa 0,7 ou 0,8.

Cálculo da corrente máxima

A fórmula central é: I = (P × 1000 × fs) / (V × fator_tensão)

Onde o fator_tensão é 1 para 230V (monofásico) ou √3 (aproximadamente 1,732) para outras tensões (trifásico). O resultado é arredondado a uma casa decimal.

Seleção do disjuntor geral e circuitos

Após obter a corrente máxima, a calculadora seleciona o disjuntor comercial imediatamente superior (16A, 25A, 32A, 40A, 50A ou 63A). Para o número de circuitos, divide a corrente máxima por 16 (corrente típica de um disjuntor terminal) e arredonda para cima, garantindo que cada circuito não fique sobrecarregado.

Dois exemplos práticos com números concretos

Exemplo 1: Habitação familiar padrão

Dados de entrada: Potência = 6,9 kW, Tensão = 230 V, Fator de simultaneidade = 0,7

  1. Cálculo: I = (6,9 × 1000 × 0,7) / (230 × 1) = 4830 / 230 = 21,0 A
  2. Disjuntor geral: 21,0 A está entre 16A e 25A, logo o ICP recomendado é 25A.
  3. Número de circuitos: 21,0 / 16 = 1,31, arredondado para cima: 2 circuitos.

Resultado final: Disjuntor geral de 25A, 2 circuitos terminais de 16A cada. Isto significa que a instalação necessita de pelo menos 2 disjuntores no quadro (ex: um para iluminação e um para tomadas).

Exemplo 2: Pequeno escritório com equipamentos

Dados de entrada: Potência = 11,5 kW, Tensão = 400 V (trifásico), Fator de simultaneidade = 0,8

  1. Cálculo: I = (11,5 × 1000 × 0,8) / (400 × 1,732) = 9200 / 692,8 = 13,3 A
  2. Disjuntor geral: 13,3 A está abaixo de 16A, mas o disjuntor mínimo comercial é 16A.
  3. Número de circuitos: 13,3 / 16 = 0,83, arredondado para cima: 1 circuito (mas na prática, um escritório precisará de vários).

Resultado final: Disjuntor geral de 16A, apenas 1 circuito estimado. Nota: neste caso, como a corrente é baixa devido à tensão trifásica, o número de circuitos parece pequeno, mas na realidade o quadro deverá ter vários circuitos para diferentes zonas ou equipamentos. Este cálculo serve como base mínima.

Erros comuns ao dimensionar o quadro elétrico

Mesmo com ferramentas como a Calculadora de Quadro Elétrico, alguns erros frequentes podem comprometer a segurança da instalação:

  • Ignorar o fator de simultaneidade: Muitas pessoas utilizam a potência total sem considerar que nem todos os equipamentos funcionam ao mesmo tempo. Isto leva a disjuntores sobredimensionados e quadros mais caros do que o necessário.
  • Confundir potência em kW com kVA: Em instalações com motores ou transformadores, a potência aparente (kVA) é diferente da potência ativa (kW). A calculadora trabalha com kW, mas se tiver aparelhos com fator de potência baixo, deve converter os valores corretamente.
  • Usar a tensão errada: Em Portugal, a tensão monofásica padrão é 230V, mas algumas instalações antigas podem ter 220V. Além disso, confundir monofásico com trifásico altera completamente o resultado da corrente.
  • Desprezar o número mínimo de circuitos: A calculadora oferece um valor estimado, mas as normas portuguesas (como a RTIEBT) exigem circuitos separados para iluminação, tomadas, cozinha e equipamentos especiais. O resultado da calculadora é um ponto de partida, não um número definitivo.
  • Não verificar a seção do cabo: A calculadora só dimensiona o disjuntor e o número de circuitos. A bitola do cabo (seção) deve ser calculada separadamente, com base na corrente e no comprimento do circuito.

Perguntas frequentes (FAQ)

1. Esta calculadora substitui um projeto elétrico completo?

Não. A Calculadora de Quadro Elétrico é uma ferramenta de apoio para uma estimativa rápida e fiável, mas não substitui o projeto detalhado que um eletricista ou engenheiro deve realizar. Ela é ideal para orçamentos, anteprojetos ou para verificar se o dimensionamento está dentro dos parâmetros normais, mas cada circuito terminal deve ser verificado individualmente.

2. Posso utilizar esta calculadora para instalações trifásicas?

Sim. Se inserir um valor de tensão diferente de 230V (por exemplo, 400V), a calculadora assume automaticamente um sistema trifásico e utiliza a raiz quadrada de 3 no cálculo. Lembre-se que a potência deve ser a potência total trifásica (soma de todas as fases).

3. Qual o melhor fator de simultaneidade a utilizar?

Para habitações familiares em Portugal, recomenda-se 0,7 (70%). Para escritórios ou comércios, 0,8 pode ser mais realista. Se tiver dúvidas, utilize 0,8 como valor conservador. Evite usar 1,0 a menos que tenha certeza de que todos os equipamentos funcionarão ao mesmo tempo (o que é raro).

Utilize a Calculadora de Quadro Elétrico sempre que precisar de uma resposta clara e rápida para o dimensionamento básico do seu quadro. Com os dados corretos, terá uma base sólida para continuar o seu projeto elétrico em conformidade com as boas práticas e as normas em vigor em Portugal.
