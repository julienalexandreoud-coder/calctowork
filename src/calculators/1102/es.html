Calcula cuánto mantillo necesitas en metros cúbicos, yardas cúbicas y sacos. Introduce las dimensiones del área y la profundidad para obtener las cantidades exactas.

Cómo usar esta calculadora

Sigue estos pasos para obtener resultados precisos:

  1. Enter your dataset as comma-separated numbers
  2. Calculate the mean first
  3. For each value: subtract the mean, square the result
  4. Sum all the squared differences
  5. For population std dev: divide by N, then take square root
  6. For sample std dev: divide by (N-1), then take square root

Cómo funciona

Esta calculadora utiliza fórmulas estándar para calcular resultados precisos basados en tus valores. Ingresa los datos y pulsa Calcular para obtener resultados al instante.

Fórmula

σ = sqrt(Σ(x-μ)²/N) population; s = sqrt(Σ(x-x̄)²/(n-1)) sample
