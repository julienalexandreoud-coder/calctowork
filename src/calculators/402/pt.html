Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Convierte altura a metros (ej: 175 cm = 1.75 m)
  2. Calcula peso mínimo: 18.5 × altura²
  3. Calcula peso máximo: 24.9 × altura²
  4. Opcional: calcula fórmulas Lorentz e Devine para referencia

Como funciona
