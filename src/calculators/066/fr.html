Pré-dimensionnez les poutrelles métalliques pour bâtiments industriels selon portée et charge. Calculatrice facile a utiliser avec decomposition de la.

Comment utiliser cette calculatrice

Suivez ces étapes simples pour obtenir des résultats précis :

  1. Saisissez luz m: m
  2. Saisissez carga kn m2: kN/m²
  3. Saisissez longueur nave m: m
  4. Interpreta resultat: {momento_kn_m}

Comment ça fonctionne

Cette calculatrice utilise des formules standard pour calculer des résultats précis basés sur vos entrées. Saisissez vos valeurs et cliquez sur Calculer pour obtenir des résultats instantanés.

Champs de saisie

  • Portée (m)
  • Charge totale (kN/m²)
  • Longueur bâtiment (m)

Résultats

  • Moment max (kN·m)
  • Module résistant min (cm³)
  • Kg par poutrelle
  • Nombre de poutrelles

Formule especifica pour estructuras-metalicas - consulta documentacion tecnica.
