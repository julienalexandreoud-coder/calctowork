Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Convertir altura a pressão: 15m = 1.5 bar (10m = 1 bar)
  2. Calcular pérdidas: 15m × 0.15 = 2.25m adicionales
  3. Presión total: 15m + 2.25m = 17.25m = 1.725 bar
  4. Estimar potência de bomba segundododo caudal requerido

Como funciona

Esta calculadora utiliza fórmulas padrão para calcular resultados precisos com base nos seus valores. Insira os dados e clique em Calcular para obter resultados instantâneos.

Fórmula

Presión_mca = altura + (altura × pérdidas%); Presión_bar = presión_mca ÷ 10
