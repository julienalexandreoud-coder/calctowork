How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Assume area: 10m x 10m = 100m2
  2. Drippers: 100m2 x 11 drippers/m2 = 1,100 drippers
  3. Total flow: 1,100 x 4 L/h = 4,400 L/h
  4. Convert to L/min: 4,400 / 60 = 73.3 L/min
