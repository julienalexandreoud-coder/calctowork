How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Identificar n (total) e r (seleção)
  2. Verificar se r ≤ n
  3. Calcular combinações: C(n,r) = n! / (r! × (n-r)!)
  4. Calcular permutações: P(n,r) = n! / (n-r)!
  5. Interpretar a diferença entre os dois valores
