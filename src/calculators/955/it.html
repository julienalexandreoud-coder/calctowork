How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Identificare la temperatura en Celsius
  2. Convertire a Fahrenheit: F = C × 9/5 + 32
  3. Convertire a Kelvin: K = C + 273.15
  4. Convertire a Rankine: R = C × 9/5 + 491.67
  5. Verificare que no sea minore que il cero absoluto
