How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Identificar a temperatura em Celsius
  2. Converter para Fahrenheit: F = C × 9/5 + 32
  3. Converter para Kelvin: K = C + 273.15
  4. Converter para Rankine: R = C × 9/5 + 491.67
  5. Verificar se não é menor que o zero absoluto
