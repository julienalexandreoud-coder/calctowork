How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Identificar la temperatura en Celsius
  2. Convertir a Fahrenheit: F = C × 9/5 + 32
  3. Convertir a Kelvin: K = C + 273.15
  4. Convertir a Rankine: R = C × 9/5 + 491.67
  5. Verificar que no sea menor que el cero absoluto
