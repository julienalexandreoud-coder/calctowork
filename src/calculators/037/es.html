Calcula el número de elementos de radiador de aluminio necesarios para calefactar una habitación Usado por profesionales y estudiantes en todo el mundo.

Cómo usar esta calculadora

Sigue estos pasos para obtener resultados precisos:

  1. Calcular volumen: 20m² × 2.5m = 50m³
  2. Factor orientación norte: 50 W/m³ (más potencia por menor sol)
  3. Calcular potencia: 50m³ × 50 W/m³ = 2500 W
  4. Elementos de aluminio: 2500W ÷ 180W/elemento = 13.89 → 14 elementos

Cómo funciona

Esta calculadora utiliza fórmulas estándar para calcular resultados precisos basados en tus valores. Ingresa los datos y pulsa Calcular para obtener resultados al instante.

Fórmula

Volumen = área × altura; Potencia = volumen × factor_orientación (35-50 W/m³)
