How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calcolare volume: 20m² × 2.5m = 50m³
  2. Factor orientación norte: 50 W/m³ (más potenza per minore sol)
  3. Calcolare potenza: 50m³ × 50 W/m³ = 2500 W
  4. Elementos de aluminio: 2500W ÷ 180W/elemento = 13.89 → 14 elementos
