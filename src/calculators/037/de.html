Heizkörper Aluminium online berechnen. Einfach Werte eingeben, Ergebnis sofort erhalten. Kostenlos und ohne Registrierung.

So verwenden Sie diesen Rechner

Folgen Sie diesen Schritten für genaue Ergebnisse:

  1. Berechnen: Volumen: 20m² × 2.5m = 50m³
  2. Faktor orientación norte: 50 W/m³ (mehr Leistung durch kleiner sol)
  3. Berechnen: Leistung: 50m³ × 50 W/m³ = 2500 W
  4. Elementos von aluminio: 2500W ÷ 180W/elemento = 13.89 → 14 elementos

Wie es funktioniert

Dieser Rechner verwendet Standardformeln, um genaue Ergebnisse basierend auf Ihren Eingaben zu berechnen. Geben Sie Ihre Werte ein und klicken Sie auf Berechnen.

Eingabefelder

  • Fläche Raum
  • Höhe Decke
  • Orientación

Ergebnisse

  • Volumen
  • Vatios notwendigs
  • Elementos von Heizkörper
  • Radiadores

Formel

Volumen = Fläche × Höhe; Leistung = Volumen × factor_orientación (35-50 W/m³)
