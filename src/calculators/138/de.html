How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Dataset: 23, 17, 31, 25, 19, 28, 22 (n=7)
  2. Sort ascending: 17, 19, 22, 23, 25, 28, 31
  3. Find middle position: (7+1)÷2 = 4th value
  4. Median = 23
