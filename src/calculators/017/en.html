How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Convert thickness: 3cm = 0.03m
  2. Calculate mortar volume: 40m² x 0.03m = 1.2m³
  3. Calculate cement: 1.2m³ x 350 kg/m³ = 420 kg (8.4 bags)
  4. Calculate sand: 1.2m³ x 1.1 = 1.32m³ of sand
  5. Apply 5% waste factor if needed
