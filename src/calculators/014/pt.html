Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Calcular área: 10m × 2.6m = 26m²
  2. Calcular placas (duplo): 26m² × 2 caras ÷ 2.88m² × 1.1 (quebra) = 20 placas
  3. Calcular perfiles canal (piso/techo): 10m × 2 × 1.1 = 22 unidades
  4. Calcular montantes (cada 0.60m): (10m ÷ 0.60m + 1) × 2 = 34 montantes
  5. Calcular tornillos: 26m² × 20 ud/m² = 520 tornillos

Como funciona

Esta calculadora utiliza fórmulas padrão para calcular resultados precisos com base nos seus valores. Insira os dados e clique em Calcular para obter resultados instantâneos.

Fórmula

Área = comprimento × alto; Placas = (área × capas) ÷ 2.88 m²/placa
