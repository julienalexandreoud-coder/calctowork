Comment utiliser cette calculatrice

Suivez ces étapes simples pour obtenir des résultats précis :

  1. Square both legs: 5.5² = 30.and 12.3² = 151.29
  2. Add squares: 30.25 + 151.29 = 181.54
  3. Take square root: √181.54 = 13.4737 (hypotenuse)
  4. surface = (5.5 × 12.3) ÷ 2 = 33.825; Perimeter = 5.5 + 12.3 + 13.4737 = 31.2737
