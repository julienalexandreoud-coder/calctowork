How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Identificare potenza útil (calore entregado): 3.5 kW
  2. Identificare consumo elettrico: 1.0 kW
  3. Calcolare COP: 3.5 ÷ 1.0 = 3.50
  4. Clasificar: COP ≥5 = A+++, ≥4 = A++, ≥3.5 = A+, ≥3 = A, <3 = B
  5. Calcolare risparmio vs resistenza: (1 - 1/3.50) × 100 = 71.4%
