How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Identifizieren Leistung útil (Wärme entregado): 3.5 kW
  2. Identifizieren Verbrauch elektrisch: 1.0 kW
  3. Berechnen: COP: 3.5 ÷ 1.0 = 3.50
  4. Clasificar: COP ≥5 = ZU+++, ≥4 = ZU++, ≥3.5 = ZU+, ≥3 = ZU, <3 = B
  5. Berechnen: Ersparnis vs Widerstand: (1 - 1/3.50) × 100 = 71.4%

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Leistung útil thermisch (kW)
  • Leistung elektrisch consumida (kW)

Result Outputs

  • COP / EER
  • EER

Formula

COP = Q_útil(kW) ÷ P_eléctrica(kW) | EER = Q_frío(BTU/h) ÷ P_eléctrica(W)
