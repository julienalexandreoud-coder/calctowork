Calculez le nombre de briques de parement, mortier et coulis pour les façades. Gratuit.

How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calculer la surface brute: 8m x 2.8m = 22.4m²
  2. Soustraire les ouvertures: 22.4m² - 4m² = 18.4m² nets
  3. Calculer les briques: 18.4m² x 55/m² x 1.07 (perte 7%) = 1 083 briques
  4. Calculer le mortier: 18.4m² x 20 kg/m² = 368 kg
  5. Calculer le coulis: 18.4m² x 0.5 kg/m² = 9.2 kg

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Longueur mur
  • Hauteur mur
  • Déduire ouvertures

Result Outputs

  • Surface brute
  • Surface nette
  • Briques de parement
  • Mortier

Formula

Surface nette = (Longueur x Hauteur) - Ouvertures; Briques = Surface nette x 55/m²
