How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calculate gross area: 8m x 2.8m = 22.4m²
  2. Subtract openings: 22.4m² - 4m² = 18.4m² net
  3. Calculate bricks: 18.4m² x 55/m² x 1.07 (7% waste) = 1,083 bricks
  4. Calculate mortar: 18.4m² x 20 kg/m² = 368 kg
  5. Calculate joint grout: 18.4m² x 0.5 kg/m² = 9.2 kg

Formula
