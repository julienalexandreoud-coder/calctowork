Verblendziegel, Mörtel und Fugenmörtel online berechnen. Kostenlos.

So verwenden Sie diesen Rechner

Folgen Sie diesen Schritten für genaue Ergebnisse:

  1. Bruttofläche berechnen: 8m x 2.8m = 22.4m²
  2. Öffnungen abziehen: 22.4m² - 4m² = 18.4m² netto
  3. Ziegel berechnen: 18.4m² x 55/m² x 1.07 (Verschnitt 7%) = 1.083 Ziegel
  4. Mörtel berechnen: 18.4m² x 20 kg/m² = 368 kg
  5. Fugenmörtel berechnen: 18.4m² x 0.5 kg/m² = 9.2 kg

Wie es funktioniert

Dieser Rechner verwendet Standardformeln, um genaue Ergebnisse basierend auf Ihren Eingaben zu berechnen. Geben Sie Ihre Werte ein und klicken Sie auf Berechnen.
