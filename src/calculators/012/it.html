How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calcolare superficie lorda: 8m x 2.8m = 22.4m²
  2. Sottrarre aperture: 22.4m² - 4m² = 18.4m² netti
  3. Calcolare mattoni: 18.4m² x 55/m² x 1.07 = 1.083 mattoni
  4. Calcolare malta: 18.4m² x 20 kg/m² = 368 kg
  5. Calcolare boiacca: 18.4m² x 0.5 kg/m² = 9.2 kg

How It Works

Input Fields

Result Outputs

  • Superficie lorda (m²)
  • Superficie netta (m²)
  • Mattoni facciavista

Formula

Area netta = (Lunghezza x Altezza) - Aperture; Mattoni = Area netta x 55/m²
