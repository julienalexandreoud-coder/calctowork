Calcule quantos tijolos aparentes, argamassa e rejunte para fachadas. Inclui fator de perda.

How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calcular área bruta: 8m x 2.8m = 22.4m²
  2. Subtrair vãos: 22.4m² - 4m² = 18.4m² líquidos
  3. Calcular tijolos: 18.4m² x 55/m² x 1.07 (perda 7%) = 1.083 tijolos
  4. Calcular argamassa: 18.4m² x 20 kg/m² = 368 kg
  5. Calcular rejunte: 18.4m² x 0.5 kg/m² = 9.2 kg

Formula

Área líquida = (Comprimento x Altura) - Vãos; Tijolos = Área líquida x 55/m²
