Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Calcular volume: 25m² × 2.6m = 65m³
  2. Factor orientación sur + isolamento bueno: 50 BTU/m³
  3. Calcular BTU: 65m³ × 50 = 3250 BTU/h
  4. Para m² método rápido: 25m² × 600 BTU/m² = 15000 BTU/h
  5. Convertir a toneladas: 15000 ÷ 12000 = 1.25 toneladas
