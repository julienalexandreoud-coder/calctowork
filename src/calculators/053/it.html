How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calcolare volume: 25m² × 2.6m = 65m³
  2. Factor orientación sur + isolamento bueno: 50 BTU/m³
  3. Calcolare BTU: 65m³ × 50 = 3250 BTU/h
  4. Para m² método veloce: 25m² × 600 BTU/m² = 15000 BTU/h
  5. Convertire a toneladas: 15000 ÷ 12000 = 1.25 toneladas
