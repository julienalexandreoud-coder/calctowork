How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calculer volume: 25m² × 2.6m = 65m³
  2. Factor orientación sur + isolation bueno: 50 BTU/m³
  3. Calculer BTU: 65m³ × 50 = 3250 BTU/h
  4. Pour m² método rapide: 25m² × 600 BTU/m² = 15000 BTU/h
  5. Convertir a toneladas: ÷ 12000 = 1.toneladas
