Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Introduz comprimento linea m: 10 m
  2. Introduz potência eq kw: 3.5 kW
  3. Introduz tipo gas: R32
  4. Pulsa calcular para obter resultados
  5. Interpreta resultado: {carga_base_kg}

Como funciona

Esta calculadora utiliza fórmulas padrão para calcular resultados precisos com base nos seus valores. Insira os dados e clique em Calcular para obter resultados instantâneos.

Campos de entrada

  • Comprimento linha frigorífica
  • Potência equipamento (kW)
  • Tipo de gás

Resultados

  • Carga base equipamento
  • Carga extra linha longa
  • Carga total gás

Fórmula

Carga_total = Base(0,04 kg/kW × P) + Extra((L-5) × fator). R32: 0,02 kg/m, R410A: 0,025 kg/m
