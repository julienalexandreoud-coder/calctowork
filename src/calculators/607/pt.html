Calcule CV (%) = (Standard deviation / Mean) × 100. Exemplo: Mean 80, SD 12 = CV = 15%. Calculadora gratuita e precisa, sem registo.

How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Obter a desviación padrão
  2. Obter a media
  3. Dividir desviación entre media
  4. Multiplicar por 100 para porcentaje

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Desvio padrão
  • Média amostral

Result Outputs

  • CV (%)

Formula

CV = (desviación padrão / media) × 100%
