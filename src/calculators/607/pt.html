Calcule CV (%) = (Standard deviation / Mean) × 100. Exemplo: Mean 80, SD 12 = CV = 15%. Calculadora gratuita e precisa, sem registo.

Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Obter a desviación padrão
  2. Obter a media
  3. Dividir desviación entre media
  4. Multiplicar por 100 para porcentaje

Como funciona

Esta calculadora utiliza fórmulas padrão para calcular resultados precisos com base nos seus valores. Insira os dados e clique em Calcular para obter resultados instantâneos.

Campos de entrada

  • Média amostral

Resultados

Fórmula

CV = (desviación padrão / media) × 100%
