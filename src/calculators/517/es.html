How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Obtener velocidad de propagación del medio
  2. Calcular tiempo de ida
  3. Multiplicar por 2 para RTT

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Distancia
  • Medio

Result Outputs

  • Latencia ms
  • RTT teórico ms

Formula

Latencia = (Distancia / Velocidad señal) × 2
