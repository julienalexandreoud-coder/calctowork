How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Identificar quais dois lados são conhecidos
  2. Aplicar o teorema de Pitágoras para encontrar o lado faltante
  3. Calcular a área: (cateto_a × cateto_b) / 2
  4. Calcular o perímetro: soma dos três lados
  5. Verificar se os valores formam um triângulo válido
