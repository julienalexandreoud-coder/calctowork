Calcola il numero di blocchi di calcestruzzo e malta per pareti. Include fattore di scarto.

How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calcolare superficie lorda: 10m x 3m = 30m²
  2. Dedurre aperture: 30m² - 3m² = 27m² netti
  3. Calcolare blocchi: 27m² x 12.5/m² x 1.05 = 354 blocchi
  4. Calcolare malta: 27m² x 12 kg/m² = 324 kg

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Lunghezza parete (m)
  • Altezza parete (m)
  • Dedurre aperture (m²)

Result Outputs

  • Superficie netta (m²)
  • Blocchi calcestruzzo
  • Malta (kg)

Formula

Area netta = (Lunghezza x Altezza) - Aperture; Blocchi = Area netta x 12.5/m²
