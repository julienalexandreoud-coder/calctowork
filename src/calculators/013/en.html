How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calculate gross area: 10m x 3m = 30m²
  2. Deduct openings: 30m² - 3m² = 27m² net
  3. Calculate blocks: 27m² x 12.5/m² x 1.05 (waste) = 354 blocks
  4. Calculate mortar: 27m² x 12 kg/m² = 324 kg

How It Works

Input Fields

  • Wall length (m)
  • Wall height (m)
  • Deduct openings (m²)

Result Outputs

  • Net area (m²)
  • Concrete blocks
  • Mortar (kg)

Formula

Net area = (Length x Height) - Openings; Blocks = Net area x 12.5 per m²
