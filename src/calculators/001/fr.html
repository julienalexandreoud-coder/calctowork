Calculez la quantité de ciment, sable et gravier pour le béton de masse. Outil gratuit pour chantiers. Inclut le facteur de perte pour les commandes.

How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Multiplier 5m × 3m = 15m² de surface
  2. Multiplier 15m² × 0.20m = 3m³ de volume de béton
  3. Calculer les matériaux: 3m³ × 7 sacs/m³ = 21 sacs de ciment
  4. Calculer le sable: 3m³ × 0.65 = 1.95m³ de sable
  5. Calculer le gravier: 3m³ × 0.90 = 2.70m³ de gravier

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Longueur
  • Largeur
  • Épaisseur / Hauteur

Result Outputs

  • Volume de béton
  • Sacs de ciment (50 kg)
  • Sable
  • Gravier

Formula

Volume = Longueur x Largeur x Epaisseur
