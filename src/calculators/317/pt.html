Calcule Total return (%) = ((Final price + Dividends - Initial price) / Initial price) × 100. Exemplo: Buy at €50, sell at €62, dividends €1.50 = Return = 27%..

How to Use This Calculator

  2. Add dividends to capital gain
  3. Divide by purchase price
  4. Convert to percentage
