Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Introduz comprimento m: 10 m
  2. Introduz separacion m: 2 m
  3. Introduz altura m: 3 m
  4. Pulsa calcular para obter resultados
  5. Interpreta resultado: {postes_ud}

Como funciona

Esta calculadora utiliza fórmulas padrão para calcular resultados precisos com base nos seus valores. Insira os dados e clique em Calcular para obter resultados instantâneos.

Campos de entrada

  • Comprimento total da cerca (m)
  • Espaçamento entre postes (m)
  • Altura da cerca (m)

Resultados

  • Postes
  • Painéis/vãos
  • Concreto para dados (L)
  • Parafusos/fixações

Fórmula

Postes = comprimento ÷ Separación + 1
