Calcula el IVA y la retención de IRPF de tus facturas de construcción. Introduce la base imponible, tipo de IVA y % de retención. Gratis.

Cómo usar esta calculadora

Sigue estos pasos para obtener resultados precisos:

  1. Introduce la base imponible de la factura (mano de obra, materiales o ambos según el caso)
  2. Aplica el tipo de IVA: 21% general para construcción (10% reducido para rehabilitación de vivienda habitual)
  3. Calcula la retención de IRPF: 15% para autónomos en alta durante los primeros 3 años, 7% a partir del 4Âº año
  4. Para facturas entre empresas (constructor subcontratista): se aplica inversión del sujeto pasivo (sin IVA en factura)
  5. El líquido a percibir es la base menos la retención de IRPF (la retención la ingresa el pagador a Hacienda)
  6. El IVA repercutido se liquida trimestralmente (modelo 303) y la retención se deduce en la declaración anual de IRPF

Cómo funciona

Esta calculadora utiliza fórmulas estándar para calcular resultados precisos basados en tus valores. Ingresa los datos y pulsa Calcular para obtener resultados al instante.
