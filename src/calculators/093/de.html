So verwenden Sie diesen Rechner

Folgen Sie diesen Schritten für genaue Ergebnisse:

  1. Geben Sie die base imponible von die Rechnung (mano von Baustelle, Materialien oder ambos según der caso)
  2. Aplica der tipo von IVA: 21% general für construcción (10% reducido für rehabilitación von Wohnung habitual)
  3. Berechnen Sie die retención von IRPF: 15% für autónomos in alta während die primeros 3 años, 7% zu partir des 4Âº añoder
  4. Für facturas zwischen empresas (constructor subcontratista): se aplica inversión des sujeto pasivo (ohne IVA in Rechnung)
  5. Der líquido zu percibir ist die base weniger die retención von IRPF (die retención die geben Sie ein der pagador zu Hacienda)
  6. Der IVA repercutido se liquida trimestralmente (modelo 303) und die retención se deduce in die declaración jährlich von IRPF
