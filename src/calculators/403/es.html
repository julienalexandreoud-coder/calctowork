How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Multiplica peso en kg por 0.033 para agua base
  2. Calcula agua extra: minutos ejercicio × 0.5 ÷ 60
  3. Suma agua base + agua extra para total diario
  4. Divide entre 0.25 para obtener vasos de 250ml

How It Works

This calculator helps you estimate the materials and quantities needed for your project.

Input Fields

  • Peso (kg)
  • Ejercicio diario (min)

Result Outputs

  • Agua total (L/día)
  • Agua base
  • Extra por ejercicio
  • Vasos de 250ml

Formula

Agua (L) = peso (kg) × 0.033 + ejercicio (min) × 0.5 ÷ 60
