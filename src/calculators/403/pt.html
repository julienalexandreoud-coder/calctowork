Quanta água deve beber por dia? Calcule a sua ingestão diária de água com base no peso e exercício. Com intervalos de referencia e contexto de saude.

Como usar esta calculadora

Siga estes passos simples para obter resultados precisos:

  1. Multiplica peso em kg por 0.033 para água base
  2. Calcula água extra: minutos ejercicio × 0.5 ÷ 60
  3. Suma água base + água extra para total diario
  4. Divide entre 0.25 para obter vasos de 250ml

Como funciona

Esta calculadora utiliza fórmulas padrão para calcular resultados precisos com base nos seus valores. Insira os dados e clique em Calcular para obter resultados instantâneos.

Campos de entrada

  • Peso (kg)
  • Exercício diário (min)

Resultados

  • Água total (L/dia)
  • Ingestão base
  • Extra exercício
  • Copos de 250ml

Fórmula

Agua (L) = peso (kg) × 0.033 + ejercicio (min) × 0.5 ÷ 60
