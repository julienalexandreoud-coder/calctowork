Calcula metros de tubería de cobre o PEX, codos, tes y llaves de escuadra para fontanería Usado por profesionales y estudiantes en todo el mundo.

Cómo usar esta calculadora

Sigue estos pasos para obtener resultados precisos:

  1. Calcular tubería con merma: 40m × 1.10 = 44m
  2. Estimar accesorios: 6 puntos × 3 = 18 accesorios (válvulas, codos, tes)
  3. Calcular aislamiento: 40m × 0.80 = 32m (solo agua caliente)
  4. Diferenciar diámetros: 1/2" para grifos, 3/4" para duchas, 1" para principales

Cómo funciona

Esta calculadora utiliza fórmulas estándar para calcular resultados precisos basados en tus valores. Ingresa los datos y pulsa Calcular para obtener resultados al instante.

Campos de entrada

  • Longitud total
  • Diámetro (mm)
  • Puntos de suministro

Resultados

  • Longitud (+10%)
  • Codos
  • Tes
  • Llaves de escuadra

Fórmula

Tubería = longitud × 1.10 (merma); Accesorios = puntos × 3
