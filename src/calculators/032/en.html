How to Use This Calculator

Follow these simple steps to get accurate results:

  1. Calculate pipe with 10% waste: 40m x 1.10 = 44m
  2. Estimate fittings: 6 points x 3 = 18 fittings (valves, elbows, tees)
  3. Calculate insulation: 40m x 0.80 = 32m (hot water only)
  4. Size diameters: 1/2" for faucets, 3/4" for showers, 1" for mains

How It Works
