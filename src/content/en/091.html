What Is a Simple Project Planning Calculator?

A simple project planning calculator is an essential tool for contractors, project managers, and tradespeople that helps create realistic project schedules by calculating task durations, dependencies, and completion dates. This calculator transforms project scope into actionable timelines, enabling better resource allocation and client communication.

Project planning is the foundation of successful construction delivery. Poor planning leads to missed deadlines, cost overruns, client disputes, and damaged reputations. A systematic approach to scheduling — breaking work into tasks, estimating durations, identifying dependencies, and tracking progress — separates profitable contractors from those struggling to complete projects on time and budget.

This calculator processes variables including task lists, durations, dependencies, start dates, and working calendars to deliver project timelines, critical paths, and completion dates. Whether planning a bathroom renovation or multi-month construction project, systematic planning ensures realistic commitments and successful delivery.

Formulas Explained: Project Scheduling Calculations

Project scheduling uses Critical Path Method (CPM) principles to calculate timelines and identify schedule constraints.

Task Duration Calculation:

Duration (days) = Quantity ÷ (Crew Size × Productivity Rate × Productive Hours/Day)

Early Start (ES) = Maximum Early Finish of all predecessor tasks

Early Finish (EF) = Early Start + Duration - 1

Late Start (LS) = Late Finish - Duration + 1

Late Finish (LF) = Minimum Late Start of all successor tasks

Float (Slack) = Late Start - Early Start (or Late Finish - Early Finish)

Task Categories by Phase:

  • Pre-construction: Permits, design finalization, material ordering, site setup. Duration: 1-8 weeks depending on complexity.
  • Substructure: Demolition, excavation, foundations, drainage. Duration: 1-4 weeks for residential.
  • Superstructure: Walls, floors, roof structure. Duration: 2-6 weeks for residential.
  • First fix: Plumbing, electrical, HVAC rough-in. Duration: 1-3 weeks.
  • Second fix: Plastering, flooring, painting, fixtures. Duration: 2-5 weeks.
  • Finishing: Final connections, cleaning, snagging, handover. Duration: 1-2 weeks.

Dependency Types:

  • Finish-to-Start (FS): Task B cannot start until Task A finishes — most common.
  • Start-to-Start (SS): Task B cannot start until Task A starts — parallel work with lag.
  • Finish-to-Finish (FF): Task B cannot finish until Task A finishes — synchronized completion.
  • Start-to-Finish (SF): Rare — Task B cannot finish until Task A starts.

Example Calculation: Simple 3-task project:

Task A: Demolition — 3 days, no predecessors

Task B: First fix — 5 days, depends on Task A (FS)

Task C: Second fix — 7 days, depends on Task B (FS)

Start date: Monday, Week 1

Task A: ES=Day 1, EF=Day 3

Task B: ES=Day 4, EF=Day 8

Task C: ES=Day 9, EF=Day 15

Project completion: Day 15 (end of Week 3)

Critical path: A → B → C (all tasks have zero float)

Step-by-Step Guide

  1. Break Project into Individual Tasks: Create Work Breakdown Structure (WBS) — hierarchical list of all work required. Level 1: Project name. Level 2: Major phases (Demolition, Structure, First Fix, Second Fix, Finishing). Level 3: Specific tasks within each phase. Example: 3-bed renovation. Level 2: Demolition. Level 3: Strip kitchen (2 days), Strip bathroom (1 day), Remove flooring (2 days), Remove walls (3 days). Level 2: First Fix. Level 3: Plumbing rough-in (3 days), Electrical first fix (4 days), HVAC ducts (2 days). Level 2: Second Fix. Level 3: Plastering (4 days), Flooring (3 days), Painting (5 days), Kitchen fit (3 days), Bathroom fit (2 days). Total tasks: 15-25 for typical residential project. Too few (under 10): tasks too large, hard to track. Too many (over 50): micromanagement, schedule too complex. Guideline: tasks should be 1-10 days duration, assignable to one trade, measurable completion.
  2. Estimate Duration for Each Task: Use productivity rates from historical data or industry standards. Formula: Duration = Quantity ÷ Daily Output. Example task estimates: Demolition — kitchen strip: 25m² ÷ 12m²/day = 2.1 days → 2 days. First fix plumbing — 18 points ÷ 6 points/day = 3 days. Plastering — 180m² ÷ 45m²/day (2 plasterers) = 4 days. Painting — 280m² ÷ 55m²/day (2 painters) = 5.1 days → 6 days. Add contingency: 10-15% for known risks (weather for external work, access constraints, complex details). Don't add contingency to every task — add to phase totals or project total. Rounding: always round up (2.1 days → 3 days). Better to finish early than late. Working days: exclude weekends, holidays. 5-day task starting Friday finishes following Friday (not Wednesday — weekend doesn't count).
  3. Identify Task Dependencies: Determine which tasks must precede others. Logical dependencies: Foundations before walls. Walls before roof. First fix before plastering. Plastering before painting. Flooring before final connections. Resource dependencies: Same crew can't do two tasks simultaneously. Preferential dependencies: Preferred sequence (paint before floor fitting to avoid drips on new floor, but can be reversed with protection). External dependencies: Inspections before covering work. Material delivery before installation. Client decisions before ordering. Document: create dependency list or network diagram. Example dependencies: Task A (Demolition) → Task B (First fix) → Task C (Plastering) → Task D (Painting) → Task E (Flooring) → Task F (Final connections). Parallel tasks: Plumbing first fix and Electrical first fix can run simultaneously (different trades). Sequence: Start both after demolition, both must finish before plastering.
  4. Calculate Critical Path and Float: Critical path = longest path through project network = minimum project duration. Tasks on critical path have zero float — any delay delays project completion. Non-critical tasks have float — can be delayed without affecting completion. Forward pass (calculate Early Start/Early Finish): Start at project beginning. ES of first task = Day 1. EF = ES + Duration - 1. ES of next task = EF of predecessor + 1 (for FS dependency). If multiple predecessors, ES = maximum EF of all predecessors. Continue to project end. Backward pass (calculate Late Start/Late Finish): Start at project end. LF of last task = EF of last task (or required completion date). LS = LF - Duration + 1. LF of predecessor = LS of successor - 1. If multiple successors, LF = minimum LS of all successors. Continue to project start. Float calculation: Float = LS - ES (or LF - EF). Zero float = critical path. Positive float = can be delayed by that amount. Example: Task A (3 days) → Task B (5 days) → Task C (4 days). Parallel: Task D (6 days) starts with Task A, no dependency on B or C. Path 1: A-B-C = 12 days. Path 2: D = 6 days. Critical path: A-B-C (12 days). Task D float: 12 - 6 = 6 days. Task D can start up to 6 days late without affecting project.
  5. Create Project Schedule and Gantt Chart: Visual representation of schedule — tasks as bars on timeline. Tools: spreadsheet (simple), project management software (MS Project, Primavera, Smartsheet), construction-specific (Procore, Buildertrend), free tools (GanttProject, ProjectLibre). Gantt chart elements: Task list (left side), Timeline (top), Bars (task duration), Dependencies (arrows between bars), Milestones (diamonds — key dates), Today line (progress tracking). Example Gantt structure: Week 1: Mon-Tue Demolition, Wed-Fri First fix. Week 2: Mon-Thu Plastering (drying time), Fri Prep for painting. Week 3: Mon-Fri Painting. Week 4: Mon-Wed Flooring, Thu-Fri Kitchen fit. Week 5: Mon-Tue Bathroom fit, Wed Final connections, Thu-Snagging, Fri-Handover. Milestones: Week 1 Fri — First fix complete. Week 2 Thu — Plastering complete. Week 4 Fri — Kitchen complete. Week 5 Fri — Project complete. Update: weekly progress tracking, actual vs. planned, forecast completion, communicate changes to stakeholders.
  6. Track Progress and Update Schedule: Schedule is living document — update as work progresses. Weekly process: Record actual start/finish dates. Calculate percent complete for ongoing tasks. Update remaining duration (not just percent — tasks often slow down near end). Recalculate critical path (may change if tasks delayed). Identify new delays or risks. Communicate changes to team and client. Example: Week 2 update. Planned: Plastering complete Thursday. Actual: Plastering 60% complete, remaining 2 days. Forecast: Complete Monday Week 3 (3 days late). Impact: Painting delayed 3 days, project completion delayed 3 days. Recovery: Add weekend work (cost +15%), or add painter (cost +20%), or accept delay (client discussion). Decision: client needs property for event — weekend work approved. Updated schedule: Painting Wed-Fri Week 3 + Sat. Project completes on time. Cost: €800 weekend premium. Lesson: early identification allows recovery options. Late identification (wait until Friday) = no recovery possible.

Real-World Examples

Example 1: Bathroom Renovation (2 Weeks) — Complete bathroom remodel. Tasks: Day 1-2: Strip out (remove sanitaryware, tiles, flooring). Day 3: Plumbing first fix (move pipes, install waste). Day 4: Electrical first fix (new circuits, extractor). Day 5: Plastering/boarding (walls, ceiling). Day 6: Drying time (no work, essential). Day 7-8: Tiling walls (18m²). Day 9: Tiling floor (4m²), grouting. Day 10: Painting, silicone. Day 11: Fit sanitaryware (toilet, basin, shower). Day 12: Fit accessories, final connections. Day 13: Snagging, clean. Day 14: Client walkthrough, handover. Critical path: All tasks sequential (single bathroom, one trade at a time). Duration: 14 calendar days (10 working days + weekend + drying day). Start: Monday 1st. Finish: Wednesday 17th (includes weekend). Dependencies: Each task depends on previous. Risk: if tiles delayed, entire schedule slips. Mitigation: order tiles before starting, verify delivery.

Example 2: Kitchen Renovation (3 Weeks) — Full kitchen replacement. Tasks: Week 1: Mon-Tue Strip old kitchen. Wed-Thu Plumbing first fix. Fri Electrical first fix. Week 2: Mon Plastering/making good. Tue-Thu Drying/curing (no work). Fri Kitchen unit installation. Week 3: Mon-Tue Worktop templating (wait for fabricator). Wed-Thu Worktop installation. Fri Appliance fit, final connections, handover. Duration: 15 working days over 3 weeks. Critical path: Strip → First fix → Plaster → Dry → Units → Template → Worktop → Complete. Parallel tasks: Flooring can be done during drying days (not on critical path). Float: Flooring has 3 days float (can be done Tue-Thu Week 2 or Mon-Tue Week 3). Risk: worktop templating depends on fabricator availability — external dependency. Mitigation: book fabricator before starting project, confirm appointment.

Example 3: Single-Storey Extension (10 Weeks) — Rear extension 5m × 4m. Phase 1 (Week 1-2): Site setup, demolition, excavation, foundations. Phase 2 (Week 3-4): Blockwork to DPC, DPM, floor slab. Phase 3 (Week 5-6): Blockwork to roof, lintels, roof structure. Phase 4 (Week 7): Roof covering, windows/doors installed (weathertight). Phase 5 (Week 8): First fix (plumbing, electrical, plaster prep). Phase 6 (Week 9): Plastering, drying. Phase 7 (Week 10): Painting, flooring, final connections, handover. Critical path: Foundations → Walls → Roof → Weathertight → First fix → Plaster → Paint → Complete. Parallel tasks: Window ordering (Week 1, 4-week lead, installed Week 7). Kitchen ordering (Week 3, 6-week lead, fitted Week 11 — post-handover). Duration: 10 weeks working (50 working days). Calendar: 12 weeks including weekends. Dependencies: Can't build walls without foundations. Can't plaster before weathertight (moisture damage). Can't paint before plaster dry. Float: Landscaping has 4 weeks float (can start Week 6 or Week 10).

Example 4: Full House Renovation (16 Weeks) — 3-bed house, full renovation. Phase 1 (Week 1-2): Strip out, demolition, asbestos removal (if found). Phase 2 (Week 3-5): Structural work (RSJ installation, wall removals, new openings). Phase 3 (Week 6-8): First fix (plumbing, electrical, HVAC throughout). Phase 4 (Week 9-10): Plastering throughout, drying time. Phase 5 (Week 11-13): Second fix (painting, flooring, tiling). Phase 6 (Week 14-15): Kitchen, bathrooms, final connections. Phase 7 (Week 16): Snagging, cleaning, handover. Critical path: Demolition → Structural → First fix → Plaster → Second fix → Kitchens/Bathrooms → Handover. Parallel tasks: Windows (order Week 1, install Week 5). Staircase (template Week 4, install Week 10). Flooring (can start Week 11 in phases). Duration: 16 weeks (80 working days). Resource leveling: 2 plasterers needed Week 9-10 (whole house). If only 1 plasterer available, duration extends to Week 11. Constraint identified early — book plasterers or adjust schedule.

Example 5: Commercial Office Fit-Out (8 Weeks) — 200m² office space. Week 1: Site setup, demolition, surveys. Week 2-3: Partitions (metal stud, boarding one side). Week 4: First fix (services in walls). Week 5: Boarding other side, ceiling grids. Week 6: Ceiling tiles, lighting installation. Week 7: Painting, flooring. Week 8: Furniture installation, IT setup, handover. Critical path: Partitions → First fix → Boarding → Ceilings → Painting → Flooring → Furniture. Parallel tasks: IT cabling (Week 4-5, parallel with first fix). Fire system (Week 5-6, parallel with ceilings). Duration: 8 weeks. Constraints: Work only outside business hours (occupied building). Effective working time: 4 hours/day (6pm-10pm). Duration doubles to 16 weeks calendar. Solution: negotiate partial floor closures, daytime work in sections. Revised: 10 weeks calendar with mixed working hours.

Example 6: Multi-Unit Development Phasing (24 Weeks) — 6 apartments, phased handover. Strategy: Complete 2 apartments every 8 weeks. Phase 1 (Week 1-8): Units 1-2 full renovation. Phase 2 (Week 5-16): Units 3-4 (starts Week 5 while Phase 1 in progress). Phase 3 (Week 9-24): Units 5-6 (starts Week 9). Overlap: First fix crew moves Unit 1→3→5. Plasterers move Unit 2→4→6. Critical path per unit: 8 weeks. Overall project: 24 weeks (not 48 weeks if sequential). Resource flow: Demolition (2 weeks per pair) → First fix (2 weeks) → Plaster (1 week) → Second fix (3 weeks). Crews move between units continuously. Benefit: cash flow (sales/rentals at Week 8, 16, 24), workforce continuity (no gaps), learning curve (Unit 1 slower than Unit 6). Risk: delay in Phase 1 cascades to Phases 2-3. Mitigation: buffer between phases, additional resources if behind.

Common Mistakes

Mistake 1: Not Including Drying/Curing Time — Many construction tasks require drying/curing before next task can start. Plaster needs 2-4 days before painting. Concrete needs 7-28 days before loading. Adhesives need 24-48 hours before loading. Paint needs 4-24 hours between coats. Example: plastering Friday, painting scheduled Monday. Reality: plaster needs 3 days dry. Painting Monday = paint failure (blistering, poor adhesion). Rework: strip paint, re-plaster, wait, re-paint. Delay: 5 days. Cost: €1,200 rework + €800 delay. Prevention: include drying time in schedule explicitly. Task: "Plaster drying" — 3 days duration, no labor, dependency between plastering and painting. Schedule reflects reality, no surprises.

Mistake 2: Assuming Resources Are Always Available — Schedule assumes trades, materials, equipment available when needed. Reality: plasterers booked 3 weeks ahead, tiles on backorder 6 weeks, scaffolding delayed. Example: schedule Week 4 plastering. Week 3: plasterer unavailable (another project overran). Week 4: idle workforce, project slips 2 weeks. Prevention: book key trades before project starts (get commitment dates). Order long-lead items immediately (windows, kitchens, special tiles). Confirm availability 2 weeks before needed. Contingency: identify backup suppliers/trades. Schedule flexibility: build in 5-10% float for resource delays.

Mistake 3: Not Accounting for Inspections and Approvals — Building control inspections, client approvals, utility company visits all take time and are on critical path. Example: first fix complete, ready for plaster. Building control inspection required before covering. Inspector available: Thursday. Today: Monday. Wait: 3 days. Plastering delayed 3 days. Project slips. Prevention: identify all required inspections (foundations, first fix, drainage, completion). Book inspections 1 week ahead (some councils 2 weeks). Include inspection tasks in schedule with duration (0 days for inspection, but dependency). Buffer: 2-3 days between task complete and inspection (inspector may reschedule). Client approvals: include in schedule (kitchen layout approval, tile selection, paint colors). Duration: 3-5 days for client decision. Dependency: ordering cannot start until approved.

Mistake 4: Overlapping Trades Without Coordination — Multiple trades working same area simultaneously causes conflicts, damage, rework. Example: painters working ceiling while floor layers working below. Paint drips on new flooring. Flooring damaged, repainting needed. Delay: 2 days. Cost: €600. Prevention: sequence trades through areas, not simultaneous. Zone scheduling: Area 1 — painters Week 1, floorers Week 2. Area 2 — painters Week 2, floorers Week 3. Or: painters work top-to-bottom (ceilings, walls, then floorers, then final paint). Coordination meetings: weekly trade coordination, agree who works where when. Protection: if overlap unavoidable, protect completed work (floor coverings, masking).

Pro Tips

Tip 1: Use Look-Ahead Schedules for Weekly Planning — Master schedule (overall project) too detailed for daily use. Create 2-week look-ahead schedules every week. Process: Every Friday, create detailed schedule for next 2 weeks. Tasks broken into daily assignments. Resources confirmed (who, what, when). Materials verified (on-site or delivery scheduled). Dependencies checked (prerequisites complete). Distributed: to all trades Thursday/Friday for following week. Benefits: trades know exactly what's expected, materials ready, no "what should I do today?" downtime, issues identified before they cause delays. Example: Week 5 look-ahead. Monday: Electricians — first fix rooms 1-3. Plumbers — first fix rooms 4-6. Tuesday: Electricians — rooms 4-6. Plumbers — rooms 1-3. Wednesday: Both — complete outstanding, inspection prep. Thursday: Building control inspection. Friday: Plasterers start (if inspection passed). Materials: all first fix materials on-site (verified Wednesday). Issues: windows not delivered — impact on plastering. Action: expedite delivery or adjust schedule (plaster other areas first).

Tip 2: Build in Buffers at Key Milestones — Strategic buffers protect critical milestones without padding every task. Types: Project buffer — at end of project (5-10% of duration). Absorbs delays without affecting handover date. Milestone buffers — after key phases (before client events, before follow-on trades). Feeding buffers — where non-critical paths merge with critical path. Example: 16-week project. Project buffer: 1 week (Week 17). Milestone buffers: Week 8 (weathertight — before winter), Week 12 (plaster complete — before painting). Feeding buffers: Window delivery (2 weeks buffer — if delayed, doesn't affect critical path until buffer consumed). Usage: buffer is not "free time" — only used if delays occur. Track buffer consumption: if 50% buffer used at 30% project complete, high risk of delay. Action: accelerate work, add resources. Communication: buffers not visible to client (commit to buffered date), internal target is unbuffered date.

Tip 3: Identify and Monitor Critical Path Religiously — Critical path tasks determine project duration — focus management attention here. Weekly review: Is critical path still accurate? (may change if tasks delayed). Are critical tasks on track? (daily monitoring). What risks could affect critical path? (proactive mitigation). Can critical path be shortened? (fast-tracking, crashing). Example: Week 4 review. Critical path: Foundations → Walls → Roof → First fix → Plaster → Paint. Status: Foundations complete on time. Walls 2 days behind (brick delay). Roof starts in 3 days — will be delayed 2 days. Impact: project completion delayed 2 days. Recovery options: Add bricklayer (cost +€400, recover 2 days). Work weekend (cost +€600, recover 2 days). Accept delay (cost €0, client impact). Decision: client has no flexibility (sale completion). Weekend work approved. Critical path maintained. Lesson: weekly critical path review enables proactive recovery. Monthly review = too late.

Tip 4: Communicate Schedule Changes Immediately — Delays happen — early communication preserves relationships and enables recovery. Process: Identify delay (daily tracking). Quantify impact (how many days, which milestones affected). Identify cause (internal, client, external, weather). Develop recovery options (accelerate, resequence, accept). Communicate to client same day (don't wait for weekly meeting). Document: email confirmation, updated schedule attached. Example: Tile delivery delayed 1 week. Impact: flooring delayed 1 week, painting delayed 1 week, completion delayed 1 week. Recovery: tile from alternative supplier (3 days longer lead, but available), or accept delay, or partial completion (handover without tiles). Communication: "Tile delivery delayed due to supplier issue. Impact: 1 week delay to completion. Options: 1) Alternative tiles available in 3 days (same spec, +€200). 2) Accept 1-week delay (no cost). 3) Handover Week 15, tiles fitted Week 16 (rent-free period). Please advise preference by EOD." Client chooses option 1. Delay avoided, minimal cost, client appreciates transparency.

Tip 5: Use Schedule for Cash Flow Planning — Schedule drives cash flow — milestone dates determine payment dates, material ordering determines outflows. Process: Link payments to milestones (deposit, Week 4, Week 8, completion). Map material payments to order dates (windows 50% on order, 50% on delivery). Calculate weekly cash flow (inflows vs. outflows). Identify shortfalls (arrange overdraft or adjust payment terms). Example: 12-week project, €60,000 value. Payments: 20% deposit (€12,000 Week 0), 25% Week 4 (€15,000), 25% Week 8 (€15,000), 25% Week 12 (€15,000), 5% retention (€3,000 Week 16). Outflows: Week 1 materials €8,000, Week 4 materials €10,000, Week 8 materials €8,000, Labor €4,000/week. Cash flow: Week 0 +€12,000. Week 1 -€12,000 (€8k materials + €4k labor). Week 2-3 -€4,000/week. Week 4 +€11,000 (€15k payment - €10k materials - €4k labor). Analysis: positive cash flow overall, but Weeks 2-3 negative. Arrange: €10,000 overdraft or negotiate material payment terms (30 days). Schedule-linked cash flow prevents surprises.

FAQs

Detail level depends on project size and complexity. Small projects (under €20,000, under 4 weeks): 10-15 tasks, weekly granularity. Medium projects (€20,000-100,000, 4-12 weeks): 20-40 tasks, 2-3 day granularity. Large projects (€100,000+, 12+ weeks): 50-100+ tasks, daily granularity for critical path, weekly for non-critical. Rule of thumb: tasks should be small enough to track progress meaningfully (can say "50% complete" with confidence), but not so small that tracking becomes burdensome. 1-10 day tasks ideal. Tasks over 2 weeks should be broken down (hard to track progress, risk hidden until too late). Too detailed (1-hour tasks): micromanagement, schedule maintenance overwhelms value, false precision. Too coarse (month-long tasks): no early warning, all tasks "on track" until suddenly "2 weeks behind." Sweet spot: tasks assignable to one trade, measurable completion, 1-10 days duration.

How do I handle client-requested changes mid-project?

Change orders affect schedule — quantify impact before committing. Process: Client requests change. Assess: what tasks affected, additional duration, knock-on effects. Calculate: additional labor, materials, time. Communicate: "This change adds €X cost and Y days to schedule. New completion date: [date]. Approve?" Document: written change order, signed by client, updated schedule attached. Example: client wants additional socket outlets Week 6. Assessment: Electrical first fix complete, walls closed. Requires: cut into plaster, add outlets, make good, repaint. Impact: 2 days electrical, 1 day plastering, 2 days painting, 1 day drying. Total: 6 days delay. Cost: €850 labor + materials. Communication: "Additional outlets require opening walls. Impact: €850 cost, 6 days delay to completion. New handover: [date + 6 days]. Alternative: surface-mounted outlets (no delay, €400, different appearance). Please advise." Client decides: proceed, alternative, or abandon change. Key: client makes informed decision with full impact visibility.

Recovery options depend on how far behind and why. Assessment: How many days behind? Which tasks affected? Is critical path impacted? Why behind (internal, external, client)? Recovery strategies: Acceleration — add resources (more workers, overtime, weekend work). Cost: 15-50% premium. Fast-tracking — overlap tasks normally sequential (paint before plaster fully dry with dehumidifiers). Risk: quality issues. Resequencing — change task order (do external work while waiting for internal materials). Scope reduction — defer non-critical items (landscaping, decorative features to Phase 2). Client negotiation — agree revised completion date, possibly with compensation (rent-free period, discount). Example: 5 days behind at Week 6 of 12-week project. Critical path affected. Cause: material delays (external). Recovery: Weekend work Weeks 7-8 (€1,200 premium, recover 4 days). Resequence flooring to later (1 day recovered). Negotiate with client: 1 day acceptable delay (no penalty). Net: 4 days recovered, 1 day delay accepted, €1,200 cost. Project completes Week 13 instead of Week 12. Client satisfied (minimal delay, transparent communication).
