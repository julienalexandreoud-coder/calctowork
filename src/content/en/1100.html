Decking Calculator Formula – How It Works

Boards Needed = (Deck Width ÷ (Board Width + Gap)) × (Deck Length ÷ Board Length) × (1 + Waste %)

Where:

  • Deck Width – The shorter dimension of your deck (perpendicular to board direction)
  • Board Width – Actual face width of decking board (not nominal size)
  • Gap – Spacing between boards (6mm for composite, 3-5mm for wood)
  • Board Length – Length of individual boards (2.4m, 3.6m, 4.8m, 5.4m)
  • Waste % – 10% for simple rectangles, 15-20% for complex shapes

Why Gap Spacing Matters

Proper gapping is critical for deck longevity:

  • Composite decking: Requires 6mm gaps minimum for thermal expansion. In hot climates, increase to 8mm.
  • Wood decking: Needs 3-5mm gaps for drainage. Wood shrinks when dry, so tight gaps can cause cupping.
  • PVC decking: Expands/contracts more than composite. Use 8-10mm gaps in direct sun.

Our calculator automatically applies the correct gap based on material type selected.

Step-by-Step: Calculate Your Deck Boards

  1. Measure deck dimensions: Measure length and width at the longest points. For L-shaped decks, split into rectangles and calculate separately.
  2. Choose board orientation: Boards typically run perpendicular to joists. If joists are 400mm apart, boards span across them. Decide if you want boards running lengthwise or widthwise—this affects waste.
  3. Select board size: Common composite deck board sizes:
    • 140mm × 25mm (5.5" × 1") – Most common, good for residential decks
    • 184mm × 30mm (7.25" × 1.25") – Wider, fewer boards needed
    • 90mm × 25mm (3.5" × 1") – Narrow, traditional look
  4. Enter gap spacing: Use 6mm for composite, 5mm for wood. Some composite brands require specific gaps—check manufacturer specs.
  5. Calculate boards: Enter values into the calculator. It will show boards per row, number of rows, and total boards needed.
  6. Add waste factor: Calculator adds 10% automatically. Increase to 15% if your deck has angles, curves, or multiple levels.

Common Decking Mistakes to Avoid

Mistake 1: Using Nominal Board Width Instead of Actual
A "6-inch" deck board is actually 140mm (5.5"), not 150mm. Using nominal width in calculations results in ordering 5-8% fewer boards than needed. Always measure actual face width.

Mistake 2: Forgetting Expansion Gaps
Composite decking expands up to 3mm per meter in hot weather. A 5m run can expand 15mm. Without proper gapping, boards will buckle and warp. Our calculator includes gap spacing automatically.

Mistake 3: Not Accounting for Board Length Optimization
If your deck is 5m long and you buy 3.6m boards, you'll need to join boards. This creates waste and requires hidden fasteners or extra joists. Consider using 5.4m boards for spans over 4m to minimize joints.

Mistake 4: Underestimating Waste on Diagonal Patterns
Diagonal board patterns (45° angle) look stunning but generate 20-25% waste vs. 10% for straight runs. Always increase waste factor for diagonal layouts.

Mistake 5: Ordering All Boards from Different Batches
Composite decking color varies slightly between production batches. Order all boards at once and check batch numbers. If you need more later, the new batch may not match perfectly.

Pro Tips for Deck Board Installation

Tip 1: Stagger End Joints Properly
When boards need to be joined, stagger joints by at least 300mm between adjacent rows. Never have joints line up—they create weak points. Plan joint locations before cutting.

Tip 2: Use Hidden Fasteners for Composite
Hidden fastener systems (Trex Hideaway, Cortex) create clean surfaces but require specific joist spacing (400mm max). They also add £2-3 per board in fastener costs.

Tip 3: Pre-Sort Boards by Color
Even within the same batch, composite boards vary slightly in color. Lay out all boards before installing and arrange from lightest to darkest. Place darker boards in high-visibility areas.

Tip 4: Acclimate Wood Boards Before Installation
Wood decking should acclimate on-site for 48-72 hours before installation. Stack boards with spacers (stickers) for airflow. This prevents excessive shrinking after installation.

Tip 5: Start from the House
For attached decks, start installing boards from the house side. This puts the cut board (which may be slightly tapered) on the outside edge where it's less noticeable.
