O que é calculadora de magnitude de número complexo?

A Calculadora de Magnitude de Número Complexo encontra o valor absoluto (módulo) de um número complexo - sua distância de zero no plano complexo. Os números complexos combinam partes reais e imaginárias, escritas como a + bi onde i = √(-1). A magnitude informa o “tamanho” do número complexo, essencial para engenharia elétrica, mecânica quântica, processamento de sinais e sistemas de controle. Seja analisando circuitos CA, estudando funções de onda ou projetando filtros, a magnitude complexa é fundamental.

Considere o número complexo 3 + 4i. Sua parte real é 3, a parte imaginária é 4. No plano complexo, isso representa um ponto 3 unidades à direita e 4 unidades acima da origem. A magnitude é a distância em linha reta de (0,0) a (3,4), calculada usando o teorema de Pitágoras: |3 + 4i| = √(3² + 4²) = √(9 + 16) = √25 = 5. Este famoso triângulo retângulo 3-4-5 demonstra como a magnitude complexa funciona geometricamente.

Os números complexos não são “imaginários” no sentido de serem irreais – eles são ferramentas indispensáveis ​​para modelar oscilações, rotações e fenômenos ondulatórios. A magnitude representa amplitude no processamento de sinal, impedância em circuitos e amplitude de probabilidade na mecânica quântica. Compreender a magnitude complexa desbloqueia aplicações em ciência e engenharia.

Como funciona a calculadora de magnitude de números complexos: fórmulas explicadas

Fórmula de magnitude:|a + bi| = √(a² + b²), onde a é a parte real e b é a parte imaginária. Isto estende o teorema de Pitágoras ao plano complexo. Exemplo: |5 + 12i| = √(5² + 12²) = √(25 + 144) = √169 = 13. Outro: |-8 + 6i| = √((-8)² + 6²) = √(64 + 36) = √100 = 10. A magnitude é sempre um número real não negativo.

Conversão de forma polar:Qualquer número complexo pode ser escrito como r(cos θ + i sin θ) ou re^(iθ), onde r = |z| é a magnitude e θ = arg(z) é o ângulo (argumento). Para z = 3 + 4i: r = 5, θ = arctan(4/3) ≈ 53,13°. Forma polar: 5(cos 53,13° + i sen 53,13°) ou 5e^(i·53,13°). A forma polar simplifica a multiplicação e divisão de números complexos.

Magnitude do produto:|z₁ × z₂| = |z₁| × |z₂|. A magnitude de um produto é igual ao produto das magnitudes. Exemplo: z₁ = 3 + 4i (|z₁| = 5), z₂ = 1 + i (|z₂| = √2). Produto: (3+4i)(1+i) = 3 + 3i + 4i + 4i² = 3 + 7i - 4 = -1 + 7i. Magnitude: |-1 + 7i| = √(1 + 49) = √50 = 5√2. Verifique: |z₁| × |z₂| = 5 × √2 = 5√2 ✓.

Propriedades de magnitude:As principais propriedades simplificam os cálculos: |z| ≥ 0 (sempre não negativo). |z| = 0 se e somente se z = 0. |-z| = |z| (a negação não altera a magnitude). |conjugado(z)| = |z| (conjugado tem a mesma magnitude). |zⁿ| = |z|ⁿ (a magnitude da potência é igual à potência da magnitude). Essas propriedades permitem a manipulação algébrica sem converter para a forma retangular.

Trabalhando com exemplos completos: Encontre |7 - 24i|. Parte real a = 7, parte imaginária b = -24. |7 - 24i| = √(7² + (-24)²) = √(49 + 576) = √625 = 25. Encontre |-5 - 12i|. a = -5, b = -12. |-5 - 12i| = √(25 + 144) = √169 = 13. Encontre |6i| (puro imaginário). uma = 0, b = 6. |0 + 6i| = √(0 + 36) = 6. Encontre |10| (puro real). a = 10, b = 0. |10 + 0i| = √(100 + 0) = 10. Números reais puros e imaginários puros seguem a mesma fórmula.

Guia passo a passo para encontrar a magnitude de números complexos

  1. Identifique a parte real (a).A parte real é o termo sem i. Em 8 + 3i, a parte real é 8. Em -5 - 7i, a parte real é -5. Em 6i, a parte real é 0 (nenhum termo real escrito). Escreva explicitamente, incluindo o sinal.
  2. Identifique a parte imaginária (b).A parte imaginária é o coeficiente de i (sem incluir o próprio i). Em 8 + 3i, b = 3. Em -5 - 7i, b = -7 (o sinal de menos faz parte de b). Em 6i, b = 6. Em 10 (real puro), b = 0. Cuidado: a parte imaginária é o número multiplicando i, sem incluir i.
  3. Esquadre ambas as partes.Calcule a² e b². Exemplo: z = -8 + 15i. a = -8, b = 15. a² = (-8)² = 64. b² = 15² = 225. Lembre-se: elevar ao quadrado um negativo dá um positivo. (-8)² = 64, não -64. É por isso que a magnitude é sempre não negativa.
  4. Adicione os quadrados.Calcule a² + b². Continuando o exemplo: 64 + 225 = 289. Esta soma representa o quadrado da distância da origem no plano complexo. Pelo teorema de Pitágoras, isso é igual ao quadrado da magnitude.
  5. Tire a raiz quadrada.|z| = √(a² + b²). Exemplo: √289 = 17. A magnitude de -8 + 15i é 17. Use uma calculadora para quadrados não perfeitos: |3 + 5i| = √(9 + 25) = √34 ≈ 5,831. Arredonde com a precisão apropriada para sua aplicação.
  6. Verifique a razoabilidade.A magnitude deve ser pelo menos tão grande quanto o maior de |a| e |b|. Para 3 + 4i, a magnitude 5 é maior que 3 e 4 ✓. Para 10 + 0i, a magnitude é igual a 10 ✓. Se sua magnitude for menor que |a| ou |b|, recalcular — algo deu errado.

Exemplos de magnitude de números complexos do mundo real

Exemplo 1: Impedância do circuito CA.Um circuito CA tem resistência R = 3 ohms e reatância indutiva X_L = 4 ohms. Impedância total Z = 3 + 4i ohms (a resistência é real, a reatância é imaginária). Magnitude |Z| = √(3² + 4²) = 5 ohms. Esta é a resistência efetiva que o circuito apresenta à corrente CA. Com 120 Vca aplicado, corrente I = V/|Z| = 120/5 = 24 amperes. A magnitude determina o fluxo real da corrente, não a impedância complexa em si.

Exemplo 2: Amplitude de processamento de sinal.Um sinal é representado como o número complexo 7 - 24i no domínio da frequência. A magnitude |7 - 24i| = √(49 + 576) = √625 = 25 representa a amplitude (força) do sinal. O ângulo de fase θ = arctan(-24/7) ≈ -73,7° representa a mudança de fase. Os engenheiros se preocupam com ambos: a amplitude determina o volume/tensão, a fase determina as relações de temporização.

Exemplo 3: Probabilidade da Mecânica Quântica.Um estado quântico tem amplitude de probabilidade ψ = 0,6 + 0,8i. A probabilidade de medir este estado é |ψ|² = (0,6)² + (0,8)² = 0,36 + 0,64 = 1,0 = 100%. Isso está devidamente normalizado. Outro estado φ = 0,3 + 0,4i tem |φ|² = 0,09 + 0,16 = 0,25 = 25% de probabilidade. A magnitude ao quadrado fornece probabilidade mensurável – números complexos são essenciais para a teoria quântica.

Exemplo 4: Estabilidade do Sistema de Controle.A função de transferência de um sistema tem um pólo em s = -2 + 3i no plano complexo. A distância da origem é |s| = √(4 + 9) = √13 ≈ 3,61. A parte real (-2) indica decaimento exponencial; a parte imaginária (3) indica oscilação a 3 rad/s. A magnitude 3,61 refere-se à frequência natural do sistema. Os engenheiros de controle analisam as magnitudes dos pólos para prever o comportamento do sistema.

Exemplo 5: Magnitude vetorial 2D.Um vetor de força em 2D possui componentes F_x = 12 N (horizontal) e F_y = -5 N (vertical). Represente como número complexo 12 - 5i. Magnitude |12 - 5i| = √(144 + 25) = √169 = 13 N. Esta é a intensidade da força total. Direção: θ = arctan(-5/12) ≈ -22,6° (22,6° abaixo da horizontal). Os números complexos fornecem cálculos vetoriais 2D elegantes.

Erros comuns com magnitude de números complexos

Esquecendo de elevar ao quadrado o coeficiente imaginário, e não o próprio i.Errado: |3 + 4i| = √(3² + 4i²) = √(9 - 4) = √5. Correto: |3 + 4i| = √(3² + 4²) = √(9 + 16) = 5. A fórmula usa b², não (bi)². O i não faz parte do que é elevado ao quadrado – b é o coeficiente. Pense nisso como o teorema de Pitágoras: as partes reais e imaginárias são “pernas” perpendiculares de um triângulo retângulo.

Eliminando o sinal negativo ao elevar ao quadrado.Para z = -5 - 12i, ambas as partes são negativas. Correto: |z| = √((-5)² + (-12)²) = √(25 + 144) = 13. Erro: |z| = √(5² + 12²) — acidentalmente dá a resposta certa porque a quadratura elimina o sinal de qualquer maneira. Mas para o argumento (ângulo), os sinais têm uma importância crítica. Sempre monitore os sinais corretamente.

Confundir magnitude com o próprio número complexo.Magnitude é um número real (distância), não um número complexo. |3 + 4i| = 5, não 5 + 0i ou apenas “5i”. A operação de magnitude |z| gera um escalar real não negativo. Não escreva |3 + 4i| = 5i ou |3 + 4i| = 3 + 4i — ambos estão errados. A magnitude elimina a direção, deixando apenas o tamanho.

Identificação incorreta da parte imaginária.Em z = 5 + 3i, a parte imaginária é 3, não 3i. Em z = 7 - 2i, a parte imaginária é -2, não -2i ou 2. A parte imaginária b é o coeficiente real de i. Fórmula: |a + bi| = √(a² + b²), onde b é um número real. Escrever b = 3i e calcular √(a² + (3i)²) = √(a² - 9) está incorreto.

Dicas profissionais para magnitude de números complexos

Use multiplicação conjugada para divisão.Para encontrar |z₁/z₂|, use |z₁/z₂| = |z₁|/|z₂| em vez de dividir números complexos diretamente. Exemplo: z₁ = 8 + 6i (|z₁| = 10), z₂ = 3 - 4i (|z₂| = 5). |z₁/z₂| = 10/5 = 2. Muito mais simples que calcular (8+6i)/(3-4i) = (8+6i)(3+4i)/(9+16) = (24+32i+18i-24)/25 = 50i/25 = 2i, então |2i| = 2.

Reconheça os triplos pitagóricos para cálculos rápidos.Os triplos pitagóricos comuns fornecem magnitudes inteiras: 3-4-5, 5-12-13, 8-15-17, 7-24-25, 20-21-29. Os números complexos 3+4i, 5+12i, 8+15i têm magnitudes 5, 13, 17 respectivamente. Memorizar essas triplas acelera os cálculos manuais e ajuda a verificar os resultados da calculadora.

Converta para a forma polar para potências e raízes.Para zⁿ, use a forma polar: se z = re^(iθ), então zⁿ = rⁿe^(inθ). Magnitude |zⁿ| = rⁿ = |z|ⁿ. Exemplo: (3+4i)³ tem magnitude 5³ = 125. Calculando (3+4i)³ diretamente: (3+4i)² = 9+24i-16 = -7+24i. Então (-7+24i)(3+4i) = -21-28i+72i-96 = -117+44i. Magnitude: √(117²+44²) = √(13689+1936) = √15625 = 125 ✓.

Aplique a desigualdade triangular.Para quaisquer números complexos: |z₁ + z₂| ≤ |z₁| + |z₂|. A grandeza de uma soma é, no máximo, a soma das grandezas. Exemplo: z₁ = 3+4i (|z₁|=5), z₂ = 5-12i (|z₂|=13). z₁+z₂ = 8-8i, |z₁+z₂| = √128 ≈ 11,31. Soma das magnitudes: 5+13 = 18. Na verdade, 11,31 ≤ 18. A igualdade é válida apenas quando z₁ e z₂ apontam na mesma direção.

Se o seu cálculo produzir uma magnitude negativa, você cometeu um erro – provavelmente esqueceu de calcular a raiz quadrada ou manuseou mal os sinais.

Todos os números complexos com a mesma magnitude estão em um círculo centrado na origem - esse círculo é chamado de "curva de nível" de magnitude constante.
