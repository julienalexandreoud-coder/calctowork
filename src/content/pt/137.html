O que é variação?

A variância mede a distribuição de um conjunto de dados calculando a média dos desvios quadrados da média. Quantifica a variabilidade: alta variância significa que os pontos de dados estão amplamente dispersos; baixa variância significa que eles se agrupam fortemente em torno da média. A variância é a base para desvio padrão, ANOVA, análise de regressão e teoria de portfólio.

A variância é expressa em unidades quadradas (por exemplo, dólares², metros²), o que a torna menos intuitiva que o desvio padrão, mas matematicamente essencial. A raiz quadrada da variância é igual ao desvio padrão, trazendo a medida de volta às unidades originais.

Visão principal:A variância tem uma propriedade única: as variâncias das variáveis ​​independentes são somadas. Se X e Y forem independentes, Var(X + Y) = Var(X) + Var(Y). Isto torna a variância indispensável para combinar incertezas em física, finanças e engenharia.

Fórmulas explicadas

Variância da amostra (mais comum):
s² = Σ(xᵢ - x̄)² / (n - 1)

Variância populacional:
σ² = Σ(xᵢ - μ)² / N

Fórmula computacional (mais fácil para cálculos):
s² = [Σxᵢ² - (Σxᵢ)²/n] / (n - 1)

Where:
xᵢ = cada valor individual
x̄ = média amostral
μ = média populacional
n = tamanho da amostra
N = tamanho da população
Σ = soma de

Por que (n-1) para amostras?
Usar (n-1) em vez de n fornece uma estimativa imparcial da variância populacional. Esta “correcção de Bessel” explica o facto de estimarmos a média a partir dos mesmos dados, o que torna os desvios ligeiramente menores do que seriam da verdadeira média populacional.

Propriedades principais:

  • Var(c) = 0 — constante não tem variação
  • Var(X + c) = Var(X) — adicionar constante não altera o spread
  • Var(cX) = c² × Var(X) — o escalonamento multiplica a variância por quadrado
  • Var(X + Y) = Var(X) + Var(Y) — para X, Y independente
  • Var(X) = E[X²] - (E[X])² — definição alternativa usando expectativas

Guia passo a passo

  1. Insira seus dados:Insira números separados por vírgula. Exemplo: 2, 4, 4, 4, 5, 5, 7, 9
  2. Clique em calcular:A calculadora calcula média e variância
  3. Leia seus resultados:Média = 5, Variância amostral = 4,57, Variância populacional = 4,0
  4. Optional:Obtenha a raiz quadrada do desvio padrão: √4,57 = 2,14
  5. Interpretar:Maior variância = mais spread; comparar entre conjuntos de dados

Exemplos reais com cálculos

Exemplo 1: Risco da carteira de investimentos
Retornos anuais da ação A (5 anos): 10%, 15%, 8%, 22%, 5%
Média: (10+15+8+22+5) / 5 = 60/5 = 12%
Desvios: -2, 3, -4, 10, -7
Desvios quadrados: 4, 9, 16, 100, 49
Sum: 178
Variância da amostra: 178/4 = 44,5 (%²)
Desvio padrão: √44,5 = 6,67%
Aplicação: Maior variância = maior risco. Compare com a ação B com variância = 25 – a ação A é mais arriscada.

Exemplo 2: Comparação do processo de fabricação
A máquina A produz peças (mm): 10,1, 9,9, 10,0, 10,2, 9,8
A máquina B produz peças (mm): 10,3, 9,7, 10,5, 9,5, 10,0
Ambos têm média = 10,0 mm
Variação da máquina A: 0,025 mm²
Variação da máquina B: 0,17 mm²
Aplicação: A máquina A tem variância 7× menor – qualidade muito mais consistente. Escolha a Máquina A para trabalhos de precisão.

Exemplo 3: análise de teste A/B
Conversões do grupo controle (10 dias): 45, 48, 52, 47, 50, 49, 51, 46, 53, 48
Média: 489/10 = 48,9
Variância: 7,66
Grupo de tratamento: 52, 55, 49, 58, 54, 51, 56, 53, 57, 50
Média: 535/10 = 53,5
Variância: 9,17
Aplicação: O tratamento tem média mais alta (+4,6), mas também variância mais alta. Use o teste t para determinar se a diferença é estatisticamente significativa.

Exemplo 4: Combinação de riscos independentes
O projeto tem 3 fatores de risco independentes:
Variação do risco A: 100 (custo excedente)
Variação do risco B: 64 (penalidade por atraso)
Variação do risco C: 36 (custo de retrabalho)
Variância total: 100 + 64 + 36 = 200
DP total: √200 = 14,14
Aplicação: A variação do portfólio aumenta para riscos independentes. O risco total (SD) é menor que a soma dos SD individuais devido à diversificação.

Exemplo 5: Gráfico de controle de qualidade
Variância histórica do processo: σ² = 2,25 (DP = 1,5)
Limites de controle: média ± 3σ = média ± 4,5
Nova amostra: 8 observações, variância = 6,8
Aplicação: A variância da amostra (6,8) excede a histórica (2,25) em 3×. O processo pode ter se tornado instável. Investigar causas atribuíveis (novo fornecedor, desgaste de equipamentos, mudança de operador).

4 erros comuns

  • Variância confusa entre amostra e população:Use (n-1) para amostras, N para populações. A variância da amostra é maior, compensando a incerteza da estimativa. A maioria das análises do mundo real usa amostras, então (n-1) é o padrão.
  • Interpretando a variação diretamente:A variação está em unidades quadradas (dólares², kg²), que são difíceis de interpretar. Pegue a raiz quadrada para obter o desvio padrão nas unidades originais. Use variância para operações matemáticas; use SD para comunicação.
  • Adicionando variâncias de variáveis ​​correlacionadas:Var(X + Y) = Var(X) + Var(Y) somente quando X e Y são independentes (não correlacionados). Para variáveis ​​correlacionadas: Var(X + Y) = Var(X) + Var(Y) + 2×Cov(X,Y). Ignorar a covariância subestima ou superestima a variância total.
  • Usando variância para distribuições distorcidas:A variância assume uma distribuição simétrica em torno da média. Para dados distorcidos (rendimentos, preços de habitação), a variância é dominada por valores extremos. Use intervalo interquartil (IQR) ou desvio absoluto mediano (MAD) para medidas de spread robustas.

4 dicas profissionais

  • Use a fórmula computacional para cálculos manuais:s² = [Σx² - (Σx)²/n] / (n-1). Para dados 2, 4, 5, 7: Σx = 18, Σx² = 94, n = 4. Variância = [94 - 324/4] / 3 = [94 - 81] / 3 = 13/3 = 4,33. Mais rápido do que calcular desvios individuais.
  • Aplicar decomposição de variância (princípio ANOVA):Variância total = variância entre grupos + variância dentro do grupo. Isso revela se as diferenças vêm da filiação ao grupo ou da variação aleatória dentro dos grupos.
  • Use variância agrupada para comparar grupos:Ao comparar duas amostras com variações semelhantes, agrupe-as: s²_pooled = [(n₁-1)s₁² + (n₂-1)s₂²] / (n₁+n₂-2). Isso fornece uma estimativa mais estável para testes t.
  • Transforme os dados para estabilizar a variância:Se a variância aumentar com a média (comum em dados de contagem), aplique raiz quadrada ou transformação logarítmica. Esta “estabilização da variância” torna os testes estatísticos mais válidos. Exemplo: Para dados de Poisson, √x tem variância aproximadamente constante.

FAQ

A quadratura tem vantagens matemáticas: (1) a derivada existe em todos os lugares (permitindo a otimização do cálculo), (2) as variâncias das variáveis ​​independentes são adicionadas, (3) os desvios quadrados penalizam mais fortemente os valores discrepantes. O desvio médio absoluto (MAD) é mais robusto, mas mais difícil de trabalhar analiticamente.

Não. A variância é sempre zero ou positiva. É uma média de termos quadrados e os quadrados nunca são negativos. Variância = 0 somente quando todos os valores são idênticos (nenhum spread).

"Alto" é relativo. Compare a variância com a média usando o coeficiente de variação: CV = DP/média = √variância/média. CV > 1 indica alta variabilidade relativa. Compare também com benchmarks do setor ou dados históricos para contextualizar.

Use variância para: derivações matemáticas, combinação de variáveis ​​independentes, ANOVA, regressão. Use o desvio padrão para: relatar resultados, interpretar spread, intervalos de confiança, comparar com a média. A variação é para cálculos; SD é para comunicação.
