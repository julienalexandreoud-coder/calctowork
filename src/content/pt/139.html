O que é um quartil?

Os quartis dividem um conjunto de dados classificado em quatro partes iguais, cada uma contendo 25% dos dados. Q1 (primeiro quartil) marca o percentil 25, Q2 (segundo quartil) é o percentil 50 (mediana) e Q3 (terceiro quartil) é o percentil 75. Os quartis revelam a dispersão e a distorção dos dados sem serem distorcidos por valores discrepantes.

O intervalo interquartil (IQR = Q3 - Q1) contém os 50% intermediários dos dados e é uma medida robusta de spread. Os gráficos de caixa visualizam quartis, mostrando a mediana como uma linha dentro de uma caixa de Q1 a Q3, com "bigodes" estendendo-se para mostrar o intervalo de valores típicos.

Visão principal:Os quartis dividem os dados em trimestres. Se você pontuar no terceiro quartil em um teste, seu desempenho será melhor do que 50-75% dos participantes. Os quartis são essenciais para identificar valores discrepantes, comparar distribuições e compreender a forma dos dados sem assumir a normalidade.

Fórmulas explicadas

Posições quartis (para n valores classificados):

Q1 (percentil 25): Posição = (n + 1) × 0,25
Q2 (percentil 50, mediana): Posição = (n + 1) × 0,50
Q3 (percentil 75): Posição = (n + 1) × 0,75

Quando a posição é um número inteiro:
Quartil = valor naquela posição

Quando a posição é fracionária:
Interpolar entre valores adjacentes:
Q = valor_abaixo + fração × (valor_acima - valor_abaixo)

Intervalo Interquartil (IQR):
IQR = Q3 - Q1

Limites discrepantes (método de Tukey):
Cerca inferior = Q1 - 1,5 × IQR
Cerca superior = Q3 + 1,5 × IQR
Os valores fora das cercas são discrepantes.

Exemplo de cálculo:
Dados: 3, 7, 8, 12, 14, 18, 21, 25, 30 (n = 9)
Posição Q1: 10 × 0,25 = 2,5 → interpolar entre as posições 2 e 3
Q1 = 7 + 0,5 × (8 - 7) = 7,5
Posição Q2: 10 × 0,50 = 5 → valor na posição 5 = 14
Posição Q3: 10 × 0,75 = 7,5 → interpolar entre as posições 7 e 8
Q3 = 21 + 0,5 × (25 - 21) = 23
AIQ = 23 - 7,5 = 15,5

Guia passo a passo

  1. Insira seus dados:Insira números separados por vírgula. Exemplo: 12, 15, 18, 22, 25, 28, 32, 35, 40, 45
  2. Clique em calcular:A calculadora classifica os dados e calcula os quartis
  3. Leia seus resultados:Q1 = 17,25, Q2 (mediana) = 26,5, Q3 = 34,25
  4. Calcular IQR:IQR = Q3 - Q1 = 34,25 - 17,25 = 17
  5. Verifique os valores discrepantes:Cerca inferior = 17,25 - 25,5 = -8,25; Cerca superior = 34,25 + 25,5 = 59,75

Exemplos reais com cálculos

Exemplo 1: Análise salarial de funcionários
Salários (milhares): 42, 45, 48, 52, 55, 58, 62, 65, 70, 75, 85, 120
Ordenado: igual (já crescente), n = 12
Posição Q1: 13 × 0,25 = 3,25 → 48 + 0,25×(52-48) = 49
Posição Q2: 13 × 0,50 = 6,5 → (58+62)/2 = 60
Posição Q3: 13 × 0,75 = 9,75 → 70 + 0,75×(75-70) = 73,75
IQR = 73,75 - 49 = 24,75
Cercas atípicas: 49 - 37,1 = 11,9 a 73,75 + 37,1 = 110,85
US$ 120 mil excedem a barreira superior – é um valor atípico (provável executivo).
Aplicação: O RH usa quartis para benchmarking de remuneração. “Seu salário está no terceiro trimestre” significa que você ganha mais de 75% dos colegas.

Exemplo 2: Distribuição das notas dos testes dos alunos
Notas do exame: 52, 58, 61, 64, 67, 70, 72, 74, 76, 78, 80, 82, 85, 88, 92
n = 15
Posição Q1: 16 × 0,25 = 4 → valor = 64
Posição Q2: 16 × 0,50 = 8 → valor = 74
Posição Q3: 16 × 0,75 = 12 → valor = 82
AIQ = 82 - 64 = 18
Aplicação: O professor vê 50% intermediários com pontuação entre 64-82. Alunos abaixo de Q1 (64) necessitam de intervenção; alunos acima do Q3 (82) podem precisar de enriquecimento. Mais acionável do que apenas maldade.

Exemplo 3: Segmentação do valor do pedido de comércio eletrônico
Valores do pedido ($): 15, 22, 28, 35, 42, 48, 55, 62, 70, 85, 95, 120, 150, 200, 350
Q1 = 35, Q2 = 62, Q3 = 120, IQR = 85
Limite de valores discrepantes: 120 + 127,5 = 247,5 → $350 é um valor discrepante
Aplicação: clientes do segmento de marketing: abaixo de Q1 = compradores com orçamento limitado, Q1-Q2 = buscadores de valor, Q2-Q3 = clientes regulares, acima de Q3 = VIPs. Adapte as promoções a cada quartil.

Exemplo 4: Monitoramento de desempenho do site
Tempos de carregamento da página (ms): 245, 268, 289, 312, 334, 356, 378, 401, 425, 458, 512, 680
Q1 = 300,5, Q2 = 367, Q3 = 447,25, IQR = 146,75
Cerca superior: 447,25 + 220 = 667 → 680ms é um valor discrepante
Aplicação: A equipe de desempenho concentra-se em P75 (Q3 = 447ms) como meta. A correção de valores discrepantes (680 ms) melhora a experiência do usuário no pior caso. O Google usa P75 para limites do Core Web Vitals.

Exemplo 5: Intervalos de referência para laboratórios clínicos
Níveis de colesterol (mg/dL) de 100 adultos saudáveis:
Q1 = 165, Q2 = 185, Q3 = 210
"Intervalo normal" geralmente definido como Q1 a Q3 (50% médio)
Aplicação: Relatório de resultados de laboratório "seu valor: 195, mediana da população: 185, seu percentil: 62%". Pacientes acima do Q3 podem necessitar de mudanças no estilo de vida; valores muito fora do AIQ justificam investigação médica.

4 erros comuns

  • Usando diferentes métodos de quartil de forma inconsistente:As calculadoras Excel, R, Python e TI usam métodos de interpolação ligeiramente diferentes. Os resultados podem variar em 0,5-1 posição. Sempre documente qual método você usa (esta calculadora usa o método (n+1) com interpolação linear).
  • Confundindo quartis com quartos:Q1 é o valor abaixo do qual 25% dos dados caem – é um limite, não um grupo. O "primeiro quartil" às vezes se refere aos 25% mais baixos dos dados (valores abaixo do Q1). Esclareça se você se refere ao limite ou ao grupo.
  • Supondo que os quartis sejam igualmente espaçados:Em dados simétricos, Q2-Q1 ≈ Q3-Q2. Mas em dados distorcidos, as lacunas são diferentes. Dados de renda: 1º trimestre = US$ 35 mil, 2º trimestre = US$ 55 mil, 3º trimestre = US$ 95 mil. Gap Q2-Q1 = US$ 20 mil; lacuna Q3-Q2 = $ 40k - revela inclinação para a direita.
  • Usando quartis para conjuntos de dados pequenos:Com n <10, os quartis são instáveis ​​e altamente sensíveis aos valores individuais. Um conjunto de dados de 5 números não pode ser dividido de forma significativa em trimestres. Use quartis para n ≥ 20; para dados menores, relatório mínimo, mediana, máx.

4 dicas profissionais

  • Use quartis para gráficos de caixa:Os gráficos de caixa visualizam resumos de cinco números: min, Q1, Q2, Q3, max. A caixa mostra o IQR (meio 50%), a linha mostra a mediana, os bigodes mostram o intervalo típico. Os gráficos de caixa lado a lado comparam os grupos de forma eficaz.
  • Aplique detecção de outlier baseada em quartil:Valores abaixo de Q1-1,5×IQR ou acima de Q3+1,5×IQR são “outliers moderados”; além de Q1-3×IQR ou Q3+3×IQR são “outliers extremos”. Mais robusto que ±3 SD para dados não normais.
  • Use decis ou percentis para granularidade mais fina:Os quartis dividem os dados em 4 partes; decis em 10; percentis em 100. Para grandes conjuntos de dados (n > 1.000), os percentis revelam mais detalhes sobre o formato da distribuição.
  • Compare quartis entre grupos:"A renda média aumentou do segundo trimestre = US$ 52 mil para o segundo trimestre = US$ 58 mil, mas o primeiro trimestre permaneceu estável em US$ 35 mil" revela o crescimento concentrado na metade superior. As comparações de quartis expõem a desigualdade que significa ocultar.

FAQ

Os quartis são percentis específicos: Q1 = P25, Q2 = P50 (mediana), Q3 = P75. Os percentis dividem os dados em 100 partes; quartis em 4. Dizer "você pontuou no 85º percentil" é mais preciso do que "você pontuou acima do Q3".

Existem vários métodos válidos: Excel QUARTILE.EXC vs QUARTILE.INC, 9 tipos de R, linear do Python vs mais próximo. As diferenças surgem de como as posições são calculadas e se a interpolação é usada. Todos os métodos convergem para grandes conjuntos de dados.

Sim. Os quartis funcionam com quaisquer dados numéricos ordenados, incluindo negativos. Conjunto de dados: -20, -15, -10, -5, 0, 5, 10 tem Q1=-12,5, Q2=-5, Q3=7,5. O método é idêntico; valores negativos não requerem tratamento especial.

Use IQR para: dados distorcidos, dados com valores discrepantes, dados ordinais ou quando você não assume normalidade. Use SD para: dados simétricos, testes paramétricos, combinação de variáveis ​​independentes ou quando propriedades matemáticas são importantes. A AIQ é robusta; SD é matematicamente conveniente.
