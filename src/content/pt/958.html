A distribuição normal padrão

O escore Z indica quantos desvios padrão um valor está da média:

  • Z = 0:exatamente na média (percentil 50)
  • Z = 1:um desvio padrão acima (~84,1º percentil)
  • Z = −1:um desvio padrão abaixo (~15,9º percentil)
  • Z = 1.96:Percentil 97,5 (usado no IC 95%)

O percentil é obtido avaliando a função de distribuição cumulativa (CDF) da normal padrão no valor Z.

Exemplo 1: Z positivo

Problem:Qual percentil corresponde a Z = 1,5?

  1. Result:
    • Φ(1,5) ≈ 0,9332 → 93,32º percentil.

Answer:Z = 1,5 corresponde ao percentil 93,32 (93,32% dos dados estão abaixo).

Exemplo 2: Z negativo

Problem:Qual percentil corresponde a Z = −2,0?

  1. Result:
    • Φ(−2,0) ≈ 0,0228 → 2,28º percentil.

Answer:Z = −2,0 corresponde ao percentil 2,28 (apenas 2,28% dos dados estão abaixo).

Usos comuns de percentis de pontuação Z

  • Interpretação de resultados de testes padronizados (SAT, GRE, IQ).
  • Avaliação de percentis de crescimento em pediatria.
  • Identificando outliers na análise de dados.
  • Calculando intervalos de confiança em estatísticas.
  • Realização de testes de hipóteses com distribuição normal.
  • Analisar os resultados do controle de qualidade na fabricação.

Erros comuns com pontuações Z

  • Interpretando Z como o percentil diretamente (Z = 1 não é o 1º percentil).
  • Não verificar se os dados seguem uma distribuição normal.
  • Percentil confuso com percentual dos dados acima.
  • Usando Z para amostras pequenas onde a distribuição t é mais apropriada.

Pro tip

A regra empírica 68-95-99,7 é útil: aproximadamente 68% dos dados estão dentro de ±1σ, 95% dentro de ±2σ e 99,7% dentro de ±3σ da média.

É o número de desvios padrão que um valor está em relação à média: Z = (valor - média) / desvio_padrão.

Z ≈ 1,645 para o percentil 95 (unicaudal). Para um IC de 95% (bicaudal), Z = 1,96.

Convencionalmente, um valor com |Z| > 3 é considerado um valor discrepante (mais de 3 desvios padrão da média).
