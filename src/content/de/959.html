Stichprobengrößenrechner: Bestimmen Sie Ihre Umfragegröße

Der Stichprobengrößenrechner ermittelt, wie viele Befragte Sie in einer Umfrage oder Studie benötigen, um das gewünschte Maß an statistischer Präzision zu erreichen. Es gleicht Konfidenzniveau, Fehlerquote und Bevölkerungsanteil aus, sodass Sie Daten sammeln können, die sowohl zuverlässig als auch kostengünstig sind.

Stichprobengrößenformel

n = Z² × p(1 − p) / e²

Where:

  • n= erforderliche Stichprobengröße
  • Z= Z-Score entsprechend dem gewünschten Konfidenzniveau (1,96 für 95 %, 2,576 für 99 %)
  • p= geschätzter Anteil der Bevölkerung (verwenden Sie 0,5, wenn die maximale Stichprobengröße unbekannt ist)
  • e= gewünschte Fehlerspanne, ausgedrückt als Dezimalzahl (z. B. 0,05 für ±5 %)

Diese als Cochran-Formel bekannte Formel geht von einer unendlichen oder sehr großen Population aus. Wenn Ihre Population klein ist, wenden Sie die endliche Populationskorrektur an: nadjusted= n / (1 + (n − 1) / N), wobei N die Gesamtbevölkerungsgröße ist.

Ausgearbeitete Beispiele

Beispiel 1: Umfrage zur Kundenzufriedenheit

Ein Einzelhandelsunternehmen möchte die Kundenzufriedenheit mit einem Konfidenzniveau von 95 % und einer Fehlerquote von ±5 % messen. Sie haben keine vorherige Schätzung des Zufriedenheitsanteils und verwenden daher p = 0,5.

Berechnung:n = 1,96² × 0,5(1 − 0,5) / 0,05² = 3,8416 × 0,25 / 0,0025 = 0,9604 / 0,0025 =384,16 → 385 Befragte

Um die gewünschte Präzision zu erreichen, benötigt das Unternehmen mindestens 385 abgeschlossene Umfragen.

Beispiel 2: Politische Umfragen mit kleiner Bevölkerung

Eine Universität mit 2.000 Studenten möchte die Meinung der Studenten mit einer Sicherheit von 99 % und einer Fehlerquote von ±3 % befragen. Unter Verwendung von p = 0,5 für maximale Variabilität:

Schritt 1 – Anfangsgröße:n = 2,576² × 0,5(1 − 0,5) / 0,03² = 6,6358 × 0,25 / 0,0009 = 1,6589 / 0,0009 =1,843

Schritt 2 – Endliche Populationskorrektur: nadj= 1.843 / (1 + (1.843 − 1) / 2.000) = 1.843 / (1 + 0,921) = 1.843 / 1,921 =959 Befragte

Nach der Korrektur müssen statt 1.843 nur noch 959 Studierende befragt werden.

Allgemeine Verwendung

  • Entwerfen von Marktforschungsumfragen mit bekannter statistischer Präzision
  • Planung akademischer Forschungsstudien und Datenerfassung für Abschlussarbeiten
  • Bestimmung der Stichprobengrößen für Qualitätskontrollprüfungen in der Fertigung
  • Festlegung von Befragtenzielen für politische Umfragen und Meinungsforschung
  • Berechnung der Teilnehmerzahlen für klinische Studien und medizinische Studien
  • Dimensionierung von Umfragen zum Mitarbeiterengagement in Personalabteilungen

Häufige Fehler

  • Wenn Sie p = 0,5 verwenden, wenn Sie zuverlässige vorherige Daten haben, erhöht sich dadurch die Stichprobengröße unnötig
  • Vergessen der endlichen Bevölkerungskorrektur bei der Befragung einer kleinen, definierten Gruppe
  • Fehlerspanne (e) mit Konfidenzniveau verwechseln – es handelt sich um unabhängige Parameter
  • Nicht-Antwortraten ignorieren und genau n Antworten sammeln, anstatt eine Überstichprobe zu verwenden, um Abbrecher zu berücksichtigen

Pro Tip

Führen Sie immer eine Überstichprobe von 15–30 % über Ihrer berechneten Stichprobengröße durch, um Antwortausfälle, unvollständige Umfragen und Ausschlüsse aus der Datenqualität zu berücksichtigen. Wenn Sie 385 Antworten benötigen, sollten Sie 500–550 Personen einladen. Dieser Puffer stellt sicher, dass Sie Ihre statistischen Anforderungen auch nach der Fluktuation erfüllen.

Häufig gestellte Fragen

Das Produkt p(1 − p) erreicht seinen Maximalwert von 0,25, wenn p = 0,5. Wenn Sie diesen Wert verwenden, erhalten Sie die größtmögliche Stichprobengröße und stellen so sicher, dass Ihre Studie unabhängig vom wahren Anteil ausreichend aussagekräftig ist.
95 % Konfidenz (Z = 1,96) ist der Standard für die meisten Sozialwissenschaften und Marktforschung. Verwenden Sie 99 % (Z = 2,576) für hochriskante Studien wie medizinische Forschung oder die Einhaltung gesetzlicher Vorschriften, bei denen Fehler kostspielig sind.
Nicht unbedingt. Ab einem bestimmten Punkt führt eine zunehmende Stichprobengröße zu einem geringeren Präzisionsertrag. Eine Stichprobe von 1.000 ergibt eine Fehlermarge von ±3,1 %, während sie bei 4.000 nur auf ±1,5 % verbessert wird. Bringen Sie Präzision mit Budget- und Zeitbeschränkungen in Einklang.
Wenden Sie die Korrektur an, wenn Ihre anfängliche Stichprobengröße 5 % der Gesamtpopulation überschreitet (n/N > 0,05). Bei großen Populationen (z. B. nationale Erhebungen) hat die Korrektur vernachlässigbare Auswirkungen und kann übersprungen werden.
