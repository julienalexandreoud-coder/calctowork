Was ist Hyperfokalentfernung?

Die Hyperfokalentfernung ist die Fokusentfernung, die die Schärfentiefe maximiert und alles von der Hälfte dieser Entfernung bis unendlich scharf macht. Wenn Sie ein 24-mm-Objektiv mit Blende 11 und einer Hyperfokalentfernung von 2,5 Metern fokussieren, wird alles von 1,25 Metern bis zum Horizont scharf abgebildet.

Landschaftsfotografen verwenden Hyperfokalfokussierung, um scharfe Felsen im Vordergrund und entfernte Berge in derselben Aufnahme zu gewährleisten. Anstatt sich auf die Unendlichkeit zu konzentrieren (wodurch DOF hinter der Unendlichkeit verschwendet wird), konzentrieren sie sich auf den berechneten Hyperfokalpunkt, um die gesamte verfügbare Schärfe effizient zu nutzen.

Straßenfotografen, die mit 35 mm und f/8 fotografieren, können auf die Hyperfokalentfernung von etwa 5 Metern vorfokussieren und wissen, dass alles von 2,5 Metern bis unendlich scharf ist. Mit dieser Zonenfokussierungstechnik können sie ohne Autofokusverzögerung fotografieren.

Hyperfokale Distanzformeln mit ausgearbeiteten Berechnungen

Die Formel für die Hyperfokalentfernung lautet:

H = (f²) / (N × c) + f

Dabei ist H die Hyperfokalentfernung, f die Brennweite in Millimetern, N die Blendenzahl (Blende) und c der Zerstreuungskreis in Millimetern.

Arbeitsbeispiel:35-mm-Objektiv mit f/8 an einer Vollformatkamera (c = 0,03 mm).

f² = 35² = 1.225 mm²

N × c = 8 × 0,03 = 0,24 mm

(f²) / (N × c) = 1.225 / 0,24 = 5.104 mm

H = 5.104 + 35 = 5.139 mm ≈ 5,14 Meter

Wenn Sie auf 5,14 Meter fokussieren, wird alles von 2,57 Metern (halber Hyperfokalwert) bis unendlich akzeptabel scharf sein.

Die vereinfachte Formel (ohne Berücksichtigung des kleinen +f-Terms) lautet:

H ≈ (f²) / (N × c)

Für das gleiche Beispiel: H ≈ 1.225 / 0,24 = 5.104 mm ≈ 5,1 Meter – nah genug für den praktischen Gebrauch.

So berechnen Sie die Hyperfokalentfernung: 6 Schritte

  1. Identifizieren Sie Ihre Brennweite:Ihr Objektiv ist auf 20 mm eingestellt. Notieren Sie dies in Millimetern. Weitwinkelobjektive haben kürzere Hyperfokalabstände als Teleobjektive.
  2. Stellen Sie Ihre Blende ein:Für maximale Schärfe planen Sie, mit f/11 zu fotografieren. Das ist Ihr N-Wert. Kleinere Blendenöffnungen (größere Blendenzahlen) verringern die Hyperfokalentfernung.
  3. Bestimmen Sie den Verwirrungskreis:Vollformatkameras verwenden c = 0,03 mm. APS-C (Canon) verwendet c = 0,019 mm. APS-C (Nikon/Sony) verwendet c = 0,02 mm. Micro Four Thirds verwendet c = 0,015 mm.
  4. Quadrieren Sie die Brennweite:20² = 400 mm². Dies ist der Zähler der Hyperfokalformel.
  5. Blende mit CoC multiplizieren:11 × 0,03 = 0,33 mm. Das ist der Nenner.
  6. Dividiere und addiere die Brennweite:400 / 0,33 = 1.212 mm. Addiere f: 1.212 + 20 = 1.232 mm ≈ 1,23 Meter. Fokussieren Sie auf 1,23 Meter und alles von 0,62 Meter bis unendlich wird scharf sein.

5 Beispiele für hyperfokale Entfernungen

Beispiel 1 – Ultraweite Landschaft:14-mm-Objektiv mit f/16, Vollformat (c = 0,03 mm). H = (14²) / (16 × 0,03) + 14 = 196 / 0,48 + 14 = 408 + 14 = 422 mm ≈ 0,42 Meter. Fokussieren Sie auf 42 cm, und von 21 cm bis unendlich ist alles scharf. Dies ermöglicht extreme Vordergrundkompositionen mit Gras oder Blumen nur wenige Zentimeter von der Linse entfernt.

Beispiel 2 – Standard-Straßenfotografie:50-mm-Objektiv mit f/5,6, Vollformat (c = 0,03 mm). H = (50²) / (5,6 × 0,03) + 50 = 2.500 / 0,168 + 50 = 14.881 + 50 = 14.931 mm ≈ 14,9 Meter. Fokussieren Sie auf 15 Meter und alles von 7,5 Metern bis unendlich ist scharf. Gut für Straßenszenen, bei denen sich die Motive selten näher als 7 Meter befinden.

Beispiel 3 – APS-C-Landschaft:18-mm-Objektiv (27-mm-Äquivalent) bei f/11, APS-C Nikon (c = 0,02 mm). H = (18²) / (11 × 0,02) + 18 = 324 / 0,22 + 18 = 1.473 + 18 = 1.491 mm ≈ 1,49 Meter. Fokussieren Sie auf 1,5 Meter für eine Schärfe von 0,75 Metern bis unendlich. Crop-Sensoren haben bei gleicher Blende einen etwas tieferen DOF.

Beispiel 4 – Micro Four Thirds:12-mm-Objektiv (24-mm-Äquivalent) bei f/8, MFT (c = 0,015 mm). H = (12²) / (8 × 0,015) + 12 = 144 / 0,12 + 12 = 1.200 + 12 = 1.212 mm ≈ 1,21 Meter. Fokussieren Sie auf 1,2 Meter für eine Schärfe von 0,6 Metern bis unendlich. Der kleine Sensor ermöglicht einen sehr tiefen DOF selbst bei moderaten Blendenöffnungen.

Beispiel 5 – Komprimierung im Tele-Querformat:85-mm-Objektiv mit Blende 11, Vollformat (c = 0,03 mm). H = (85²) / (11 × 0,03) + 85 = 7.225 / 0,33 + 85 = 21.894 + 85 = 21.979 mm ≈ 22 Meter. Fokussieren Sie auf 22 Meter für eine Schärfe von 11 Metern bis unendlich. Nicht für Vordergrundinteressen geeignet, funktioniert aber für entfernte Bergketten, bei denen nichts näher als 11 Meter ist.

4 häufige hyperfokale Fehler

  • Verwendung des falschen Verwirrungskreises:Die Anwendung von c = 0,03 mm im Vollformat auf eine APS-C-Kamera führt zu 50 % zu langen Hyperfokalabständen. Ein APS-C-Fotograf, der 0,03 mm verwendet, würde auf 7,7 Meter fokussieren, anstatt auf die korrekten 5,1 Meter für ein 35-mm-f/8-Setup, wodurch die Vordergrundelemente weich bleiben.
  • Stattdessen auf Unendlich fokussieren:Viele Fotografen drehen das Objektiv auf das Unendlichkeitssymbol und verschwenden so die Schärfentiefe, die sich hinter der Unendlichkeit erstreckt. Bei einem 24-mm-f/11-Setup macht die Fokussierung auf Unendlich alles von 3 Metern bis Unendlich scharf, aber die Fokussierung auf Hyperfokal (2,5 m) macht alles von 1,25 Metern bis Unendlich scharf – was eine Vordergrundschärfe von 1,75 Metern ergibt.
  • Ohne Berücksichtigung der Linsenmarkierungen:Ältere Objektive verfügen über DOF-Skalen, die Hyperfokalabstände für verschiedene Blendenöffnungen anzeigen. Richten Sie bei Blende 11 das Unendlichkeitssymbol an der Markierung Blende 11 auf der DOF-Skala aus, und der Fokusindex zeigt automatisch auf die Hyperfokalentfernung. Bei modernen Objektiven fehlen diese Skalen oft.
  • Beugung ignorieren:Das Abblenden auf f/22 zur Minimierung der Hyperfokalentfernung führt zu einer Beugungsweichmachung. Ein 24-mm-Objektiv mit f/22 hat H ≈ 0,7 Meter, aber das gesamte Bild ist aufgrund der Beugung leicht weich. Wenn Sie eine größere Schärfe im Vordergrund benötigen, ist es besser, mit Blende 11 (H ≈ 1,4 m) und Fokusstapel zu fotografieren.

5 Tipps zur hyperfokalen Fokussierung

  • Verwenden Sie die Methode mit der doppelten Distanz:Schätzen Sie die Entfernung zum nächstgelegenen Objekt, das Sie scharf stellen möchten, und fokussieren Sie dann auf die doppelte Entfernung. Wenn Sie einen Stein mit einer Schärfe von 1 Meter benötigen, fokussieren Sie auf 2 Meter. Dies entspricht ohne Berechnung einer Hyperfokalaufnahme und eignet sich gut für schnelle Aufnahmen.
  • Überprüfen Sie mit Live-View-Zoom:Nach der Fokussierung auf die berechnete Hyperfokalentfernung verwenden Sie die Live-Ansicht bei 10-facher Vergrößerung, um sowohl das nächstgelegene Vordergrundelement als auch ein entferntes Objekt zu überprüfen. Wenn eines davon weich ist, stellen Sie den Fokus etwas näher ein und überprüfen Sie es erneut.
  • Markieren Sie Ihr Objektiv:Sobald Sie den Hyperfokalwert für Ihre bevorzugten Objektiv-/Blendenkombinationen berechnet haben, markieren Sie den Fokusabstand auf dem Objektivtubus mit einem kleinen Stück Klebeband oder einem trocken abwischbaren Marker. Markieren Sie für ein 24-mm-f/11-Setup 2,5 Meter für eine sofortige hyperfokale Fokussierung.
  • Smartphone-Apps nutzen:Apps wie PhotoPills, Hyperfocal Pro oder DOF Calculator berechnen sofort die Hyperfokalentfernung für Ihre spezifische Kamera. Richten Sie Ihr Telefon auf die Szene und die App zeigt mithilfe von Augmented Reality genau, wo Sie fokussieren müssen.
  • Klammern Sie Ihren Fokus ein:Wenn die Hyperfokalentfernung kritisch ist, machen Sie drei Aufnahmen: eine auf die berechnete Hyperfokalentfernung fokussiert, eine auf 0,5 Meter näher fokussiert und eine auf 0,5 Meter weiter fokussiert. Überprüfen Sie die Bilder später zu 100 % und behalten Sie die Schärfe bei, oder überblenden Sie sie in der Nachbearbeitung.
