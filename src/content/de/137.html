Was ist Varianz?

Die Varianz misst, wie weit ein Datensatz verteilt ist, indem sie den Durchschnitt der quadrierten Abweichungen vom Mittelwert berechnet. Es quantifiziert die Variabilität: Eine hohe Varianz bedeutet, dass die Datenpunkte weit verstreut sind. Eine geringe Varianz bedeutet, dass sie eng um den Durchschnitt liegen. Varianz ist die Grundlage für Standardabweichung, ANOVA, Regressionsanalyse und Portfoliotheorie.

Die Varianz wird in quadrierten Einheiten (z. B. Dollar², Meter²) ausgedrückt, was sie weniger intuitiv als die Standardabweichung macht, aber mathematisch wichtig ist. Die Quadratwurzel der Varianz entspricht der Standardabweichung und bringt das Maß auf die ursprünglichen Einheiten zurück.

Wichtige Erkenntnis:Varianz hat eine einzigartige Eigenschaft: Varianzen unabhängiger Variablen addieren sich. Wenn X und Y unabhängig sind, gilt Var(X + Y) = Var(X) + Var(Y). Dies macht Varianz für die Kombination von Unsicherheiten in Physik, Finanzen und Ingenieurwesen unverzichtbar.

Formeln erklärt

Stichprobenvarianz (am häufigsten):
s² = Σ(xᵢ - x̄)² / (n - 1)

Populationsvarianz:
σ² = Σ(xᵢ - μ)² / N

Rechenformel (einfacher für Berechnungen):
s² = [Σxᵢ² - (Σxᵢ)²/n] / (n - 1)

Where:
xᵢ = jeder einzelne Wert
x̄ = Stichprobenmittelwert
μ = Bevölkerungsmittelwert
n = Stichprobengröße
N = Bevölkerungsgröße
Σ = Summe von

Warum (n-1) für Proben?
Die Verwendung von (n-1) anstelle von n liefert eine unvoreingenommene Schätzung der Populationsvarianz. Diese „Bessel-Korrektur“ trägt der Tatsache Rechnung, dass wir den Mittelwert aus denselben Daten schätzen, wodurch die Abweichungen etwas geringer sind, als sie vom wahren Mittelwert der Grundgesamtheit wären.

Schlüsseleigenschaften:

  • Var(c) = 0 – Konstante hat keine Varianz
  • Var(X + c) = Var(X) – das Hinzufügen einer Konstante ändert die Streuung
  • nicht Var(cX) = c² × Var(X) – Skalierung multipliziert die Varianz mit dem Quadrat
  • Var(X + Y) = Var(X) + Var(Y) – für unabhängiges X, Y
  • Var(X) = E[X²] - (E[X])² – alternative Definition unter Verwendung von Erwartungen

Schritt-für-Schritt-Anleitung

  1. Geben Sie Ihre Daten ein:Geben Sie durch Kommas getrennte Zahlen ein. Beispiel: 2, 4, 4, 4, 5, 5, 7, 9
  2. Klicken Sie auf Berechnen:Der Rechner berechnet Mittelwert und Varianz
  3. Lesen Sie Ihre Ergebnisse:Mittelwert = 5, Stichprobenvarianz = 4,57, Populationsvarianz = 4,0
  4. Optional:Ziehen Sie die Quadratwurzel für die Standardabweichung: √4,57 = 2,14
  5. Interpretieren:Höhere Varianz = mehr Streuung; datensatzübergreifend vergleichen

Reale Beispiele mit Berechnungen

Beispiel 1: Risiko des Anlageportfolios
Jährliche Rendite der Aktie A (5 Jahre): 10 %, 15 %, 8 %, 22 %, 5 %
Mittelwert: (10+15+8+22+5) / 5 = 60/5 = 12 %
Abweichungen: -2, 3, -4, 10, -7
Quadratische Abweichungen: 4, 9, 16, 100, 49
Sum: 178
Stichprobenvarianz: 178 / 4 = 44,5 (%²)
Standardabweichung: √44,5 = 6,67 %
Anwendung: Höhere Varianz = höheres Risiko. Vergleichen Sie mit Aktie B mit einer Varianz von 25 – Aktie A ist riskanter.

Beispiel 2: Vergleich des Herstellungsprozesses
Maschine A produziert Teile (mm): 10,1, 9,9, 10,0, 10,2, 9,8
Maschine B produziert Teile (mm): 10,3, 9,7, 10,5, 9,5, 10,0
Beide haben einen Mittelwert von 10,0 mm
Abweichung der Maschine A: 0,025 mm²
Abweichung Maschine B: 0,17 mm²
Anwendung: Maschine A hat eine 7-mal geringere Varianz – eine viel gleichmäßigere Qualität. Wählen Sie Maschine A für Präzisionsarbeiten.

Beispiel 3: A/B-Testanalyse
Konvertierungen der Kontrollgruppe (10 Tage): 45, 48, 52, 47, 50, 49, 51, 46, 53, 48
Mittelwert: 489/10 = 48,9
Varianz: 7,66
Behandlungsgruppe: 52, 55, 49, 58, 54, 51, 56, 53, 57, 50
Mittelwert: 535/10 = 53,5
Varianz: 9,17
Anwendung: Die Behandlung hat einen höheren Mittelwert (+4,6), aber auch eine höhere Varianz. Verwenden Sie den T-Test, um zu bestimmen, ob der Unterschied statistisch signifikant ist.

Beispiel 4: Kombination unabhängiger Risiken
Das Projekt hat 3 unabhängige Risikofaktoren:
Risiko-A-Varianz: 100 (Kostenüberschreitung)
Risiko-B-Varianz: 64 (Verzögerungsstrafe)
Risiko-C-Varianz: 36 (Nacharbeitskosten)
Gesamtvarianz: 100 + 64 + 36 = 200
Gesamt-SD: √200 = 14,14
Anwendung: Portfoliovarianz addiert sich für unabhängige Risiken. Aufgrund der Diversifizierung ist das Gesamtrisiko (SD) geringer als die Summe der einzelnen SDs.

Beispiel 5: Qualitätskontrollkarte
Historische Prozessvarianz: σ² = 2,25 (SD = 1,5)
Kontrollgrenzen: Mittelwert ± 3σ = Mittelwert ± 4,5
Neue Stichprobe: 8 Beobachtungen, Varianz = 6,8
Anwendung: Die Stichprobenvarianz (6,8) übersteigt den historischen Wert (2,25) um das Dreifache. Der Prozess ist möglicherweise instabil geworden. Untersuchen Sie zuordenbare Ursachen (neuer Lieferant, Geräteverschleiß, Bedienerwechsel).

4 häufige Fehler

  • Verwirrende Stichproben- vs. Populationsvarianz:Verwenden Sie (n-1) für Stichproben, N für Populationen. Die Stichprobenvarianz ist größer und gleicht die Schätzungsunsicherheit aus. Die meisten realen Analysen verwenden Stichproben, daher ist (n-1) Standard.
  • Varianz direkt interpretieren:Die Varianz wird in quadratischen Einheiten (Dollar², kg²) angegeben, die schwer zu interpretieren sind. Ziehen Sie die Quadratwurzel, um die Standardabweichung in Originaleinheiten zu erhalten. Varianz für mathematische Operationen nutzen; Verwenden Sie SD für die Kommunikation.
  • Varianzen korrelierter Variablen hinzufügen:Var(X + Y) = Var(X) + Var(Y) nur, wenn X und Y unabhängig (unkorreliert) sind. Für korrelierte Variablen: Var(X + Y) = Var(X) + Var(Y) + 2×Cov(X,Y). Durch das Ignorieren der Kovarianz wird die Gesamtvarianz unter- oder überschätzt.
  • Verwendung der Varianz für schiefe Verteilungen:Die Varianz geht von einer symmetrischen Streuung um den Mittelwert aus. Bei verzerrten Daten (Einkommen, Immobilienpreise) wird die Varianz von Extremwerten dominiert. Verwenden Sie den Interquartilbereich (IQR) oder die mittlere absolute Abweichung (MAD) für robuste Spread-Messungen.

4 Profi-Tipps

  • Für Handberechnungen verwenden Sie die Rechenformel:s² = [Σx² - (Σx)²/n] / (n-1). Für Daten 2, 4, 5, 7: Σx = 18, Σx² = 94, n = 4. Varianz = [94 – 324/4] / 3 = [94 – 81] / 3 = 13/3 = 4,33. Schneller als die Berechnung einzelner Abweichungen.
  • Varianzzerlegung anwenden (ANOVA-Prinzip):Gesamtvarianz = Varianz zwischen Gruppen + Varianz innerhalb der Gruppe. Dies zeigt, ob Unterschiede auf Gruppenzugehörigkeit oder zufällige Variationen innerhalb von Gruppen zurückzuführen sind.
  • Verwenden Sie die gepoolte Varianz zum Vergleichen von Gruppen:Wenn Sie zwei Stichproben mit ähnlichen Varianzen vergleichen, fassen Sie sie zusammen: s²_pooled = [(n₁-1)s₁² + (n₂-1)s₂²] / (n₁+n₂-2). Dies ergibt eine stabilere Schätzung für t-Tests.
  • Daten transformieren, um die Varianz zu stabilisieren:Wenn die Varianz mit dem Mittelwert zunimmt (üblich bei Zähldaten), wenden Sie eine Quadratwurzel- oder Logarithmustransformation an. Diese „Varianzstabilisierung“ macht statistische Tests valider. Beispiel: Für Poisson-Daten hat √x eine ungefähr konstante Varianz.

Verwenden Sie die Standardabweichung für: Berichtsergebnisse, Interpretation der Streuung, Konfidenzintervalle, Vergleich mit dem Mittelwert.
