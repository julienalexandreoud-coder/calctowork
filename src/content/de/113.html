Was ist ein Rechner für komplexe Zahlengrößen?

Der Betragsrechner für komplexe Zahlen ermittelt den Absolutwert (Modul) einer komplexen Zahl – ihren Abstand von Null in der komplexen Ebene. Komplexe Zahlen kombinieren Real- und Imaginärteile und werden als a + bi geschrieben, wobei i = √(-1). Die Größe gibt Auskunft über die „Größe“ der komplexen Zahl, die für Elektrotechnik, Quantenmechanik, Signalverarbeitung und Steuerungssysteme von wesentlicher Bedeutung ist. Ob bei der Analyse von Wechselstromkreisen, der Untersuchung von Wellenfunktionen oder dem Entwurf von Filtern, komplexe Größen sind von grundlegender Bedeutung.

Betrachten Sie die komplexe Zahl 3 + 4i. Sein Realteil ist 3, sein Imaginärteil ist 4. Auf der komplexen Ebene wird dies als Punkt dargestellt, der 3 Einheiten rechts und 4 Einheiten oberhalb des Ursprungs liegt. Der Betrag ist der geradlinige Abstand von (0,0) zu (3,4), berechnet nach dem Satz des Pythagoras: |3 + 4i| = √(3² + 4²) = √(9 + 16) = √25 = 5. Dieses berühmte rechtwinklige 3-4-5-Dreieck zeigt, wie komplexe Größen geometrisch funktionieren.

Komplexe Zahlen sind nicht „imaginär“ im Sinne von unwirklich – sie sind unverzichtbare Werkzeuge zur Modellierung von Schwingungen, Rotationen und Wellenphänomenen. Die Größe repräsentiert die Amplitude in der Signalverarbeitung, die Impedanz in Schaltkreisen und die Wahrscheinlichkeitsamplitude in der Quantenmechanik. Das Verständnis komplexer Größenordnungen erschließt Anwendungen in Wissenschaft und Technik.

Funktionsweise des Betragsrechners für komplexe Zahlen: Erklärung der Formeln

Größenformel:|a + bi| = √(a² + b²), wobei a der Realteil und b der Imaginärteil ist. Dies erweitert den Satz des Pythagoras auf die komplexe Ebene. Beispiel: |5 + 12i| = √(5² + 12²) = √(25 + 144) = √169 = 13. Ein weiterer: |-8 + 6i| = √((-8)² + 6²) = √(64 + 36) = √100 = 10. Der Betrag ist immer eine nicht negative reelle Zahl.

Konvertierung der Polarform:Jede komplexe Zahl kann als r(cos θ + i sin θ) oder re^(iθ) geschrieben werden, wobei r = |z| ist der Betrag und θ = arg(z) ist der Winkel (Argument). Für z = 3 + 4i: r = 5, θ = arctan(4/3) ≈ 53,13°. Polarform: 5(cos 53,13° + i sin 53,13°) oder 5e^(i·53,13°). Die Polarform vereinfacht die Multiplikation und Division komplexer Zahlen.

Produktmenge:|z₁ × z₂| = |z₁| × |z₂|. Die Größe eines Produkts entspricht dem Produkt der Größen. Beispiel: z₁ = 3 + 4i (|z₁| = 5), z₂ = 1 + i (|z₂| = √2). Produkt: (3+4i)(1+i) = 3 + 3i + 4i + 4i² = 3 + 7i - 4 = -1 + 7i. Betrag: |-1 + 7i| = √(1 + 49) = √50 = 5√2. Überprüfen Sie: |z₁| × |z₂| = 5 × √2 = 5√2 ✓.

Größeneigenschaften:Schlüsseleigenschaften vereinfachen Berechnungen: |z| ≥ 0 (immer nicht negativ). |z| = 0 genau dann, wenn z = 0. |-z| = |z| (Negation ändert die Größe nicht). |conjugate(z)| = |z| (Konjugat hat die gleiche Größe). |zⁿ| = |z|ⁿ (Größe der Kraft ist gleich Kraft der Größe). Diese Eigenschaften ermöglichen eine algebraische Manipulation ohne Konvertierung in eine rechteckige Form.

Durcharbeiten vollständiger Beispiele: Finden Sie |7 - 24i|. Realteil a = 7, Imaginärteil b = -24. |7 - 24i| = √(7² + (-24)²) = √(49 + 576) = √625 = 25. Finden Sie |-5 - 12i|. a = -5, b = -12. |-5 - 12i| = √(25 + 144) = √169 = 13. Finden Sie |6i| (reine Einbildung). a = 0, b = 6. |0 + 6i| = √(0 + 36) = 6. Finden Sie |10| (rein real). a = 10, b = 0. |10 + 0i| = √(100 + 0) = 10. Reine reelle und reine imaginäre Zahlen folgen derselben Formel.

Schritt-für-Schritt-Anleitung zum Ermitteln der Größe komplexer Zahlen

  1. Identifizieren Sie den Realteil (a).Der Realteil ist der Term ohne i. In 8 + 3i beträgt der Realteil 8. In -5 - 7i beträgt der Realteil -5. In 6i ist der Realteil 0 (kein Realteil geschrieben). Schreiben Sie eine explizit auf, einschließlich des Vorzeichens.
  2. Identifizieren Sie den Imaginärteil (b).Der Imaginärteil ist der Koeffizient von i (ohne i selbst). In 8 + 3i ist b = 3. In -5 - 7i ist b = -7 (das Minuszeichen ist Teil von b). In 6i ist b = 6. In 10 (rein reell) ist b = 0. Seien Sie vorsichtig: Der Imaginärteil ist die Zahl, die i multipliziert, ohne i.
  3. Quadrieren Sie beide Teile.Berechnen Sie a² und b². Beispiel: z = -8 + 15i. a = -8, b = 15. a² = (-8)² = 64. b² = 15² = 225. Denken Sie daran: Das Quadrieren eines Negativs ergibt ein Positives. (-8)² = 64, nicht -64. Aus diesem Grund ist die Größe immer nicht negativ.
  4. Fügen Sie die Quadrate hinzu.Berechnen Sie a² + b². Fortsetzung des Beispiels: 64 + 225 = 289. Diese Summe stellt den quadrierten Abstand vom Ursprung in der komplexen Ebene dar. Nach dem Satz des Pythagoras entspricht dies dem Quadrat der Größe.
  5. Ziehe die Quadratwurzel.|z| = √(a² + b²). Beispiel: √289 = 17. Der Betrag von -8 + 15i ist 17. Verwenden Sie einen Rechner für nicht perfekte Quadrate: |3 + 5i| = √(9 + 25) = √34 ≈ 5,831. Runden Sie auf die für Ihre Anwendung geeignete Präzision.
  6. Überprüfen Sie die Angemessenheit.Der Betrag sollte mindestens so groß sein wie der größere von |a| und |b|. Für 3 + 4i ist Magnitude 5 größer als 3 und 4 ✓. Für 10 + 0i beträgt die Größe 10 ✓. Wenn Ihre Größe kleiner als |a| ist oder |b|, neu berechnen – etwas ist schief gelaufen.

Beispiele für die Größenordnung komplexer Zahlen aus der Praxis

Beispiel 1: Impedanz eines Wechselstromkreises.Ein Wechselstromkreis hat einen Widerstand R = 3 Ohm und eine induktive Reaktanz X_L = 4 Ohm. Gesamtimpedanz Z = 3 + 4i Ohm (Widerstand ist real, Reaktanz ist imaginär). Betrag |Z| = √(3² + 4²) = 5 Ohm. Dies ist der effektive Widerstand, den der Stromkreis dem Wechselstrom entgegensetzt. Bei angelegter 120-V-Wechselspannung beträgt der Strom I = V/|Z| = 120/5 = 24 Ampere. Die Größe bestimmt den tatsächlichen Stromfluss, nicht die komplexe Impedanz selbst.

Beispiel 2: Signalverarbeitungsamplitude.Ein Signal wird im Frequenzbereich als komplexe Zahl 7 - 24i dargestellt. Die Größe |7 - 24i| = √(49 + 576) = √625 = 25 stellt die Amplitude (Stärke) des Signals dar. Der Phasenwinkel θ = arctan(-24/7) ≈ -73,7° stellt die Phasenverschiebung dar. Ingenieure kümmern sich um beides: Die Amplitude bestimmt die Lautstärke/Spannung, die Phase bestimmt die zeitlichen Beziehungen.

Beispiel 3: Wahrscheinlichkeit der Quantenmechanik.Ein Quantenzustand hat eine Wahrscheinlichkeitsamplitude ψ = 0,6 + 0,8i. Die Wahrscheinlichkeit, diesen Zustand zu messen, beträgt |ψ|² = (0,6)² + (0,8)² = 0,36 + 0,64 = 1,0 = 100 %. Dies ist ordnungsgemäß normalisiert. Ein anderer Zustand φ = 0,3 + 0,4i hat |φ|² = 0,09 + 0,16 = 0,25 = 25 % Wahrscheinlichkeit. Die quadrierte Größe ergibt eine messbare Wahrscheinlichkeit – komplexe Zahlen sind für die Quantentheorie von wesentlicher Bedeutung.

Beispiel 4: Stabilität des Kontrollsystems.Die Übertragungsfunktion eines Systems hat einen Pol bei s = -2 + 3i in der komplexen Ebene. Der Abstand vom Ursprung beträgt |s| = √(4 + 9) = √13 ≈ 3,61. Der Realteil (-2) zeigt einen exponentiellen Abfall an; Der Imaginärteil (3) zeigt eine Schwingung mit 3 rad/s an. Die Stärke 3,61 bezieht sich auf die Eigenfrequenz des Systems. Steuerungsingenieure analysieren Polgrößen, um das Systemverhalten vorherzusagen.

Beispiel 5: 2D-Vektorgröße.Ein Kraftvektor in 2D hat Komponenten F_x = 12 N (horizontal) und F_y = -5 N (vertikal). Als komplexe Zahl 12 - 5i darstellen. Größe |12 - 5i| = √(144 + 25) = √169 = 13 N. Dies ist die Gesamtkraftstärke. Richtung: θ = arctan(-5/12) ≈ -22,6° (22,6° unter der Horizontalen). Komplexe Zahlen ermöglichen elegante 2D-Vektorberechnungen.

Häufige Fehler bei komplexen Zahlengrößen

Ich habe vergessen, den imaginären Koeffizienten zu quadrieren, nicht i selbst.Falsch: |3 + 4i| = √(3² + 4i²) = √(9 - 4) = √5. Richtig: |3 + 4i| = √(3² + 4²) = √(9 + 16) = 5. Die Formel verwendet b², nicht (bi)². Das i ist nicht Teil dessen, was quadriert wird – b ist der Koeffizient. Betrachten Sie es als den Satz des Pythagoras: Real- und Imaginärteil sind senkrechte „Beine“ eines rechtwinkligen Dreiecks.

Beim Quadrieren das Minuszeichen weglassen.Für z = -5 - 12i sind beide Teile negativ. Richtig: |z| = √((-5)² + (-12)²) = √(25 + 144) = 13. Fehler: |z| = √(5² + 12²) – dies ergibt versehentlich die richtige Antwort, da das Vorzeichen durch die Quadrierung ohnehin eliminiert wird. Aber für das Argument (Winkel) sind Zeichen von entscheidender Bedeutung. Verfolgen Sie die Schilder stets korrekt.

Verwechslung der Größe mit der komplexen Zahl selbst.Die Größe ist eine reelle Zahl (Entfernung), keine komplexe Zahl. |3 + 4i| = 5, nicht 5 + 0i oder nur „5i“. Die Betragsoperation |z| gibt einen nicht negativen reellen Skalar aus. Schreiben Sie nicht |3 + 4i| = 5i oder |3 + 4i| = 3 + 4i – beides ist falsch. Die Größe entfernt die Richtung und lässt nur die Größe übrig.

Den Imaginärteil falsch identifizieren.In z = 5 + 3i ist der Imaginärteil 3, nicht 3i. In z = 7 - 2i ist der Imaginärteil -2, nicht -2i oder 2. Der Imaginärteil b ist der Realkoeffizient von i. Formel: |a + bi| = √(a² + b²), wobei b eine reelle Zahl ist. Es ist falsch, b = 3i zu schreiben und √(a² + (3i)²) = √(a² - 9) zu berechnen.

Profi-Tipps für komplexe Zahlengrößen

Verwenden Sie zur Division die konjugierte Multiplikation.Um |z₁/z₂| zu finden, verwenden Sie |z₁/z₂| = |z₁|/|z₂| anstatt komplexe Zahlen direkt zu dividieren. Beispiel: z₁ = 8 + 6i (|z₁| = 10), z₂ = 3 - 4i (|z₂| = 5). |z₁/z₂| = 10/5 = 2. Viel einfacher als die Berechnung von (8+6i)/(3-4i) = (8+6i)(3+4i)/(9+16) = (24+32i+18i-24)/25 = 50i/25 = 2i, dann |2i| = 2.

Erkennen Sie pythagoräische Tripel für eine schnelle Berechnung.Gemeinsame pythagoräische Tripel ergeben ganzzahlige Größen: 3-4-5, 5-12-13, 8-15-17, 7-24-25, 20-21-29. Die komplexen Zahlen 3+4i, 5+12i, 8+15i haben die Beträge 5, 13 bzw. 17. Das Auswendiglernen dieser Tripel beschleunigt manuelle Berechnungen und hilft bei der Überprüfung der Rechnerergebnisse.

Konvertieren Sie Potenzen und Wurzeln in die Polarform.Verwenden Sie für zⁿ die Polarform: Wenn z = re^(iθ), dann ist zⁿ = rⁿe^(inθ). Betrag |zⁿ| = rⁿ = |z|ⁿ. Beispiel: (3+4i)³ hat die Größe 5³ = 125. (3+4i)³ direkt berechnen: (3+4i)² = 9+24i-16 = -7+24i. Dann ist (-7+24i)(3+4i) = -21-28i+72i-96 = -117+44i. Betrag: √(117²+44²) = √(13689+1936) = √15625 = 125 ✓.

Wenden Sie die Dreiecksungleichung an.Für beliebige komplexe Zahlen: |z₁ + z₂| ≤ |z₁| + |z₂|. Die Größe einer Summe ist höchstens die Summe der Größen. Beispiel: z₁ = 3+4i (|z₁|=5), z₂ = 5-12i (|z₂|=13). z₁+z₂ = 8-8i, |z₁+z₂| = √128 ≈ 11,31. Summe der Größen: 5+13 = 18. Tatsächlich gilt 11,31 ≤ 18. Gleichheit gilt nur, wenn z₁ und z₂ in die gleiche Richtung zeigen.

Häufig gestellte Fragen

Für reine reelle Zahlen (b=0): |a + 0i| = √(a² + 0) = |a|, der gewöhnliche Absolutwert. Für reine imaginäre Zahlen (a=0): |0 + bi| = √(0 + b²) = |b|. Beispiele: |7| = 7, |-5| = 5, |6i| = 6, |-9i| = 9. Komplexe Größe verallgemeinert den realen Absolutwert auf zwei Dimensionen.

Nein. Die Größe ist ein Abstand, und der Abstand ist immer nicht negativ. |z| ≥ 0 für alle komplexen Zahlen z. Die einzige komplexe Zahl mit der Größe 0 ist z = 0 (sowohl Real- als auch Imaginärteil sind Null). Wenn Ihre Berechnung eine negative Größe ergibt, haben Sie einen Fehler gemacht – wahrscheinlich haben Sie vergessen, die Quadratwurzel zu ziehen, oder Sie haben die Vorzeichen falsch behandelt.

In der Polarform z = r(cos θ + i sin θ) oder z = re^(iθ) ist die Größe einfach r – sie ist in die Darstellung eingebaut. Für z = 5e^(i·π/3) beträgt der Betrag 5. Für z = 2,5(cos 60° + i sin 60°) beträgt der Betrag 2,5. Die Polarform trennt die Größe (r) explizit von der Richtung (θ), wodurch die Größe sofort sichtbar wird.

Auf der komplexen Ebene (reale Achse horizontal, imaginäre Achse vertikal) wird eine komplexe Zahl a + bi als Punkt (a, b) aufgetragen. Der Betrag |a + bi| = √(a² + b²) ist der euklidische Abstand vom Ursprung (0,0) zum Punkt (a, b). Alle komplexen Zahlen gleicher Größe liegen auf einem Kreis mit Mittelpunkt im Ursprung – dieser Kreis wird „Niveaukurve“ konstanter Größe genannt.
