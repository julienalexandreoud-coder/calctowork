Datos de entrada

  • precio_original en EUR (valor por defecto: 100)
  • Primer descuento % en % (valor por defecto: 20)
  • Segundo descuento % en % (valor por defecto: 10)

Resultados

  • Precio final en EUR
  • Ahorro total en EUR
  • Ahorro % en %

Fórmula: Calculate double discount based on input values.

Ejemplo de uso: precio_original: 100 EUR, Primer descuento %: 20 %, Segundo descuento %: 10 %.

Desglose del cálculo

  1. Enter input values: precio_original: 100, descuento1: 20, descuento2: 10
  2. Calculate and display the final_price result
  3. Calculate and display the total_savings result
  4. Calculate and display the savings_pct result

Errores frecuentes

  • Confusing unit options when entering values
  • Using incorrect input values outside recommended range
