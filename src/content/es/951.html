Calculadora de una repetición máxima (1RM): fuerza máxima estimada

Esta calculadora estima tu repetición máxima (1RM), el peso máximo que puedes levantar una sola vez, basándose en el peso que puedes levantar para múltiples repeticiones usando la fórmula de Epley.

Estima tu repeticion maxima (1RM) usando la formula de Brzycki. Esencial para progresar en pesas. Con precision de 10+ decimales.

Datos de entrada

  • Peso (kg) en kg (valor por defecto: 80)
  • Repeticiones (valor por defecto: 5)

Resultados

  • 1RM en kg
  • 90% en kg
  • 80% en kg
  • 70% en kg

Fórmula: Calculate one rep max based on input values.

Ejemplo de uso: Peso (kg): 80 kg, Repeticiones: 5 .

Desglose del cálculo

  1. Enter input values: peso_kg: 80, repeticiones: 5
  2. Calculate and display the one_rep_max result
  3. Calculate and display the at_90pct result
  4. Calculate and display the at_80pct result
  5. Calculate and display the at_70pct result

Errores frecuentes

  • Confusing unit options when entering values
  • Entering zero or negative values where positive required
  • Using incorrect input values outside recommended range
