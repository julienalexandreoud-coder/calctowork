¿Qué es la Calculadora de Calorías Quemadas?

Calcula Calorías = MET × Peso (kg) × Tiempo (h). Ejemplo: Correr (MET 9,8), 70 kg, 45 min = Calorías quemadas ≈ 514 kcal. Calculadora gratuita y precisa, sin registro.

Cómo usar esta calculadora

Introduce los datos:

  • Peso (kg)
  • Duración (min)
  • Intensidad MET

Resultados:

  • Calorías quemadas
  • Calorías por hora

Cómo funciona el cálculo

Fórmula: Calculate calorias ejercicio based on input values.

Errores comunes a evitar

  • Using incorrect input values outside recommended range
