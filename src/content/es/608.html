Cómo funciona el cálculo

Fórmula: s² = Σ(xᵢ - x̄)² / (n-1) (muestra) o Σ(xᵢ - x̄)² / n (población)

Ejemplo paso a paso

  1. Calcular la media
  2. Restar la media a cada valor
  3. Elevar cada diferencia al cuadrado
  4. Sumar y dividir entre n-1 o n

Errores comunes a evitar

  • Menos de 2 valores
  • Confundir varianza de muestra vs población
  • No elevar al cuadrado las diferencias

Consejos útiles

  • Verifica siempre las unidades antes de calcular
  • Compara resultados con otras fuentes cuando sea posible
  • Guarda tus cálculos para referencia futura
  • Esta herramienta es gratuita, sin registro
