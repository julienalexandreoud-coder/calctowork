Calculadora de relación cintura-altura (WHtR): indicador de salud

La relación cintura-altura (Waist-to-Height Ratio, WHtR) es un indicador simple que compara tu perímetro de cintura con tu estatura. Esta calculadora evalúa tu riesgo de salud basado en esta proporción.

¿Qué es la Calculadora de Índice Cintura-Estatura?

Calcula tu indice cintura-estatura, mejor predictor de riesgo cardiovascular que el IMC. Con rangos de referencia y contexto de salud.

Cómo usar esta calculadora

Introduce los datos:

  • Altura (cm)
  • cintura_cm

Resultados:

  • Indice cintura-cadera
  • Riesgo
  • Ideal

Cómo funciona el cálculo

Fórmula: Calculate waist height ratio based on input values.

Ejemplo paso a paso

  1. Enter input values: altura_cm: 175, cintura_cm: 80
  2. Calculate and display the whr result
  3. Calculate and display the risk result
  4. Calculate and display the ideal result

Errores comunes a evitar

  • Confusing unit options when entering values
  • Entering zero or negative values where positive required
  • Using incorrect input values outside recommended range

Consejos útiles
