¿Qué es la variación?

La varianza mide qué tan disperso está un conjunto de datos calculando el promedio de las desviaciones al cuadrado de la media. Cuantifica la variabilidad: una variación alta significa que los puntos de datos están muy dispersos; Una varianza baja significa que se agrupan estrechamente alrededor del promedio. La varianza es la base de la desviación estándar, ANOVA, análisis de regresión y teoría de carteras.

La varianza se expresa en unidades al cuadrado (por ejemplo, dólares², metros²), lo que la hace menos intuitiva que la desviación estándar pero matemáticamente esencial. La raíz cuadrada de la varianza es igual a la desviación estándar, lo que devuelve la medida a las unidades originales.

Información clave:La varianza tiene una propiedad única: las varianzas de las variables independientes se suman. Si X e Y son independientes, Var(X + Y) = Var(X) + Var(Y). Esto hace que la varianza sea indispensable para combinar incertidumbres en física, finanzas e ingeniería.

Fórmulas explicadas

Varianza muestral (más común):
s² = Σ(xᵢ - x̄)² / (n - 1)

Variación de la población:
σ² = Σ(xᵢ - μ)² / N

Fórmula computacional (más fácil para los cálculos):
s² = [Σxᵢ² - (Σxᵢ)²/n] / (n - 1)

Where:
xᵢ = cada valor individual
x̄ = media muestral
μ = media poblacional
n = tamaño de la muestra
N = tamaño de la población
Σ = suma de

¿Por qué (n-1) para muestras?
Usar (n-1) en lugar de n proporciona una estimación insesgada de la varianza poblacional. Esta "corrección de Bessel" explica el hecho de que estimamos la media a partir de los mismos datos, lo que hace que las desviaciones sean ligeramente menores de lo que serían con respecto a la media poblacional real.

Propiedades clave:

  • Var(c) = 0 — la constante no tiene varianza
  • Var(X + c) = Var(X) — agregar constante no cambia la dispersión
  • Var(cX) = c² × Var(X) — el escalado multiplica la varianza por el cuadrado
  • Var(X + Y) = Var(X) + Var(Y) — para X independiente, Y
  • Var(X) = E[X²] - (E[X])² — definición alternativa utilizando expectativas

Guía paso a paso

  1. Introduce tus datos:Introduzca números separados por comas. Ejemplo: 2, 4, 4, 4, 5, 5, 7, 9
  2. Haga clic en calcular:La calculadora calcula la media y la varianza
  3. Lea sus resultados:Media = 5, Varianza muestral = 4,57, Varianza poblacional = 4,0
  4. Optional:Saque la raíz cuadrada para la desviación estándar: √4,57 = 2,14
  5. Interpretar:Mayor varianza = más dispersión; comparar entre conjuntos de datos

Ejemplos reales con cálculos

Ejemplo 1: Riesgo de cartera de inversiones
Rentabilidad anual de la acción A (5 años): 10%, 15%, 8%, 22%, 5%
Media: (10+15+8+22+5) / 5 = 60/5 = 12%
Desviaciones: -2, 3, -4, 10, -7
Desviaciones al cuadrado: 4, 9, 16, 100, 49
Sum: 178
Varianza muestral: 178 / 4 = 44,5 (%²)
Desviación estándar: √44,5 = 6,67%
Aplicación: Mayor varianza = mayor riesgo. Compare con la acción B con varianza = 25: la acción A es más riesgosa.

Ejemplo 2: Comparación del proceso de fabricación
La máquina A produce piezas (mm): 10,1, 9,9, 10,0, 10,2, 9,8
La máquina B produce piezas (mm): 10,3, 9,7, 10,5, 9,5, 10,0
Ambos tienen media = 10,0 mm
Variación de la máquina A: 0,025 mm²
Variación de la máquina B: 0,17 mm²
Aplicación: La máquina A tiene una variación 7 veces menor: una calidad mucho más consistente. Elija la Máquina A para trabajos de precisión.

Ejemplo 3: Análisis de prueba A/B
Conversiones del grupo de control (10 días): 45, 48, 52, 47, 50, 49, 51, 46, 53, 48
Media: 489/10 = 48,9
Variación: 7,66
Grupo de tratamiento: 52, 55, 49, 58, 54, 51, 56, 53, 57, 50
Media: 535/10 = 53,5
Variación: 9,17
Aplicación: El tratamiento tiene una media más alta (+4,6) pero también una varianza más alta. Utilice la prueba t para determinar si la diferencia es estadísticamente significativa.

Ejemplo 4: Combinación de riesgos independientes
El proyecto tiene 3 factores de riesgo independientes:
Variación del riesgo A: 100 (sobrecoste)
Variación del riesgo B: 64 (penalización por retraso)
Variación del riesgo C: 36 (costo de retrabajo)
Varianza total: 100 + 64 + 36 = 200
DE total: √200 = 14,14
Aplicación: La variación de la cartera se suma para riesgos independientes. El riesgo total (DE) es menor que la suma de los DE individuales debido a la diversificación.

Ejemplo 5: Gráfico de control de calidad
Varianza histórica del proceso: σ² = 2,25 (SD = 1,5)
Límites de control: media ± 3σ = media ± 4,5
Nueva muestra: 8 observaciones, varianza = 6,8
Aplicación: La varianza muestral (6,8) supera la histórica (2,25) en 3 veces. Es posible que el proceso se haya vuelto inestable. Investigar causas asignables (nuevo proveedor, desgaste de equipos, cambio de operador).

4 errores comunes

  • Varianza confusa entre muestra y población:Utilice (n-1) para muestras, N para poblaciones. La varianza muestral es mayor, lo que compensa la incertidumbre de la estimación. La mayoría de los análisis del mundo real utilizan muestras, por lo que (n-1) es el estándar.
  • Interpretar la varianza directamente:La varianza está en unidades al cuadrado (dólares², kg²), que son difíciles de interpretar. Saca la raíz cuadrada para obtener la desviación estándar en las unidades originales. Utilice la varianza para operaciones matemáticas; Utilice SD para la comunicación.
  • Sumar varianzas de variables correlacionadas:Var(X + Y) = Var(X) + Var(Y) sólo cuando X e Y son independientes (no correlacionados). Para variables correlacionadas: Var(X + Y) = Var(X) + Var(Y) + 2×Cov(X,Y). Ignorar la covarianza subestima o sobreestima la varianza total.
  • Uso de la varianza para distribuciones asimétricas:La varianza supone una dispersión simétrica alrededor de la media. Para datos asimétricos (ingresos, precios de la vivienda), la varianza está dominada por valores extremos. Utilice el rango intercuartil (IQR) o la desviación absoluta mediana (MAD) para obtener medidas de diferencial sólidas.

4 consejos profesionales

  • Utilice la fórmula computacional para cálculos manuales:s² = [Σx² - (Σx)²/n] / (n-1). Para los datos 2, 4, 5, 7: Σx = 18, Σx² = 94, n = 4. Varianza = [94 - 324/4] / 3 = [94 - 81] / 3 = 13/3 = 4,33. Más rápido que calcular desviaciones individuales.
  • Aplicar descomposición de varianza (principio ANOVA):Varianza total = varianza entre grupos + varianza dentro del grupo. Esto revela si las diferencias provienen de la pertenencia a un grupo o de variaciones aleatorias dentro de los grupos.
  • Utilice la varianza agrupada para comparar grupos:Al comparar dos muestras con varianzas similares, combínelas: s²_pooled = [(n₁-1)s₁² + (n₂-1)s₂²] / (n₁+n₂-2). Esto da una estimación más estable para las pruebas t.
  • Transformar datos para estabilizar la varianza:Si la varianza aumenta con la media (común en los datos de recuento), aplique la raíz cuadrada o la transformación logarítmica. Esta "estabilización de la varianza" hace que las pruebas estadísticas sean más válidas. Ejemplo: para los datos de Poisson, √x tiene una varianza aproximadamente constante.

Utilice la desviación estándar para: informar resultados, interpretar la dispersión, intervalos de confianza y comparar con la media. La variación es para cálculos; SD es para comunicación.
