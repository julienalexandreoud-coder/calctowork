Calcula C(n,k) = n! / (k! × (n-k)!). Ejemplo: ¿De cuántas formas elegir 3 personas de un grupo de 10? = C(10,3) = 120. Calculadora gratuita y precisa, sin registro.

Datos de entrada

  • Elementos elegidos (r) (valor por defecto: 3)

Fórmula: C(n,r) = n! / (r! × (n-r)!)

Ejemplo de uso: Total de elementos (n): 10 , Elementos elegidos (r): 3 .

Desglose del cálculo

  1. Verificar que r ≤ n
  2. Calcular el factorial de n
  3. Calcular el factorial de r y (n-r)
  4. Aplicar la fórmula de combinaciones

Errores frecuentes

  • Usar r mayor que n
  • Confundir combinaciones con permutaciones
  • Valores negativos
