¿Qué es la Calculadora Combinaciones?

Calcula C(n,k) = n! / (k! × (n-k)!). Ejemplo: ¿De cuántas formas elegir 3 personas de un grupo de 10? = C(10,3) = 120. Calculadora gratuita y precisa, sin registro.

Cómo usar esta calculadora

Introduce los datos:

  • Total de elementos (n)
  • Elementos elegidos (r)

Resultados:

  • Resultado

Cómo funciona el cálculo

Fórmula: C(n,r) = n! / (r! × (n-r)!)

Ejemplo paso a paso

  1. Verificar que r ≤ n
  2. Calcular el factorial de n
  3. Calcular el factorial de r y (n-r)
  4. Aplicar la fórmula de combinaciones

Errores comunes a evitar

  • Usar r mayor que n
  • Confundir combinaciones con permutaciones
  • Valores negativos

Consejos útiles

  • Verifica siempre las unidades antes de calcular
  • Compara resultados con otras fuentes cuando sea posible
  • Guarda tus cálculos para referencia futura
  • Esta herramienta es gratuita, sin registro
