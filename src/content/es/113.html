¿Qué es la calculadora de magnitud de números complejos?

La Calculadora de magnitud de números complejos encuentra el valor absoluto (módulo) de un número complejo: su distancia desde cero en el plano complejo. Los números complejos combinan partes reales e imaginarias, escritas como a + bi donde i = √(-1). La magnitud indica el "tamaño" del número complejo, esencial para la ingeniería eléctrica, la mecánica cuántica, el procesamiento de señales y los sistemas de control. Ya sea analizando circuitos de CA, estudiando funciones de onda o diseñando filtros, la magnitud compleja es fundamental.

Considere el número complejo 3 + 4i. Su parte real es 3, la parte imaginaria es 4. En el plano complejo, esto se traza como un punto 3 unidades hacia la derecha y 4 unidades hacia arriba desde el origen. La magnitud es la distancia en línea recta de (0,0) a (3,4), calculada utilizando el teorema de Pitágoras: |3 + 4i| = √(3² + 4²) = √(9 + 16) = √25 = 5. Este famoso triángulo rectángulo 3-4-5 demuestra cómo funciona geométricamente la magnitud compleja.

Los números complejos no son "imaginarios" en el sentido de irreales: son herramientas indispensables para modelar oscilaciones, rotaciones y fenómenos ondulatorios. La magnitud representa la amplitud en el procesamiento de señales, la impedancia en los circuitos y la amplitud de probabilidad en la mecánica cuántica. Comprender la magnitud compleja desbloquea aplicaciones en la ciencia y la ingeniería.

Cómo funciona la calculadora de magnitud de números complejos: explicación de las fórmulas

Fórmula de magnitud:|a + bi| = √(a² + b²), donde a es la parte real y b es la parte imaginaria. Esto extiende el teorema de Pitágoras al plano complejo. Ejemplo: |5 + 12i| = √(5² + 12²) = √(25 + 144) = √169 = 13. Otro: |-8 + 6i| = √((-8)² + 6²) = √(64 + 36) = √100 = 10. La magnitud es siempre un número real no negativo.

Conversión de forma polar:Cualquier número complejo se puede escribir como r(cos θ + i sin θ) o re^(iθ), donde r = |z| es la magnitud y θ = arg(z) es el ángulo (argumento). Para z = 3 + 4i: r = 5, θ = arctan(4/3) ≈ 53,13°. Forma polar: 5(cos 53,13° + i sen 53,13°) o 5e^(i·53,13°). La forma polar simplifica la multiplicación y división de números complejos.

Magnitud del producto:|z₁ × z₂| = |z₁| × |z₂|. La magnitud de un producto es igual al producto de magnitudes. Ejemplo: z₁ = 3 + 4i (|z₁| = 5), z₂ = 1 + i (|z₂| = √2). Producto: (3+4i)(1+i) = 3 + 3i + 4i + 4i² = 3 + 7i - 4 = -1 + 7i. Magnitud: |-1 + 7i| = √(1 + 49) = √50 = 5√2. Verificar: |z₁| × |z₂| = 5 × √2 = 5√2 ✓.

Propiedades de magnitud:Las propiedades clave simplifican los cálculos: |z| ≥ 0 (siempre no negativo). |z| = 0 si y sólo si z = 0. |-z| = |z| (la negación no cambia la magnitud). |conjugado(z)| = |z| (el conjugado tiene la misma magnitud). |zⁿ| = |z|ⁿ (la magnitud de la potencia es igual a la potencia de la magnitud). Estas propiedades permiten la manipulación algebraica sin convertirla a forma rectangular.

Trabajando con ejemplos completos: Encuentre |7 - 24i|. Parte real a = 7, parte imaginaria b = -24. |7 - 24i| = √(7² + (-24)²) = √(49 + 576) = √625 = 25. Encuentra |-5 - 12i|. a = -5, b = -12. |-5-12i| = √(25 + 144) = √169 = 13. Encuentra |6i| (puro imaginario). a = 0, b = 6. |0 + 6i| = √(0 + 36) = 6. Encuentra |10| (puro real). a = 10, b = 0. |10 + 0i| = √(100 + 0) = 10. Los números reales puros y los imaginarios puros siguen la misma fórmula.

Guía paso a paso para encontrar la magnitud de números complejos

  1. Identifica la parte real (a).La parte real es el término sin i. En 8 + 3i, la parte real es 8. En -5 - 7i, la parte real es -5. En 6i, la parte real es 0 (no se escribe ningún término real). Escriba una explícitamente, incluido el signo.
  2. Identifica la parte imaginaria (b).La parte imaginaria es el coeficiente de i (sin incluir i en sí). En 8 + 3i, b = 3. En -5 - 7i, b = -7 (el signo menos es parte de b). En 6i, b = 6. En 10 (real puro), b = 0. Ojo: la parte imaginaria es el número que multiplica i, sin incluir i.
  3. Cuadre ambas partes.Calcula a² y b². Ejemplo: z = -8 + 15i. a = -8, b = 15. a² = (-8)² = 64. b² = 15² = 225. Recuerda: elevar al cuadrado un negativo da un positivo. (-8)² = 64, no -64. Por eso la magnitud siempre es no negativa.
  4. Agrega los cuadrados.Calcula a² + b². Siguiendo con el ejemplo: 64 + 225 = 289. Esta suma representa la distancia al cuadrado desde el origen en el plano complejo. Según el teorema de Pitágoras, esto es igual al cuadrado de la magnitud.
  5. Saca la raíz cuadrada.|z| = √(a² + b²). Ejemplo: √289 = 17. La magnitud de -8 + 15i es 17. Usa una calculadora para cuadrados no perfectos: |3 + 5i| = √(9 + 25) = √34 ≈ 5,831. Redondee a la precisión adecuada para su aplicación.
  6. Verificar la razonabilidad.La magnitud debe ser al menos tan grande como el mayor de |a| y |b|. Para 3 + 4i, la magnitud 5 es mayor que 3 y 4 ✓. Para 10 + 0i, la magnitud es igual a 10 ✓. Si su magnitud es menor que |a| o |b|, recalcular: algo salió mal.

Ejemplos de magnitud de números complejos del mundo real

Ejemplo 1: Impedancia del circuito de CA.Un circuito de CA tiene una resistencia R = 3 ohmios y una reactancia inductiva X_L = 4 ohmios. Impedancia total Z = 3 + 4i ohmios (la resistencia es real, la reactancia es imaginaria). Magnitud |Z| = √(3² + 4²) = 5 ohmios. Esta es la resistencia efectiva que presenta el circuito a la corriente alterna. Con 120 VCA aplicados, la corriente I = V/|Z| = 120/5 = 24 amperios. La magnitud determina el flujo de corriente real, no la impedancia compleja en sí.

Ejemplo 2: Amplitud de procesamiento de señales.Una señal se representa como el número complejo 7 - 24i en el dominio de la frecuencia. La magnitud |7 - 24i| = √(49 + 576) = √625 = 25 representa la amplitud (intensidad) de la señal. El ángulo de fase θ = arctan(-24/7) ≈ -73,7° representa el cambio de fase. Los ingenieros se preocupan por ambos: la amplitud determina el volumen/voltaje, la fase determina las relaciones de sincronización.

Ejemplo 3: Probabilidad de la Mecánica Cuántica.Un estado cuántico tiene una amplitud de probabilidad ψ = 0,6 + 0,8i. La probabilidad de medir este estado es |ψ|² = (0,6)² + (0,8)² = 0,36 + 0,64 = 1,0 = 100%. Esto está adecuadamente normalizado. Otro estado φ = 0,3 + 0,4i tiene |φ|² = 0,09 + 0,16 = 0,25 = 25% de probabilidad. La magnitud al cuadrado proporciona una probabilidad mensurable: los números complejos son esenciales para la teoría cuántica.

Ejemplo 4: Estabilidad del sistema de control.La función de transferencia de un sistema tiene un polo en s = -2 + 3i en el plano complejo. La distancia desde el origen es |s| = √(4 + 9) = √13 ≈ 3,61. La parte real (-2) indica decaimiento exponencial; la parte imaginaria (3) indica una oscilación a 3 rad/s. La magnitud 3,61 se relaciona con la frecuencia natural del sistema. Los ingenieros de control analizan las magnitudes de los polos para predecir el comportamiento del sistema.

Ejemplo 5: Magnitud vectorial 2D.Un vector de fuerza en 2D tiene componentes F_x = 12 N (horizontal) y F_y = -5 N (vertical). Representar como número complejo 12 - 5i. Magnitud |12 - 5i| = √(144 + 25) = √169 = 13 N. Esta es la fuerza total. Dirección: θ = arctan(-5/12) ≈ -22,6° (22,6° por debajo de la horizontal). Los números complejos proporcionan elegantes cálculos vectoriales 2D.

Errores comunes con la magnitud de números complejos

Olvidar elevar al cuadrado el coeficiente imaginario, no el propio i.Incorrecto: |3 + 4i| = √(3² + 4i²) = √(9 - 4) = √5. Correcto: |3 + 4i| = √(3² + 4²) = √(9 + 16) = 5. La fórmula usa b², no (bi)². La i no es parte de lo que se eleva al cuadrado: b es el coeficiente. Piense en ello como el teorema de Pitágoras: las partes real e imaginaria son "catetos" perpendiculares de un triángulo rectángulo.

Eliminando el signo negativo al elevar al cuadrado.Para z = -5 - 12i, ambas partes son negativas. Correcto: |z| = √((-5)² + (-12)²) = √(25 + 144) = 13. Error: |z| = √(5² + 12²): esto accidentalmente da la respuesta correcta porque elevar al cuadrado elimina el signo de todos modos. Pero para el argumento (ángulo), los signos tienen una importancia crítica. Siga siempre las señales correctamente en todo momento.

Confundir magnitud con el propio número complejo.La magnitud es un número real (distancia), no un número complejo. |3 + 4i| = 5, no 5 + 0i o simplemente "5i". La operación de magnitud |z| genera un escalar real no negativo. No escribas |3 + 4i| = 5i o |3 + 4i| = 3 + 4i: ambos están equivocados. La magnitud elimina la dirección, dejando sólo el tamaño.

Identificar erróneamente la parte imaginaria.En z = 5 + 3i, la parte imaginaria es 3, no 3i. En z = 7 - 2i, la parte imaginaria es -2, no -2i o 2. La parte imaginaria b es el coeficiente real de i. Fórmula: |a + bi| = √(a² + b²), donde b es un número real. Escribir b = 3i y calcular √(a² + (3i)²) = √(a² - 9) es incorrecto.

Consejos profesionales para la magnitud de números complejos

Utilice la multiplicación conjugada para la división.Para encontrar |z₁/z₂|, use |z₁/z₂| = |z₁|/|z₂| en lugar de dividir números complejos directamente. Ejemplo: z₁ = 8 + 6i (|z₁| = 10), z₂ = 3 - 4i (|z₂| = 5). |z₁/z₂| = 10/5 = 2. Mucho más simple que calcular (8+6i)/(3-4i) = (8+6i)(3+4i)/(9+16) = (24+32i+18i-24)/25 = 50i/25 = 2i, entonces |2i| = 2.

Reconocer ternas pitagóricas para realizar cálculos rápidos.Las ternas pitagóricas comunes dan magnitudes enteras: 3-4-5, 5-12-13, 8-15-17, 7-24-25, 20-21-29. Los números complejos 3+4i, 5+12i, 8+15i tienen magnitudes 5, 13, 17 respectivamente. Memorizar estos triples acelera los cálculos manuales y ayuda a verificar los resultados de la calculadora.

Convierta a forma polar para potencias y raíces.Para zⁿ, use la forma polar: si z = re^(iθ), entonces zⁿ = rⁿe^(inθ). Magnitud |zⁿ| = rⁿ = |z|ⁿ. Ejemplo: (3+4i)³ tiene magnitud 5³ = 125. Calculando (3+4i)³ directamente: (3+4i)² = 9+24i-16 = -7+24i. Entonces (-7+24i)(3+4i) = -21-28i+72i-96 = -117+44i. Magnitud: √(117²+44²) = √(13689+1936) = √15625 = 125 ✓.

Aplica la desigualdad del triángulo.Para cualquier número complejo: |z₁ + z₂| ≤ |z₁| + |z₂|. La magnitud de una suma es como máximo la suma de magnitudes. Ejemplo: z₁ = 3+4i (|z₁|=5), z₂ = 5-12i (|z₂|=13). z₁+z₂ = 8-8i, |z₁+z₂| = √128 ≈ 11,31. Suma de magnitudes: 5+13 = 18. De hecho, 11,31 ≤ 18. La igualdad se cumple sólo cuando z₁ y z₂ apuntan en la misma dirección.
