Cómo usar esta calculadora

Introduce los datos:

  • Media muestral
  • Desviación estándar
  • Total de elementos (n)
  • Nivel de confianza (%)

Resultados:

  • Límite inferior
  • Límite superior
  • Margen de error
  • Error estándar

Cómo funciona el cálculo

Fórmula: IC = media ± (z × σ/√n)

Ejemplo paso a paso

  1. Determinar el nivel de confianza y valor z
  2. Calcular el error estándar (σ/√n)
  3. Multiplicar z por el error estándar
  4. Sumar y restar a la media

Errores comunes a evitar

  • Muestra menor a 2
  • Desviación estándar en cero
  • Nivel de confianza fuera de rango

Consejos útiles

  • Verifica siempre las unidades antes de calcular
  • Compara resultados con otras fuentes cuando sea posible
  • Guarda tus cálculos para referencia futura
  • Esta herramienta es gratuita, sin registro
