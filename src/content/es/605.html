Calcula P(n,k) = n! / (n-k)!. Ejemplo: ¿Cuántas formas de ordenar 3 libros de una estantería de 5? = P(5,3) = 60. Calculadora gratuita y precisa, sin registro.

Datos de entrada

  • Total de elementos (n) (valor por defecto: 10)
  • Elementos elegidos (r) (valor por defecto: 3)

Resultados

  • Resultado
  • Fórmula

Fórmula: P(n,r) = n! / (n-r)!

Ejemplo de uso: Total de elementos (n): 10 , Elementos elegidos (r): 3 .

Desglose del cálculo

  1. Verificar que r ≤ n
  2. Calcular el factorial de n
  3. Calcular el factorial de (n-r)
  4. Dividir n! entre (n-r)!

Errores frecuentes

  • Usar r mayor que n
  • Confundir permutaciones con combinaciones
  • Valores negativos
