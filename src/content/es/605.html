¿Qué es la Calculadora Permutaciones?

Calcula P(n,k) = n! / (n-k)!. Ejemplo: ¿Cuántas formas de ordenar 3 libros de una estantería de 5? = P(5,3) = 60. Calculadora gratuita y precisa, sin registro.

Cómo usar esta calculadora

Introduce los datos:

  • Total de elementos (n)
  • Elementos elegidos (r)

Resultados:

  • Resultado
  • Fórmula

Cómo funciona el cálculo

Fórmula: P(n,r) = n! / (n-r)!

Ejemplo paso a paso

  1. Verificar que r ≤ n
  2. Calcular el factorial de n
  3. Calcular el factorial de (n-r)
  4. Dividir n! entre (n-r)!

Errores comunes a evitar

  • Usar r mayor que n
  • Confundir permutaciones con combinaciones
  • Valores negativos

Consejos útiles

  • Verifica siempre las unidades antes de calcular
  • Compara resultados con otras fuentes cuando sea posible
  • Guarda tus cálculos para referencia futura
  • Esta herramienta es gratuita, sin registro
