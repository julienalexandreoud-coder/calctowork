Qu'est-ce que le calculateur d'approximation intégrale de règle trapézoïdale ?

Le calculateur d'approximation intégrale de règle trapézoïdale estime l'aire sous une courbe en la divisant en trapèzes et en additionnant leurs aires. Lorsqu'une intégration exacte est impossible ou peu pratique, les méthodes numériques comme la règle trapézoïdale fournissent des approximations précises. Cette technique apparaît dans l'ingénierie (calcul du travail à partir de courbes de force), la physique (trouver un déplacement à partir de la vitesse), l'économie (calcul du surplus du consommateur) et les statistiques (trouver des probabilités à partir de fonctions de densité).

Envisagez de trouver l'aire sous f(x) = x² de x = 0 à x = 4. L'intégrale exacte est 64/3 ≈ 21,333. En utilisant la règle du trapèze à 4 sous-intervalles : divisez [0,4] en intervalles de largeur h = 1. Évaluez : f(0) = 0, f(1) = 1, f(2) = 4, f(3) = 9, f(4) = 16. Appliquez la formule : Aire ≈ (h/2)[f(0) + 2f(1) + 2f(2) + 2f(3) + f(4)] = (1/2)[0 + 2(1) + 2(4) + 2(9) + 16] = (1/2)[0 + 2 + 8 + 18 + 16] = (1/2)(44) = 22. L'approximation 22 est à moins de 3 % de la valeur exacte 21,333.

La règle trapézoïdale fonctionne en approchant la courbe comme une série de lignes droites connectées, formant des trapèzes. L'aire de chaque trapèze est (h/2)(y₁ + y₂), où h est la largeur et y₁, y₂ sont les valeurs de fonction aux extrémités. La somme de tous les trapèzes donne la superficie totale. Un plus grand nombre de sous-intervalles (trapèzes plus petits) produisent de meilleures approximations — les trapèzes suivent la courbe de plus près.

Fonctionnement du calculateur de règle trapézoïdale : explication des formules

Formule de règle trapézoïdale :∫[a,b] f(x)dx ≈ (h/2)[f(x₀) + 2f(x₁) + 2f(x₂) + ... + 2f(xₙ₋₁) + f(xₙ)], où h = (b-a)/n est la largeur du sous-intervalle, n est le nombre de sous-intervalles, et x₀, x₁, ..., xₙ sont les points équidistants de a à b. Le premier et le dernier termes ont le coefficient 1 ; tous les termes intérieurs ont le coefficient 2.

Zone trapézoïdale simple :Pour un intervalle de x₁ à x₂ de largeur h = x₂ - x₁, l'aire du trapèze est (h/2)[f(x₁) + f(x₂)]. Il s'agit de la hauteur moyenne multipliée par la largeur. Exemple : f(x) = x² de x = 2 à x = 3. h = 1, f(2) = 4, f(3) = 9. Aire ≈ (1/2)(4 + 9) = 6,5. Intégrale exacte : ∫[2,3] x²dx = [x³/3][2,3] = 27/3 - 8/3 = 19/3 ≈ 6,333. Erreur : environ 2,6 %.

Estimation de l'erreur :L'erreur dans la règle trapézoïdale est d'environ E ≈ -(b-a)³/(12n²) × f''(ξ) pour certains ξ dans [a,b]. Pour les fonctions qui se comportent bien, l'erreur diminue de 1/n² — doubler le nombre de sous-intervalles réduit l'erreur d'un facteur 4. Exemple : si n = 4 donne une erreur de 0,1, alors n = 8 donne une erreur d'environ 0,025. Cette convergence prévisible rend la règle trapézoïdale fiable.

Comparaison avec l'intégration exacte :Pour les polynômes de degré 1 (fonctions linéaires), la règle des trapèzes est exacte : les trapèzes correspondent parfaitement aux lignes droites. Pour les polynômes de degré supérieur et d'autres fonctions, il y a toujours une erreur, mais elle devient négligeable avec suffisamment de sous-intervalles. La règle est particulièrement précise pour les fonctions fluides et à variation lente.

Travailler sur des exemples complets : approximez ∫[0,2] (x² + 1)dx avec n = 4 sous-intervalles. h = (2-0)/4 = 0,5. Points : x₀ = 0, x₁ = 0,5, x₂ = 1, x₃ = 1,5, x₄ = 2. Valeurs de fonction : f(0) = 1, f(0,5) = 1,25, f(1) = 2, f(1,5) = 3,25, f(2) = 5. Appliquer la formule : Aire ≈ (0,5/2)[1 + 2(1,25) + 2(2) + 2(3,25) + 5] = 0,25[1 + 2,5 + 4 + 6,5 + 5] = 0,25[19] = 4,75. Valeur exacte : [x³/3 + x][0,2] = (8/3 + 2) - 0 = 14/3 ≈ 4,667. Erreur : environ 1,8 %.

Guide étape par étape pour l'approximation des règles trapézoïdales

  1. Identifiez les limites et la fonction de l’intégration.Déterminez a (limite inférieure), b (limite supérieure) et f(x) (la fonction à intégrer). Exemple : Intégrez f(x) = √x de a = 1 à b = 4. Notez clairement la fonction – vous devrez l'évaluer à plusieurs points.
  2. Choisissez le nombre de sous-intervalles n.Un plus grand nombre de sous-intervalles donne une meilleure précision mais nécessite plus de calculs. Commencez par n = 4 ou n = 10 pour une précision raisonnable. Pour une haute précision, utilisez n = 100 ou plus. Exemple : Choisissez n = 6 sous-intervalles pour intégrer √x de 1 à 4.
  3. Calculez la largeur du sous-intervalle h.h = (b - a) / n. Exemple : h = (4 - 1) / 6 = 3 / 6 = 0,5. Chaque sous-intervalle a une largeur de 0,5 unité. Cette largeur détermine la précision avec laquelle vous échantillonnez la fonction.
  4. Générez les valeurs X.En partant de x₀ = a, ajoutez h à plusieurs reprises : x₁ = a + h, x₂ = a + 2h, ..., xₙ = a + nh = b. Exemple : x₀ = 1, x₁ = 1,5, x₂ = 2, x₃ = 2,5, x₄ = 3, x₅ = 3,5, x₆ = 4. Vous devriez avoir n + 1 points (y compris les deux points finaux).
  5. Évaluez la fonction à chaque valeur x.Calculez f(x₀), f(x₁), ..., f(xₙ). Exemple : f(1) = 1, f(1,5) ≈ 1,225, f(2) ≈ 1,414, f(2,5) ≈ 1,581, f(3) ≈ 1,732, f(3,5) ≈ 1,871, f(4) = 2. Conservez plusieurs décimales pour plus de précision.
  6. Appliquez la formule trapézoïdale.Aire ≈ (h/2)[f(x₀) + 2f(x₁) + 2f(x₂) + ... + 2f(xₙ₋₁) + f(xₙ)]. Exemple : Aire ≈ (0,5/2)[1 + 2(1,225) + 2(1,414) + 2(1,581) + 2(1,732) + 2(1,871) + 2] = 0,25[1 + 2,45 + 2,828 + 3,162 + 3,464 + 3,742 + 2] = 0,25[18,646] ≈ 4,662. Valeur exacte : ∫[1,4] √x dx = [2x^(3/2)/3][1,4] = (2×8/3) - (2×1/3) = 14/3 ≈ 4,667. Erreur : environ 0,1 %.

Exemples de règles trapézoïdales réelles

Exemple 1 : Distance à partir des données de vitesse.La vitesse d'une voiture est mesurée toutes les 5 secondes : v(0) = 0, v(5) = 12, v(10) = 18, v(15) = 22, v(20) = 25 m/s. Trouver la distance totale parcourue. Distance = ∫[0,20] v(t)dt ≈ (5/2)[0 + 2(12) + 2(18) + 2(22) + 25] = 2,5[0 + 24 + 36 + 44 + 25] = 2,5[129] = 322,5 mètres. La règle trapézoïdale intègre les données de vitesse pour trouver le déplacement, même sans formule pour v(t).

Exemple 2 : Travail effectué par force variable.Un ressort nécessite une force F(x) = 10x + 2x² newtons pour comprimer x mètres. Trouvez le travail effectué en compressant de x = 0 à x = 0,5 m. Utilisez la règle trapézoïdale avec n = 5. h = 0,1. F(0) = 0, F(0,1) = 1,02, F(0,2) = 2,08, F(0,3) = 3,18, F(0,4) = 4,32, F(0,5) = 5,5. Travail ≈ (0,1/2)[0 + 2(1,02) + 2(2,08) + 2(3,18) + 2(4,32) + 5,5] = 0,05[0 + 2,04 + 4,16 + 6,36 + 8,64 + 5,5] = 0,05[26,7] = 1,335 joules. Exact : ∫[0,0,5] (10x + 2x²)dx = [5x² + 2x³/3][0,0,5] = 1,25 + 0,083 = 1,333 J. Erreur : 0,15 %.

Exemple 3 : Surplus du consommateur en économie.La fonction de demande est P(q) = 100 - 2q - 0,1q². Au prix du marché P = 60, la quantité est q = 20 (résolution 60 = 100 - 2q - 0,1q²). Surplus du consommateur = ∫[0,20] (100 - 2q - 0,1q²)dq - (60 × 20). En utilisant la règle des trapèzes avec n = 10 : h = 2. Évaluez P à q = 0, 2, 4, ..., 20. Additionnez les trapèzes, puis soustrayez les dépenses totales (1 200 €). Cela donne le bénéfice net que les consommateurs reçoivent en achetant au prix du marché.

Exemple 4 : Probabilité à partir de la distribution normale.Trouvez P(0 < Z < 1,5) pour la distribution normale standard. La PDF est f(z) = (1/√2π)e^(-z²/2). Cela n’a pas de primitive élémentaire, donc une intégration numérique est requise. En utilisant la règle trapézoïdale avec n = 30 de z = 0 à z = 1,5 : h = 0,05. Évaluez le PDF à 31 points, appliquez la formule. Résultat ≈ 0,4332, correspondant aux tables normales standard. C'est ainsi qu'un logiciel statistique calcule les probabilités.

Exemple 5 : Superficie d’un terrain irrégulier.Les géomètres mesurent la largeur d'une parcelle à intervalles de 10 mètres : 0, 15, 22, 28, 25, 18, 0 mètres (en commençant et en finissant à la limite). Superficie totale ≈ (10/2)[0 + 2(15) + 2(22) + 2(28) + 2(25) + 2(18) + 0] = 5[0 + 30 + 44 + 56 + 50 + 36 + 0] = 5[216] = 1 080 m². C’est exactement ainsi que les géomètres calculent les superficies à partir de mesures sur le terrain – la règle trapézoïdale appliquée aux données réelles.

Erreurs courantes dans les calculs de règles trapézoïdales

Oublier le coefficient 2 pour les points intérieurs.Faux : (h/2)[f(x₀) + f(x₁) + f(x₂) + ... + f(xₙ)]. Correct : (h/2)[f(x₀) + 2f(x₁) + 2f(x₂) + ... + 2f(xₙ₋₁) + f(xₙ)]. Seuls le premier et le dernier termes ont le coefficient 1 ; tous les termes intérieurs sont multipliés par 2. L'absence de ce facteur sous-estime considérablement la zone.

Utilisation d'une mauvaise largeur de sous-intervalle.h doit être égal à (b - a) / n, et non à (b - a) / (n - 1). Pour intégrer de 0 à 4 avec n = 4 sous-intervalles : h = 4/4 = 1, donnant les points 0, 1, 2, 3, 4 (5 points, 4 intervalles). Utiliser h = 4/3 ≈ 1,33 donnerait un mauvais espacement et une zone incorrecte.

Comptage incorrect des intervalles par rapport aux points.n sous-intervalles nécessitent n + 1 points (y compris les deux points finaux). Pour n = 4, vous avez besoin de 5 évaluations de fonctions : f(x₀), f(x₁), f(x₂), f(x₃), f(x₄). Confondre les intervalles avec les points conduit à manquer le point final ou à en ajouter un supplémentaire.

Ne pas utiliser suffisamment de sous-intervalles pour les fonctions courbes.Pour les fonctions très courbées (dérivée seconde changeant rapidement), un petit n produit de grandes erreurs. Exemple : intégrer sin(x) de 0 à π avec n = 2 donne une aire ≈ (π/4)[0 + 2(1) + 0] = π/2 ≈ 1,57, mais la valeur exacte est 2. L'erreur est de 21 % ! Utilisez n = 10 ou plus pour de telles fonctions : n = 10 donne une aire ≈ 1,98, erreur seulement 1 %.

Conseils de pro pour l'intégration des règles trapézoïdales

Utilisez un nombre pair de sous-intervalles pour la mise à niveau des règles de Simpson.Si vous avez besoin d'une meilleure précision, la règle de Simpson (qui utilise des paraboles au lieu de lignes) nécessite même n et donne de bien meilleurs résultats pour les fonctions fluides. Calculez d'abord le trapèze, puis appliquez celui de Simpson si nécessaire. L'erreur de la règle de Simpson diminue à mesure que 1/n⁴ par rapport à 1/n² du trapèze.

Exploitez la symétrie lorsque cela est possible.Si f(x) est symétrique par rapport au milieu de [a,b], vous pouvez intégrer la moitié de l'intervalle et doubler le résultat. Pour les fonctions paires intégrées de -a à a : ∫[-a,a] f(x)dx = 2∫[0,a] f(x)dx. Cela réduit le calcul de moitié tout en conservant la précision.

Vérifiez la convergence en doublant n.Calculez l'intégrale avec n sous-intervalles, puis avec 2n sous-intervalles. Si les résultats correspondent à la précision souhaitée, vous disposez de suffisamment de sous-intervalles. Sinon, doublez à nouveau. Exemple : n = 10 donne 4,65, n = 20 donne 4,666, n = 40 donne 4,6667 — convergence obtenue à 4,667.

À utiliser pour les données tabulées sans formules.La règle trapézoïdale brille lorsque vous disposez de points de données mais pas de formule de fonction : mesures expérimentales, données financières, relevés de capteurs. Branchez simplement les valeurs y dans la formule. Cela le rend inestimable pour les applications du monde réel où les fonctions ne sont pas connues analytiquement.

Foire aux questions

Pour les fonctions fluides, n = 10 à 20 donne souvent une précision de 1 à 2 %. Pour une haute précision (0,1 % ou mieux), utilisez n = 100 ou plus. Pour des estimations approximatives, n = 4 à 6 suffit. La meilleure approche : calculez avec n, puis 2n, puis 4n jusqu'à ce que les résultats successifs correspondent à la précision souhaitée. Les ordinateurs modernes peuvent gérer n = 1000+ instantanément.

La règle trapézoïdale est exacte pour les fonctions linéaires (polynômes de degré 1) car les trapèzes correspondent parfaitement aux lignes droites. Pour les constantes (degré 0), c'est aussi exact. Pour les quadratiques et supérieurs, il y a toujours une erreur, même si elle devient négligeable avec suffisamment de sous-intervalles. La règle de Simpson est exacte pour les cubes (degré 3).

Oui, si f(x) est négatif sur l'intervalle d'intégration. La règle trapézoïdale calcule l'aire signée — positive où f(x) > 0, négative où f(x) < 0. Pour ∫[0,2π] sin(x)dx, la règle trapézoïdale donne correctement environ 0 (l'aire positive de 0 à π annule l'aire négative de π à 2π).

La règle trapézoïdale fait la moyenne des sommes de Riemann gauche et droite, donnant une meilleure précision que l'une ou l'autre seule. Riemann gauche : utilise l'extrémité gauche de chaque intervalle. Right Riemann : utilise le point final droit. Trapézoïdal : utilise la moyenne des deux points finaux. Pour les fonctions monotones, la véritable intégrale se situe entre les sommes de Riemann gauche et droite, et la forme trapézoïdale est généralement plus proche de la vraie valeur.
