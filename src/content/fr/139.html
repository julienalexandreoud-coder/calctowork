Qu’est-ce qu’un quartile ?

Les quartiles divisent un ensemble de données triées en quatre parties égales, chacune contenant 25 % des données. Q1 (premier quartile) correspond au 25e centile, Q2 (deuxième quartile) est le 50e centile (médiane) et Q3 (troisième quartile) est le 75e centile. Les quartiles révèlent la répartition et l'asymétrie des données sans être déformés par des valeurs aberrantes.

L'intervalle interquartile (IQR = Q3 - Q1) contient la moitié médiane des données et constitue une mesure robuste de la propagation. Les boîtes à moustaches visualisent les quartiles, montrant la médiane sous la forme d'une ligne à l'intérieur d'une boîte de Q1 à Q3, avec des « moustaches » s'étendant pour montrer la plage de valeurs typiques.

Aperçu clé :Les quartiles divisent les données en quarts. Si vous obtenez un score dans le 3e quartile à un test, vous avez obtenu de meilleurs résultats que 50 à 75 % des candidats. Les quartiles sont essentiels pour identifier les valeurs aberrantes, comparer les distributions et comprendre la forme des données sans supposer qu'elles sont normales.

Formules expliquées

Positions des quartiles (pour n valeurs triées) :

Q1 (25e centile) : Position = (n + 1) × 0,25
Q2 (50e centile, médiane) : Position = (n + 1) × 0,50
Q3 (75e centile) : Position = (n + 1) × 0,75

Lorsque la position est un nombre entier :
Quartile = valeur à cette position

Lorsque la position est fractionnaire :
Interpoler entre des valeurs adjacentes :
Q = valeur_ci-dessous + fraction × (valeur_ci-dessus - valeur_ci-dessous)

Échelle interquartile (IQR) :
IQR = Q3 - Q1

Limites aberrantes (méthode de Tukey) :
Clôture inférieure = Q1 - 1,5 × IQR
Clôture supérieure = Q3 + 1,5 × IQR
Les valeurs en dehors des limites sont des valeurs aberrantes.

Exemple de calcul :
Données : 3, 7, 8, 12, 14, 18, 21, 25, 30 (n = 9)
Position Q1 : 10 × 0,25 = 2,5 → interpoler entre les positions 2 et 3
Q1 = 7 + 0,5 × (8 - 7) = 7,5
Position Q2 : 10 × 0,50 = 5 → valeur en position 5 = 14
Position Q3 : 10 × 0,75 = 7,5 → interpoler entre les positions 7 et 8
Q3 = 21 + 0,5 × (25 - 21) = 23
IQR = 23 - 7,5 = 15,5

Guide étape par étape

  1. Entrez vos données :Saisissez des nombres séparés par des virgules. Exemple : 12, 15, 18, 22, 25, 28, 32, 35, 40, 45
  2. Cliquez sur calculer :La calculatrice trie les données et calcule les quartiles
  3. Lisez vos résultats :Q1 = 17,25, Q2 (médiane) = 26,5, Q3 = 34,25
  4. Calculer l'IQR :IQR = Q3 - Q1 = 34,25 - 17,25 = 17
  5. Vérifiez les valeurs aberrantes :Clôture inférieure = 17,25 - 25,5 = -8,25 ; Clôture supérieure = 34,25 + 25,5 = 59,75

Exemples réels avec calculs

Exemple 1 : Analyse des salaires des employés
Salaires (en milliers) : 42, 45, 48, 52, 55, 58, 62, 65, 70, 75, 85, 120
Trié : identique (déjà croissant), n = 12
Position Q1 : 13 × 0,25 = 3,25 → 48 + 0,25 × (52-48) = 49
Position Q2 : 13 × 0,50 = 6,5 → (58+62)/2 = 60
Position Q3 : 13 × 0,75 = 9,75 → 70 + 0,75 × (75-70) = 73,75
IQR = 73,75 - 49 = 24,75
Clôtures aberrantes : 49 - 37,1 = 11,9 à 73,75 + 37,1 = 110,85
120 000 $ dépassent la clôture supérieure – c'est une valeur aberrante (probablement un cadre).
Application : les RH utilisent des quartiles pour l'analyse comparative de la rémunération. « Votre salaire est au troisième trimestre » signifie que vous gagnez plus de 75 % de celui de vos pairs.

Exemple 2 : Distribution des résultats des tests des étudiants
Résultats des examens : 52, 58, 61, 64, 67, 70, 72, 74, 76, 78, 80, 82, 85, 88, 92
n = 15
Position Q1 : 16 × 0,25 = 4 → valeur = 64
Position Q2 : 16 × 0,50 = 8 → valeur = 74
Position Q3 : 16 × 0,75 = 12 → valeur = 82
IQR = 82 - 64 = 18
Application : L'enseignant voit que les 50 % moyens ont un score compris entre 64 et 82. Les étudiants en dessous du Q1 (64) ont besoin d'une intervention ; les étudiants au-dessus du Q3 (82) peuvent avoir besoin d’un enrichissement. Plus exploitable que méchant seul.

Exemple 3 : Segmentation de la valeur des commandes de commerce électronique
Valeurs de commande ($) : 15, 22, 28, 35, 42, 48, 55, 62, 70, 85, 95, 120, 150, 200, 350
Q1 = 35, Q2 = 62, Q3 = 120, IQR = 85
Clôture aberrante : 120 + 127,5 = 247,5 → 350 $ est une valeur aberrante
Application : marketing segmente les clients : en dessous du T1 = acheteurs à petit budget, T1-T2 = chercheurs de valeur, T2-T3 = clients réguliers, au-dessus du T3 = VIP. Adaptez les promotions à chaque quartile.

Exemple 4 : Surveillance des performances du site Web
Temps de chargement des pages (ms) : 245, 268, 289, 312, 334, 356, 378, 401, 425, 458, 512, 680
Q1 = 300,5, Q2 = 367, Q3 = 447,25, IQR = 146,75
Clôture supérieure : 447,25 + 220 = 667 → 680 ms est une valeur aberrante
Application : L'équipe Performance se concentre sur P75 (T3 = 447 ms) comme objectif. La correction des valeurs aberrantes (680 ms) améliore l'expérience utilisateur dans le pire des cas. Google utilise P75 pour les seuils Core Web Vitals.

Exemple 5 : Plages de référence du laboratoire clinique
Niveaux de cholestérol (mg/dL) de 100 adultes en bonne santé :
Q1 = 165, Q2 = 185, Q3 = 210
« Plage normale » souvent définie comme Q1 à Q3 (moyen 50 %)
Application : Les résultats de laboratoire rapportent « votre valeur : 195, médiane de la population : 185, votre percentile : 62 % ». Les patients au-dessus du troisième trimestre peuvent avoir besoin de changements dans leur mode de vie ; des valeurs bien en dehors de l’IQR justifient une enquête médicale.

4 erreurs courantes

  • Utilisation incohérente de différentes méthodes de quartile :Les calculatrices Excel, R, Python et TI utilisent des méthodes d'interpolation légèrement différentes. Les résultats peuvent varier de 0,5 à 1 position. Documentez toujours la méthode que vous utilisez (cette calculatrice utilise la méthode (n+1) avec interpolation linéaire).
  • Confondre les quartiles avec les quarts :Q1 est la valeur en dessous de laquelle se situent 25 % des données : il s'agit d'une limite, pas d'un groupe. Le « premier quartile » fait parfois référence aux 25 % les plus bas des données (valeurs inférieures au T1). Précisez si vous parlez de la frontière ou du groupe.
  • En supposant que les quartiles sont équidistants :En données symétriques, Q2-Q1 ≈ Q3-Q2. Mais dans le cas de données asymétriques, les écarts diffèrent. Données sur les revenus : T1 = 35 000 $, Q2 = 55 000 $, Q3 = 95 000 $. Écart T2-T1 = 20 000 $ ; écart Q3-Q2 = 40 000 $ — révèle une asymétrie à droite.
  • Utilisation de quartiles pour de petits ensembles de données :Avec n < 10, les quartiles sont instables et très sensibles aux valeurs individuelles. Un ensemble de données de 5 nombres ne peut pas être divisé de manière significative en trimestres. Utilisez des quartiles pour n ≥ 20 ; pour les données plus petites, indiquez min, médiane, max.

4 conseils de pro

  • Utilisez des quartiles pour les diagrammes en boîte :Les boîtes à moustaches visualisent des résumés à cinq chiffres : min, Q1, Q2, Q3, max. La boîte montre l'IQR (moyen 50 %), la ligne montre la médiane, les moustaches montrent la plage typique. Les diagrammes en boîte côte à côte comparent efficacement les groupes.
  • Appliquer la détection des valeurs aberrantes par quartile :Les valeurs inférieures à Q1-1,5×IQR ou supérieures à Q3+1,5×IQR sont des « légères valeurs aberrantes » ; au-delà de Q1-3×IQR ou Q3+3×IQR sont des « valeurs aberrantes extrêmes ». Plus robuste que ±3 SD pour les données non normales.
  • Utilisez des déciles ou des percentiles pour une granularité plus fine :Les quartiles divisent les données en 4 parties ; déciles en 10 ; percentiles en 100. Pour les grands ensembles de données (n > 1 000), les percentiles révèlent plus de détails sur la forme de la distribution.
  • Comparez les quartiles entre les groupes :« Le revenu médian a augmenté du T2 = 52 000 $ au T2 = 58 000 $, mais le premier trimestre est resté stable à 35 000 $ » révèle une croissance concentrée dans la moitié supérieure. Les comparaisons par quartile révèlent des inégalités qui signifient se cacher.

FAQs

Les quartiles sont des percentiles spécifiques : Q1 = P25, Q2 = P50 (médiane), Q3 = P75. Les percentiles divisent les données en 100 parties ; quartiles en 4. Dire « vous avez obtenu un score dans le 85e centile » est plus précis que « vous avez obtenu un score supérieur au Q3 ».
Plusieurs méthodes valides existent : Excel QUARTILE.EXC vs QUARTILE.INC, les 9 types de R, le linéaire de Python vs le plus proche. Les différences proviennent de la manière dont les positions sont calculées et de l'utilisation ou non de l'interpolation. Toutes les méthodes convergent pour les grands ensembles de données.
Oui. Les quartiles fonctionnent avec toutes les données numériques ordonnées, y compris les données négatives. Ensemble de données : -20, -15, -10, -5, 0, 5, 10 a Q1=-12,5, Q2=-5, Q3=7,5. La méthode est identique ; les valeurs négatives ne nécessitent pas de traitement particulier.
Utilisez IQR pour : les données asymétriques, les données avec des valeurs aberrantes, les données ordinales ou lorsque vous ne présumez pas la normalité. Utilisez SD pour : les données symétriques, les tests paramétriques, la combinaison de variables indépendantes ou lorsque les propriétés mathématiques sont importantes. L'IQR est robuste ; SD est mathématiquement pratique.
