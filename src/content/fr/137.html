Qu’est-ce que l’écart ?

La variance mesure l'étendue d'un ensemble de données en calculant la moyenne des écarts carrés par rapport à la moyenne. Il quantifie la variabilité : une variance élevée signifie que les points de données sont largement dispersés ; une faible variance signifie qu'ils se regroupent étroitement autour de la moyenne. La variance est le fondement de l'écart type, de l'ANOVA, de l'analyse de régression et de la théorie du portefeuille.

La variance est exprimée en unités carrées (par exemple, dollars², mètres²), ce qui la rend moins intuitive que l'écart type mais mathématiquement essentielle. La racine carrée de la variance est égale à l'écart type, ramenant la mesure aux unités d'origine.

Aperçu clé :La variance a une propriété unique : les variances des variables indépendantes s'additionnent. Si X et Y sont indépendants, Var(X + Y) = Var(X) + Var(Y). Cela rend la variance indispensable pour combiner les incertitudes en physique, en finance et en ingénierie.

Formules expliquées

Variance de l'échantillon (la plus courante) :
s² = Σ(xᵢ - x̄)² / (n - 1)

Variance de la population :
σ² = Σ(xᵢ - μ)² / N

Formule de calcul (plus facile pour les calculs) :
s² = [Σxᵢ² - (Σxᵢ)²/n] / (n - 1)

Where:
xᵢ = chaque valeur individuelle
x̄ = moyenne de l'échantillon
μ = moyenne de la population
n = taille de l'échantillon
N = taille de la population
Σ = somme de

Pourquoi (n-1) pour les échantillons ?
L'utilisation de (n-1) au lieu de n fournit une estimation impartiale de la variance de la population. Cette « correction de Bessel » explique le fait que nous estimons la moyenne à partir des mêmes données, ce qui rend les écarts légèrement plus faibles qu'ils ne le seraient par rapport à la moyenne réelle de la population.

Propriétés clés :

  • Var(c) = 0 — la constante n'a pas de variance
  • Var(X + c) = Var(X) — l'ajout d'une constante ne change pas la propagation
  • Var(cX) = c² × Var(X) — la mise à l'échelle multiplie la variance par le carré
  • Var(X + Y) = Var(X) + Var(Y) — pour X indépendant, Y
  • Var(X) = E[X²] - (E[X])² — définition alternative utilisant les attentes

Exemples réels avec calculs

Exemple 1 : Risque du portefeuille d'investissement
Rendements annuels de l'action A (5 ans) : 10 %, 15 %, 8 %, 22 %, 5 %
Moyenne : (10+15+8+22+5) / 5 = 60/5 = 12 %
Écarts : -2, 3, -4, 10, -7
Écarts au carré : 4, 9, 16, 100, 49
Sum: 178
Variance de l'échantillon : 178 / 4 = 44,5 (%²)
Écart type : √44,5 = 6,67 %
Application : Variance plus élevée = risque plus élevé. Comparez avec l'action B avec une variance = 25 — L'action A est plus risquée.

Exemple 2 : Comparaison des processus de fabrication
La machine A produit des pièces (mm) : 10,1, 9,9, 10,0, 10,2, 9,8
La machine B produit des pièces (mm) : 10,3, 9,7, 10,5, 9,5, 10,0
Les deux ont une moyenne = 10,0 mm
Écart machine A : 0,025 mm²
Variation machine B : 0,17 mm²
Application : La machine A a une variance 7 fois inférieure – une qualité beaucoup plus constante. Choisissez la machine A pour un travail de précision.

Exemple 3 : analyse de test A/B
Conversions du groupe témoin (10 jours) : 45, 48, 52, 47, 50, 49, 51, 46, 53, 48
Moyenne : 489/10 = 48,9
Écart : 7,66
Groupe de traitement : 52, 55, 49, 58, 54, 51, 56, 53, 57, 50
Moyenne : 535/10 = 53,5
Écart : 9,17
Application : Le traitement a une moyenne plus élevée (+4,6) mais également une variance plus élevée. Utilisez le test t pour déterminer si la différence est statistiquement significative.

Exemple 4 : Combinaison de risques indépendants
Le projet présente 3 facteurs de risque indépendants :
Risque A écart : 100 (dépassement de coût)
Variance du risque B : 64 (pénalité de retard)
Écart du risque C : 36 (coût de reprise)
Variance totale : 100 + 64 + 36 = 200
Écart-type total : √200 = 14,14
Application : La variance du portefeuille s'ajoute pour les risques indépendants. Le risque total (SD) est inférieur à la somme des SD individuels en raison de la diversification.

Exemple 5 : Carte de contrôle qualité
Variance historique du processus : σ² = 2,25 (SD = 1,5)
Limites de contrôle : moyenne ± 3σ = moyenne ± 4,5
Nouvel échantillon : 8 observations, variance = 6,8
Application : La variance de l'échantillon (6,8) dépasse l'historique (2,25) de 3 ×. Le processus est peut-être devenu instable. Enquêter sur les causes attribuables (nouveau fournisseur, usure des équipements, changement d'opérateur).

4 erreurs courantes

  • Variance confuse entre échantillon et population :Utilisez (n-1) pour les échantillons, N pour les populations. La variance de l'échantillon est plus grande, compensant l'incertitude de l'estimation. La plupart des analyses du monde réel utilisent des échantillons, donc (n-1) est standard.
  • Interprétation directe de la variance :La variance est exprimée en unités carrées (dollars², kg²), difficiles à interpréter. Prenez la racine carrée pour obtenir l’écart type en unités d’origine. Utiliser la variance pour les opérations mathématiques ; utilisez SD pour la communication.
  • Ajout de variances de variables corrélées :Var(X + Y) = Var(X) + Var(Y) uniquement lorsque X et Y sont indépendants (non corrélés). Pour les variables corrélées : Var(X + Y) = Var(X) + Var(Y) + 2×Cov(X,Y). Ignorer la covariance sous-estime ou surestime la variance totale.
  • Utilisation de la variance pour les distributions asymétriques :La variance suppose un écart symétrique autour de la moyenne. Pour les données asymétriques (revenus, prix de l’immobilier), la variance est dominée par les valeurs extrêmes. Utilisez l’intervalle interquartile (IQR) ou l’écart médian absolu (MAD) pour des mesures de propagation robustes.

4 conseils de pro

  • Utilisez la formule de calcul pour les calculs manuels :s² = [Σx² - (Σx)²/n] / (n-1). Pour les données 2, 4, 5, 7 : Σx = 18, Σx² = 94, n = 4. Variance = [94 - 324/4] / 3 = [94 - 81] / 3 = 13/3 = 4,33. Plus rapide que le calcul des écarts individuels.
  • Appliquer la décomposition de la variance (principe de l'ANOVA) :Variance totale = variance inter-groupes + variance intra-groupe. Cela révèle si les différences proviennent de l'appartenance à un groupe ou de variations aléatoires au sein des groupes.
  • Utilisez la variance groupée pour comparer les groupes :Lorsque vous comparez deux échantillons avec des variances similaires, regroupez-les : s²_pooled = [(n₁-1)s₁² + (n₂-1)s₂²] / (n₁+n₂-2). Cela donne une estimation plus stable pour les tests t.
  • Transformer les données pour stabiliser la variance :Si la variance augmente avec la moyenne (ce qui est courant dans les données de comptage), appliquez la racine carrée ou la transformation logarithmique. Cette « stabilisation de la variance » rend les tests statistiques plus valides. Exemple : Pour les données de Poisson, √x a une variance à peu près constante.
