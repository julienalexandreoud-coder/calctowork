Qu'est-ce que le calculateur de produit scalaire ?

Dot Product Calculator calcule le produit scalaire de deux vecteurs - un nombre unique qui révèle le degré d'alignement des vecteurs, projette un vecteur sur un autre et calcule le travail effectué par les forces. Le produit scalaire apparaît dans la physique (travail et énergie), l’ingénierie (traitement du signal), l’infographie (éclairage et ombrage) et l’apprentissage automatique (mesures de similarité). Contrairement à l'addition vectorielle ou au produit croisé, le produit scalaire génère un scalaire (nombre régulier), pas un vecteur.

Considérons deux vecteurs : a = (3, 4) et b = (5, 0). Le produit scalaire est a · b = (3)(5) + (4)(0) = 15 + 0 = 15. Géométriquement, cela équivaut à |a| × |b| × cos(θ), où θ est l'angle qui les sépare. Le vecteur a a une magnitude de 5, le vecteur b a une magnitude de 5 et l'angle entre eux est arccos(15/25) = arccos(0,6) ≈ 53,13°. Le produit scalaire de 15 nous indique que les vecteurs sont modérément alignés – ni perpendiculaires (point = 0) ni parallèles (point = 25).

La signification physique du produit scalaire devient claire avec la force et le déplacement. Si vous poussez avec une force F = (10, 0, 0) newtons et qu'un objet se déplace de d = (3, 4, 0) mètres, le travail effectué est F · d = (10)(3) + (0)(4) + (0)(0) = 30 joules. Seule la composante de la force dans la direction du mouvement contribue au travail — le produit scalaire extrait automatiquement cette composante parallèle.

Fonctionnement du calculateur de produit scalaire : explication des formules

Formule algébrique :Pour les vecteurs a = (a₁, a₂, a₃) et b = (b₁, b₂, b₃), le produit scalaire est a · b = a₁b₁ + a₂b₂ + a₃b₃. Multipliez les composants correspondants, puis additionnez les produits. Exemple : a = (2, -3, 5), b = (4, 1, -2). une · b = (2)(4) + (-3)(1) + (5)(-2) = 8 - 3 - 10 = -5. Le résultat est un scalaire et non un vecteur.

Formule géométrique :une · b = |une| × |b| × cos(θ), où |a| et |b| sont les grandeurs et θ est l'angle entre les vecteurs. Cela révèle la signification géométrique du produit scalaire : c'est le produit de la magnitude d'un vecteur multiplié par la projection de l'autre vecteur sur lui. Exemple : |a| = 5, |b| = 8, θ = 60°. a · b = 5 × 8 × cos(60°) = 40 × 0,5 = 20.

Trouver l'angle :Réorganisez la formule géométrique : cos(θ) = (a · b) / (|a| × |b|). Alors θ = arccos[(a · b) / (|a| × |b|)]. Exemple : a = (3, 4), b = (5, 0). une · b = 15, |une| = 5, |b| = 5. cos(θ) = 15 / (5 × 5) = 0,6. θ = arccos(0,6) ≈ 53,13°. C’est ainsi que vous trouvez l’angle entre deux vecteurs quelconques.

Formule de projection :La projection du vecteur a sur le vecteur b est : proj_b(a) = (a · b / |b|²) × b. Cela donne la composante de a qui pointe dans la direction de b. Exemple : a = (5, 0), b = (3, 4). a · b = 15, |b|² = 25. proj_b(a) = (15/25) × (3, 4) = 0,6 × (3, 4) = (1,8, 2,4). La projection a une magnitude |a| × cos(θ) = 5 × 0,6 = 3.

Travailler à travers des exemples complets : Trouver (2, 3, 4) · (1, -1, 2). Produit scalaire = (2)(1) + (3)(-1) + (4)(2) = 2 - 3 + 8 = 7. Trouvez l'angle entre (1, 0, 0) et (0, 1, 0). Produit scalaire = 0, donc cos(θ) = 0, ce qui signifie θ = 90° — les vecteurs sont perpendiculaires (orthogonaux). Trouvez la projection de (6, 8) sur (3, 0). a · b = 18, |b|² = 9. proj = (18/9) × (3, 0) = 2 × (3, 0) = (6, 0). Le vecteur entier (6, 8) se projette vers (6, 0) sur l'axe des x.

Guide étape par étape pour calculer les produits scalaires

  1. Vérifiez que les vecteurs ont la même dimension.Les deux vecteurs doivent avoir le même nombre de composantes. Vous pouvez calculer le produit scalaire de deux vecteurs 3D, mais pas d'un vecteur 2D et d'un vecteur 3D. Exemple : (2, 3, 4) et (1, -1, 2) sont tous deux 3D — valides. (2, 3) et (1, -1, 2) sont incompatibles — ne peuvent pas calculer le produit scalaire.
  2. Multipliez les composants correspondants.Associez les composants par position : premier avec premier, deuxième avec deuxième, troisième avec troisième. Exemple : a = (2, -3, 5), b = (4, 1, -2). Produits : 2×4 = 8, (-3)×1 = -3, 5×(-2) = -10. Suivez attentivement les signes : les temps négatifs positifs sont négatifs, les temps négatifs négatifs sont positifs.
  3. Additionnez les produits.Additionnez tous les produits composants : 8 + (-3) + (-10) = 8 - 3 - 10 = -5. Le produit scalaire de (2, -3, 5) et (4, 1, -2) est -5. Un produit scalaire négatif indique que les vecteurs pointent dans des directions généralement opposées (angle > 90°).
  4. Interprétez le résultat.Produit scalaire > 0 : les vecteurs sont aigus (angle < 90°), pointant dans des directions similaires. Produit scalaire = 0 : les vecteurs sont perpendiculaires (orthogonaux). Produit scalaire < 0 : les vecteurs sont obtus (angle > 90°), pointant dans des directions générales opposées. L'ampleur du produit scalaire indique la force de l'alignement.
  5. Calculez l'angle si nécessaire.Utilisez θ = arccos[(a · b) / (|a| × |b|)]. Trouvez d’abord les magnitudes : |a| = √(a₁² + a₂² + a₃²), |b| = √(b₁² + b₂² + b₃²). Calculez ensuite cos(θ) = (a · b) / (|a| × |b|). Enfin, prenons arccos pour trouver θ en degrés ou en radians.
  6. Appliquez-vous à votre problème.Utilisez le produit scalaire pour les calculs de travail (W = F · d), la recherche d'angles, la projection de vecteurs, le test d'orthogonalité ou la mesure de similarité. Dans l'apprentissage automatique, le produit scalaire mesure la similitude de deux vecteurs de caractéristiques : un produit scalaire plus élevé signifie plus similaire.

Exemples de produits scalaires concrets

Exemple 1 : Travail effectué par une force.Une force F = (50, 30, 0) newtons pousse un objet par un déplacement d = (10, 5, 0) mètres. Travail effectué : W = F · d = (50)(10) + (30)(5) + (0)(0) = 500 + 150 + 0 = 650 joules. La force a une magnitude √(2500 + 900) = √3400 ≈ 58,3 N. Le déplacement a une magnitude √(100 + 25) = √125 ≈ 11,2 m. Angle : cos(θ) = 650 / (58,3 × 11,2) ≈ 0,996, donc θ ≈ 5° — presque aligné, expliquant le rendement de travail élevé.

Exemple 2 : Éclairage d’infographie.Une surface a un vecteur normal n = (0, 1, 0) pointant vers le haut. La direction d'une source lumineuse est L = (0,6, 0,8, 0) (déjà normalisée). Intensité de l'éclairage diffus = n · L = (0)(0,6) + (1)(0,8) + (0)(0) = 0,8. La surface reçoit 80% de l'intensité lumineuse maximale. Si la surface était verticale avec la normale (1, 0, 0), l'intensité serait (1)(0,6) + (0)(0,8) = 0,6 = 60 %. Ce calcul de produit scalaire s’exécute des millions de fois par seconde dans les jeux 3D.

Exemple 3 : Test de l'orthogonalité du vecteur.Les vecteurs (3, 4, 0) et (4, -3, 5) sont-ils perpendiculaires ? Calculer le produit scalaire : (3)(4) + (4)(-3) + (0)(5) = 12 - 12 + 0 = 0. Le produit scalaire nul confirme qu'ils sont orthogonaux (perpendiculaires). Ce test est essentiel en infographie (vérifier si la surface fait face à la caméra), en physique (décomposer les forces en composantes perpendiculaires) et en algèbre linéaire (trouver des bases orthogonales).

Exemple 4 : Corrélation avec le traitement du signal.Deux signaux discrets sont représentés sous forme de vecteurs : s₁ = (1, 2, 3, 4, 5) et s₂ = (2, 4, 6, 8, 10). Produit scalaire : 1×2 + 2×4 + 3×6 + 4×8 + 5×10 = 2 + 8 + 18 + 32 + 50 = 110. Une valeur positive élevée indique une forte corrélation – s₂ est exactement 2×s₁. Si s₂ était (5, 4, 3, 2, 1), le produit scalaire serait 1 × 5 + 2 × 4 + 3 × 3 + 4 × 2 + 5 × 1 = 35 — corrélation inférieure. Si s₂ était (-1, -2, -3, -4, -5), le produit scalaire serait de -55 — corrélation négative (relation inverse).

Exemple 5 : Projection d'une force sur un plan incliné.La gravité agit vers le bas : g = (0, 0, -9,8) m/s². Un plan incliné a une direction de surface s = (3, 0, 4) (montant de 4 unités pour 3 unités vers l'avant). Vecteur de surface unitaire : ŝ = (3/5, 0, 4/5) = (0,6, 0, 0,8). Composante de la gravité le long du plan : g · ŝ = (0)(0,6) + (0)(0) + (-9,8)(0,8) = -7,84 m/s². Le signe négatif indique que la gravité tire l’avion vers le bas. Un objet sur ce plan accélère à 7,84 m/s² sur la pente (en ignorant le frottement).

Erreurs courantes dans les calculs de produits scalaires

Confondre produit scalaire et produit croisé.Le produit scalaire donne un scalaire (nombre) ; le produit vectoriel donne un vecteur. Produit scalaire : a · b = a₁b₁ + a₂b₂ + a₃b₃. Produit vectoriel : a × b = (a₂b₃ - a₃b₂, a₃b₁ - a₁b₃, a₁b₂ - a₂b₁). Exemple : (1, 0, 0) · (0, 1, 0) = 0 (scalaire). (1, 0, 0) × (0, 1, 0) = (0, 0, 1) (vecteur). Ils servent à des fins différentes : le point mesure l'alignement, la croix produit un vecteur perpendiculaire.

Ajouter au lieu de multiplier des composants.Faux : (2, 3) · (4, 5) = (2+4, 3+5) = (6, 8). Correct : (2, 3) · (4, 5) = 2×4 + 3×5 = 8 + 15 = 23. Le produit scalaire multiplie les composants correspondants, puis les additionne – il ne produit pas de vecteur. Le résultat est toujours un scalaire.

Oublier ce produit scalaire peut être négatif.Un produit scalaire négatif n’indique pas une erreur : cela signifie que les vecteurs pointent dans des directions généralement opposées (angle compris entre 90° et 180°). Exemple : (1, 0) · (-1, 0) = -1. Les vecteurs sont anti-parallèles (espacés de 180°). Le cosinus de 180° est de -1, donc le produit scalaire négatif est correct.

Positions des composants incompatibles.Multipliez la première composante par la première, la seconde par la seconde, etc. Faux : (2, 3, 4) · (5, 6, 7) = 2×6 + 3×7 + 4×5 = 12 + 21 + 20 = 53. Correct : (2, 3, 4) · (5, 6, 7) = 2×5 + 3×6 + 4×7 = 10 + 18 + 28 = 56. La position compte – les composants doivent s’aligner par index.

Conseils de pro pour les applications de produits scalaires

Utilisez le produit scalaire pour des tests d’orthogonalité rapides.Au lieu de calculer des angles, vérifiez simplement si a · b = 0. Ceci est moins coûteux en calcul et évite les erreurs en virgule flottante des arccos. Dans la modélisation 3D, testez si une normale à une surface est perpendiculaire à la direction de la vue (a · b ≈ 0 signifie silhouette de bord).

Appliquer à la mesure de similarité.Dans l'apprentissage automatique et la récupération d'informations, le produit scalaire (ou similarité cosinus, qui normalise par les amplitudes) mesure la similitude de deux vecteurs. Les vecteurs de documents avec un produit scalaire élevé partagent des distributions de mots similaires. Les systèmes de recommandation utilisent des produits scalaires pour faire correspondre les préférences des utilisateurs aux fonctionnalités des articles.

Décomposez efficacement les vecteurs.Tout vecteur v peut être divisé en composantes parallèles et perpendiculaires à une direction n : v_parallel = (v · n̂) n̂ et v_perpendiculaire = v - v_parallel, où n̂ est un vecteur unitaire. Cette décomposition est essentielle pour analyser les forces sur des plans inclinés, séparer le signal du bruit et calculer les réflexions.

Reconnaissez le lien avec la multiplication matricielle.Le produit scalaire de deux vecteurs est égal au produit matriciel d'un vecteur ligne multiplié par un vecteur colonne : [a₁, a₂, a₃] × [b₁ ; b₂; b₃] = a₁b₁ + a₂b₂ + a₃b₃. Cette connexion est à la base des calculs de réseaux neuronaux, où chaque neurone calcule un produit scalaire d'entrées avec des poids.

Foire aux questions

Le produit scalaire nul signifie que les vecteurs sont perpendiculaires (orthogonaux) – l’angle entre eux est exactement de 90°. Géométriquement, un vecteur n’a aucune composante dans la direction de l’autre. Physiquement, une force perpendiculaire au déplacement ne fait aucun travail. En graphique, une surface dont la normale est perpendiculaire à la direction de la lumière ne reçoit aucun éclairage direct.

Le produit scalaire maximum se produit lorsque les vecteurs sont parallèles (θ = 0°, cos = 1), ce qui donne a · b = |a| × |b|. Si votre calcul montre a · b > |a| × |b|, il y a une erreur arithmétique.

Pour les vecteurs complexes, utilisez le produit scalaire hermitien : a · b = Σ aᵢ* × bᵢ, où aᵢ* est le conjugué complexe de aᵢ. Cela garantit que le produit scalaire d’un vecteur avec lui-même est réel et non négatif. Pour les vecteurs réels, la conjugaison n’a aucun effet, la formule standard s’applique donc. Les produits scalaires complexes sont essentiels en mécanique quantique.

La loi des cosinus c² = a² + b² - 2ab×cos(C) peut être dérivée du produit scalaire. Pour les vecteurs a et b formant deux côtés d'un triangle, le troisième côté est (a - b). Alors |a - b|² = (a - b) · (a - b) = |a|² + |b|² - 2(a · b) = |a|² + |b|² - 2|a||b|×cos(θ). C'est la loi des cosinus sous forme vectorielle.
