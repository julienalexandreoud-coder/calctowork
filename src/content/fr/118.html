Qu'est-ce que la calculatrice de dérivée polynomiale quadratique ?

La calculatrice de dérivée polynomiale quadratique trouve la dérivée (taux de changement) d'une fonction quadratique - la pente de la ligne tangente en tout point. Les fonctions quadratiques décrivent les trajectoires, les courbes de profit, les problèmes d'optimisation et l'accélération sous force constante. La dérivée révèle où la fonction augmente, diminue ou atteint son maximum ou son minimum – essentiel pour l'optimisation, l'analyse physique et la compréhension du comportement de la fonction.

Considérons la fonction quadratique f(x) = 2x² - 8x + 5. Sa dérivée est f'(x) = 4x - 8. À x = 0, la pente est f'(0) = -8 — la fonction diminue fortement. À x = 2, la pente est f'(2) = 0 — c'est le sommet (point minimum). À x = 5, la pente est f'(5) = 12 — la fonction augmente fortement. La dérivée transforme la parabole en une fonction linéaire qui code le taux de changement instantané en chaque point.

La dérivée d'un quadratique ax² + bx + c est toujours linéaire : 2ax + b. Ce résultat élégant, découvert par Newton et Leibniz dans les années 1600, permet le calcul instantané des pentes, l'identification des extrema et la solution des problèmes d'optimisation sans graphique. De la maximisation du profit à la recherche du sommet de la trajectoire d'un projectile, les dérivées quadratiques sont des outils indispensables.

Fonctionnement de la calculatrice de dérivée polynomiale quadratique : explication des formules

Règle de puissance pour les quadratiques :Pour f(x) = ax² + bx + c, la dérivée est f'(x) = 2ax + b. Le terme x² devient 2x (multipliez par l'exposant, réduisez l'exposant par 1). Le terme x devient 1 (multiplié par l'exposant 1, x⁰ = 1). La constante c disparaît (la dérivée d'une constante est nulle). Exemple : f(x) = 3x² + 5x - 7 a f'(x) = 6x + 5.

Interprétation géométrique :La dérivée f'(x) donne la pente de la tangente à la parabole au point x. Au sommet (maximum ou minimum), la tangente est horizontale, donc f'(x) = 0. La résolution de 2ax + b = 0 donne x = -b/(2a) — la coordonnée x du sommet. Exemple : f(x) = 2x² - 8x + 5 a un sommet en x = -(-8)/(2×2) = 8/4 = 2.

Trouver les extrema :Définissez f'(x) = 0 et résolvez x pour trouver les points critiques. Pour les quadratiques, il y a exactement un point critique (le sommet). Déterminez s'il s'agit d'un maximum ou d'un minimum en vérifiant la dérivée seconde f''(x) = 2a. Si a > 0, la parabole s'ouvre vers le haut (minimum). Si a < 0, il s'ouvre vers le bas (maximum). Exemple : f(x) = -x² + 6x - 5 a f'(x) = -2x + 6. Le réglage -2x + 6 = 0 donne x = 3. Puisque a = -1 < 0, c'est un maximum.

Taux de changement des demandes :Si f(t) représente la position au temps t, alors f'(t) représente la vitesse. Pour un objet qui tombe avec une position f(t) = -4,9t² + 20t + 100 (mètres), la vitesse est f'(t) = -9,8t + 20 (m/s). À t = 0, la vitesse est de 20 m/s vers le haut. À t = 2, la vitesse est de -9,8(2) + 20 = 0,4 m/s (presque au pic). À t = 3, la vitesse est de -9,4 m/s (en baisse).

Travailler à travers des exemples complets : Trouvez la dérivée de f(x) = 5x² - 3x + 9. En utilisant la formule : f'(x) = 2(5)x + (-3) = 10x - 3. Trouvez le sommet de f(x) = x² - 10x + 25. Dérivée : f'(x) = 2x - 10. Mettez à zéro : 2x - 10 = 0, donc x = 5. Le sommet est à x = 5. Trouvez le maximum de f(x) = -2x² + 12x - 10. Dérivée : f'(x) = -4x + 12. Mis à zéro : -4x + 12 = 0, donc x = 3. Puisque a = -2 < 0, c'est un maximum. Valeur maximale : f(3) = -2(9) + 12(3) - 10 = -18 + 36 - 10 = 8.

Guide étape par étape pour trouver des dérivées quadratiques

  1. Identifiez les coefficients a, b et c.Écrivez le quadratique sous la forme standard f(x) = ax² + bx + c. Exemple : f(x) = 7 - 2x + 4x² doit être réécrit comme f(x) = 4x² - 2x + 7, donnant a = 4, b = -2, c = 7. L'ordre compte — a est le coefficient de x², b est le coefficient de x, c est la constante.
  2. Appliquez la formule dérivée.f'(x) = 2ax + b. Multipliez a par 2, gardez x, puis ajoutez b. Exemple : a = 4, b = -2. f'(x) = 2(4)x + (-2) = 8x - 2. La constante c n'apparaît pas dans la dérivée — elle disparaît lors de la différenciation.
  3. Simplifiez si nécessaire.Combinez des termes similaires et simplifiez les coefficients. Exemple : f(x) = 0,5x² + 3x - 1 a f'(x) = 2(0,5)x + 3 = x + 3. Les coefficients fractionnaires conviennent : conservez les valeurs exactes plutôt que d'arrondir aux décimales lorsque cela est possible.
  4. Trouver les points critiques (facultatif).Fixez f'(x) = 0 et résolvez : 2ax + b = 0, donc x = -b/(2a). Exemple : f'(x) = 8x - 2 = 0 donne 8x = 2, donc x = 0,25. Il s'agit de la coordonnée x du sommet – où la parabole passe de décroissante à croissante (ou vice versa).
  5. Déterminez le maximum ou le minimum.Vérifiez le signe d'un. Si a > 0, le point critique est un minimum (la parabole s'ouvre vers le haut). Si a < 0, c'est un maximum (la parabole s'ouvre vers le bas). Exemple : f(x) = 4x² - 2x + 7 a a = 4 > 0, donc x = 0,25 est un point minimum.
  6. Trouvez la valeur de la fonction aux points critiques.Rebranchez la valeur x critique dans f(x) pour trouver la coordonnée y du sommet. Exemple : f(0,25) = 4(0,25)² - 2(0,25) + 7 = 4(0,0625) - 0,5 + 7 = 0,25 - 0,5 + 7 = 6,75. Le sommet est à (0,25, 6,75), et c'est la valeur minimale de la fonction.

Exemples de dérivées quadratiques du monde réel

Exemple 1 : Pic de mouvement du projectile.Une balle est lancée vers le haut avec une hauteur h(t) = -4,9t² + 15t + 2 (mètres). Quand atteint-il sa hauteur maximale ? Dérivée : h'(t) = -9,8t + 15. Mis à zéro : -9,8t + 15 = 0, donc t = 15/9,8 ≈ 1,53 secondes. Hauteur maximale : h(1,53) = -4,9(1,53)² + 15(1,53) + 2 ≈ -11,48 + 22,95 + 2 ≈ 13,47 mètres. Le ballon culmine à environ 13,5 mètres après 1,5 seconde, puis retombe.

Exemple 2 : Maximisation des bénéfices.La fonction de profit d'une entreprise est P(x) = -0,5x² + 100x - 500, où x représente les unités produites. Quel niveau de production maximise le profit ? Dérivée : P'(x) = -x + 100. Mis à zéro : -x + 100 = 0, donc x = 100 unités. Bénéfice maximum : P(100) = -0,5(10000) + 100(100) - 500 = -5000 + 10000 - 500 = 4 500 €. Produire plus de 100 unités réduit les bénéfices en raison de rendements décroissants.

Exemple 3 : Optimisation de la zone rectangulaire.Un agriculteur dispose de 200 mètres de clôture pour un enclos rectangulaire. Quelles dimensions maximisent la surface ? Si la largeur est x, la longueur est (200 - 2x)/2 = 100 - x. Aire : A(x) = x(100 - x) = 100x - x². Dérivée : A'(x) = 100 - 2x. Mis à zéro : 100 - 2x = 0, donc x = 50 mètres. Longueur = 100 - 50 = 50 mètres. Superficie maximale : 50 × 50 = 2 500 m². Une enceinte carrée maximise la surface pour un périmètre donné.

Exemple 4 : Analyse de la distance d'arrêt.La distance d'arrêt d'une voiture est d(v) = 0,05v² + 0,8v (mètres), où v est la vitesse en km/h. À quelle vitesse la distance d’arrêt augmente-t-elle à 60 km/h ? Dérivée : d'(v) = 0,1v + 0,8. À v = 60 : d'(60) = 0,1(60) + 0,8 = 6,8 mètres par km/h. Chaque km/h supplémentaire ajoute environ 6,8 mètres à la distance d'arrêt à cette vitesse, ce qui explique pourquoi les accidents à grande vitesse sont bien plus graves.

Exemple 5 : Optimisation des revenus.Un théâtre vend des billets au prix p, avec une fréquentation n(p) = 500 - 10p. Les revenus sont R(p) = p × n(p) = p(500 - 10p) = 500p - 10p². Quel prix maximise les revenus ? Dérivée : R'(p) = 500 - 20p. Mis à zéro : 500 - 20p = 0, donc p = 25 €. A ce prix, la fréquentation est de 500 - 10(25) = 250 personnes. Revenu maximum : 25 × 250 = 6 250 €. Des prix plus élevés réduisent trop la fréquentation ; les prix plus bas ne compensent pas par le volume.

Erreurs courantes dans les calculs de dérivées quadratiques

Oublier de multiplier par l'exposant.Faux : la dérivée de 3x² est 3x. Correct : la dérivée de 3x² est 6x. La règle des puissances dit : multipliez par l'exposant (2), puis réduisez l'exposant par 1 (x² devient x). Donc 3x² → 3×2×x¹ = 6x. Manquer cette multiplication donne des réponses trop petites d'un facteur 2.

Garder le terme constant dans la dérivée.Faux : la dérivée de 2x² + 5x + 7 est 4x + 5 + 7. Correct : la dérivée est 4x + 5. La dérivée de toute constante est nulle — les constantes ne changent pas, donc leur taux de changement est nul. La constante c dans ax² + bx + c disparaît complètement dans la dérivée.

Mauvaise gestion des coefficients négatifs.Pour f(x) = -3x² + 4x - 2, la dérivée est f'(x) = -6x + 4, et non -6x - 4. Le coefficient b est +4, donc il reste +4 dans la dérivée. Exemple : f(x) = -x² - 5x + 3 a f'(x) = -2x - 5. Les deux termes sont négatifs car a et b sont tous deux négatifs.

Confondre dérivé avec la fonction d'origine.La dérivée f'(x) est une fonction différente de f(x). Pour f(x) = x², la dérivée f'(x) = 2x est une droite, pas une parabole. À x = 3, f(3) = 9 (la valeur de la fonction) mais f'(3) = 6 (la pente). Ne branchez pas x dans la dérivée en attendant la valeur de la fonction - la dérivée donne le taux de changement, pas le résultat.

Conseils de pro pour les dérivées quadratiques

Utilisez la formule de sommet comme raccourci.Pour tout quadratique ax² + bx + c, la coordonnée x du sommet est x = -b/(2a). Cela vient directement de la définition de la dérivée 2ax + b = 0 et de la résolution. Mémorisez cette formule - c'est plus rapide que de calculer la dérivée et de la résoudre à chaque fois. Exemple : f(x) = 3x² - 12x + 7 a un sommet en x = -(-12)/(2×3) = 12/6 = 2.

Reconnaître que la dérivée seconde est constante.Pour les quadratiques, f''(x) = 2a — une constante. Cela signifie que le taux de changement de la pente est constant (accélération uniforme en physique). Si f(x) est la position, f'(x) est la vitesse et f''(x) est l'accélération. Pour f(x) = -4,9x² + 20x, l'accélération est f''(x) = -9,8 m/s² — la gravité terrestre.

Appliquer aux équations de lignes tangentes.La tangente à f(x) en x = a a l'équation : y = f(a) + f'(a)(x - a). Exemple : f(x) = x² - 4x + 5 à x = 3. f(3) = 9 - 12 + 5 = 2. f'(x) = 2x - 4, donc f'(3) = 2. Ligne tangente : y = 2 + 2(x - 3) = 2x - 4. Cette ligne touche la parabole en exactement un point (3, 2).

À utiliser pour les problèmes de taux associés.Lorsque deux quantités sont liées par une quadratique, leurs taux de variation sont liés par la dérivée. Si aire A = πr², alors dA/dt = 2πr × dr/dt. Lorsque r = 5 cm et dr/dt = 0,2 cm/s, la surface augmente à dA/dt = 2π(5)(0,2) = 2π ≈ 6,28 cm²/s. La dérivée lie les taux.

Foire aux questions

La dérivée mesure le taux de changement. Une constante ne change jamais : sa valeur est la même partout. La pente d'une ligne horizontale y = c est nulle en tout point. Mathématiquement, en utilisant la définition limite : lim[h→0] (c - c)/h = 0/h = 0. La constante ne contribue en rien au taux de changement, elle disparaît donc dans la dérivée.

Oui, la dérivée 2ax + b peut être positive, négative ou nulle selon x. Pour f(x) = x² - 4x + 3, la dérivée f'(x) = 2x - 4 est négative lorsque x < 2 (fonction décroissante), nulle en x = 2 (sommet) et positive lorsque x > 2 (fonction croissante). Le signe de la dérivée vous indique si la fonction monte ou descend à ce stade.

Si f(x) = ax² + c (pas de terme x), la dérivée est f'(x) = 2ax. Le sommet est à x = -0/(2a) = 0 — l'axe y. Exemple : f(x) = 3x² + 7 a f'(x) = 6x. À x = 0, la pente est de 0 (sommet). À x = 2, la pente est de 12. À x = -2, la pente est de -12. La parabole est symétrique par rapport à l'axe des y.

La formule quadratique trouve où f(x) = 0 (x-ordonnées à l'origine/racines). La dérivée trouve où f'(x) = 0 (sommet/extremum). Ils répondent à différentes questions. Pour f(x) = ax² + bx + c, les racines sont x = (-b ± √(b²-4ac))/(2a), tandis que le sommet est à x = -b/(2a). Notez que la coordonnée x du sommet est la moyenne des deux racines – elle se situe exactement à mi-chemin entre elles.
