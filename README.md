# [CalcToWork](https://calcto.work) — Free Online Calculators

**460+ free calculators in 6 languages** (English, Spanish, French, Portuguese, German, Italian).

### Categories
- 🏗️ Construction & Structure (concrete, foundations, retaining walls)
- 🧱 Masonry & Enclosures (bricks, blocks, drywall)
- 🪨 Flooring & Tiling (ceramic, laminate, marble)
- 🔧 Plumbing & Water (pipes, tanks, heating)
- ⚡ Electricity & Lighting (cables, solar panels, LED)
- 🌡️ HVAC & Climate (AC, ducts, heat pumps)
- 🪵 Carpentry & Metalwork (windows, doors, stairs)
- 🎨 Painting & Finishes (wall paint, wallpaper, varnish)
- 📊 Management & Costs (budgets, insurance, productivity)
- 🔢 Mathematics (area, volume, formulas, statistics)
- ⚛️ Science & Physics (forces, energy, electricity)
- ❤️ Health (BMI, calories, body fat, BMR)
- 💰 Finance (mortgage, compound interest, tax)
- 🧪 Chemistry & more

### Tech Stack
- **Static site generator**: Python 3 + Jinja2 → 19,362 pages
- **Hosting**: Firebase Hosting
- **Backend**: 48 Cloud Functions (Node.js 20)
- **AI**: DeepSeek (autonomous SEO optimization)
- **Analytics**: Firestore + GSC integration

### Deploy
```bash
pip install -r requirements.txt
py scripts/generate_calctowork.py
firebase deploy --only hosting
```
